# Day 058｜塞萬提斯與《堂吉訶德》
## Cervantes and Don Quixote

今天，  
要暫時離開  
莎士比亞（William Shakespeare）的世界。  

從 Day050 到 Day057，  
一路看見：

- 哈姆雷特（Hamlet）的懷疑
- 馬克白（Macbeth）的野心
- 李爾王（King Lear）的自我誤認

這些作品  
共同完成了一件事：

```text
發現人的內在世界。
```

今天將遇見  
另一位同樣偉大的作家：

```text
米格爾·德·塞萬提斯
（Miguel de Cervantes）
```

如果說  
莎士比亞塑造了現代人的心理，  
那麼塞萬提斯  
幾乎創造了現代小說。  

這裡先補一個重要精度。  
把《堂吉訶德》（Don Quixote）  
稱為“第一部現代小說”，  
是文學史上非常常見、  
也很有教學力量的說法。  
但若更嚴格，  
最好理解為：

```text
它通常被視為現代小說的奠基之作，
或最重要原型之一，
而不是一個完全無爭議、可以機械式斷言的“唯一第一”。
```

同時，  
塞萬提斯的重要性  
也不只是寫出一個  
“把風車看成巨人”的瘋子。  
更重要的是，  
他把以下問題  
推到文學中心：

- 閱讀如何塑造現實
- 文本如何進入生活
- 文學如何開始反思自己

抓住這個精度，  
今天的主題就會更清楚：  

```text
《堂吉訶德》不只是一部滑稽小說，
也是現代小說意識的起點之一。
```

## 一、為什麼《堂吉訶德》這麼重要？因為它常被視為現代小說的起點之一

很多文學史家認為：

```text
《堂吉訶德》
（Don Quixote）
```

是第一部現代小說，  
或者至少是  
現代小說最重要的起點之一。  

這部作品  
第一部出版於：

```text
1605 年
```

第二部則出版於：

```text
1615 年
```

時間上  
與莎士比亞幾乎重疊。  

因此可以說：

```text
塞萬提斯與莎士比亞，
都站在現代文學世界的起點附近。
```

這句話非常重要。  

也就是說，  
十七世紀初  
不只是出現了幾部名作而已。  
更是文學形式  
開始發生深刻轉向的時刻。  

一邊，  
莎士比亞把人的內心  
推上舞台中央。  
另一邊，  
塞萬提斯把：

```text
普通人、閱讀經驗、現實錯位與小說自我反思
```

帶進文學中心。  

---

## 二、故事其實很荒唐：一個讀騎士小說讀到把自己活成小說的人

主角是一位  
鄉下沒落小貴族：

```text
堂吉訶德
（Don Quixote）
```

更準確地說，  
他原本名叫：

```text
阿隆索·吉哈諾
（Alonso Quixano）
```

他沉迷於騎士小說  
（Chivalric Romances）。  

讀得太多。  
多到一個程度，  
開始相信  
那些故事是真的。  

於是他決定：

```text
自己也要成為騎士。
```

問題在於：

```text
真正的騎士時代早就結束了。
```

這句話非常重要。  

因此故事  
從一開始就帶有  
強烈的荒謬感。  

但這種荒謬  
並不只是用來搞笑。  
更深層地說，  
塞萬提斯是在問：

```text
如果一個人把書中的世界當成現實世界，
會發生什麼？
```

這就是整部作品  
真正的核心裝置。  

---

## 三、一個活在錯誤時代的人：他的悲喜劇來自現實與故事的錯位

這是《堂吉訶德》  
最重要的設定之一。  

堂吉訶德相信，  
世界仍然充滿：

- 巨人（Giants）
- 惡龍（Dragons）
- 魔法師（Enchanters）
- 騎士冒險（Knight-errantry）

但現實世界  
早已不是那樣。  

於是：

```text
他活在兩個世界之間。
```

一個是：

```text
現實世界。
```

另一個是：

```text
故事世界。
```

這句話非常重要。  

也正因如此，  
他的處境才如此新。  
因為他不是  
單純面對命運、  
不是面對神諭、  
也不是只面對自己。  

他是在面對：

```text
自己被文本塑造之後的世界感。
```

這就是《堂吉訶德》  
之所以如此現代的原因之一。

## 四、最有名的場景：風車之所以重要，不只是滑稽，而是象徵“看見的世界已被閱讀改寫”

你可能最熟悉的，  
就是這個故事。  

堂吉訶德看見風車。  
但他認為：

```text
那不是風車，
而是巨人。
```

於是他衝上去戰鬥，  
結果被打得很慘。  

這段情節  
後來成為世界文學  
最著名的象徵之一。  

這句話非常重要。  

因為風車情節  
不只是好笑。  
它真正象徵的是：

```text
人的看見，
可能早已被自己讀過的故事改寫。
```

也就是說，  
堂吉訶德不是沒有眼睛。  
他當然看見了東西。  
問題在於，  
他用什麼框架  
來解釋眼前的東西。  

因此風車場景  
真正驚人的地方是：

```text
它把“閱讀如何影響現實知覺”寫成了一個可見場面。
```

## 五、他瘋了嗎？作品真正追問的，不只是瘋狂，而是幻想與現實誰更有力量

表面上看，  
答案似乎很簡單：

```text
是的，
他瘋了。
```

他把風車看成巨人，  
當然像是發瘋。  

但塞萬提斯  
真正想問的問題  
並不止於此。  

更深一層，  
他在問：

```text
現實一定比幻想更真實嗎？
```

這句話非常重要。  

因為堂吉訶德  
雖然經常誤認世界，  
但他追求的東西  
卻未必是錯的。  

他追求：

- 正義（Justice）
- 榮譽（Honor）
- 勇氣（Courage）
- 理想（Ideal）

這些價值  
本身並沒有錯。  
問題在於：

```text
他用錯了理解世界的形式。
```

也因此，  
《堂吉訶德》  
才會同時讓人發笑、  
又讓人感到悲傷與敬意。  

---

## 六、一個新的文學問題：從面對命運、救贖與自我，走到“人面對文本”

可以把前面學過的  
主題簡單串起來：

- 《伊利亞德》（Iliad）裡，英雄面對命運
- 《神曲》（Divine Comedy）裡，靈魂面對救贖
- 《哈姆雷特》（Hamlet）裡，人面對自己

到了《堂吉訶德》，  
出現一個全新問題：

```text
人開始面對文本。
```

這句話非常重要。  

也就是說，  
塞萬提斯讓文學  
第一次大規模地處理：

```text
人會被自己讀過的故事塑造。
```

這是非常深刻的轉變。  

因為從這裡開始，  
文學不再只是  
反映世界。  
它也開始意識到：

```text
文學本身就在參與建造人對世界的理解。
```

這就是小說意識  
真正成熟的重要一步。  

---

## 七、閱讀改變現實：堂吉訶德不是被神諭改變，而是被書改變

堂吉訶德  
不是因為戰爭改變。  
也不是因為神諭改變。  
更不是因為王位鬥爭改變。  

他之所以變成堂吉訶德，  
最直接的原因是：

```text
讀書。
```

這是文學史上  
極其重要的轉折。  

因為它提出了一個  
非常現代的問題：

```text
故事能塑造人的世界觀。
```

這句話非常重要。  

也就是說，  
閱讀不只是接收內容。  
閱讀會：

- 形成想像
- 影響判斷
- 重組價值
- 改變人對現實的理解方式

因此《堂吉訶德》  
不只是一本關於騎士妄想的書。  
它更是一本  
關於：

```text
閱讀如何進入現實生活
```

的書。  

---

## 八、桑丘：理想主義與現實主義之間的經典對照

另一位極其重要的角色是：

```text
桑丘·潘沙
（Sancho Panza）
```

他是堂吉訶德的隨從，  
也是小說中  
最重要的對照人物之一。  

堂吉訶德代表：

```text
理想主義（Idealism）
```

桑丘則更接近：

```text
現實主義（Pragmatism / Realism）
```

這句話非常重要。  

兩人形成經典對照。  

但要注意，  
這不是簡單的  
“瘋子 + 正常人”組合。  
更深層地說，  
兩人彼此影響、彼此修正。  

堂吉訶德讓桑丘  
看到理想與想像的力量。  
桑丘則讓堂吉訶德  
不斷撞回現實。  

因此他們的關係  
不是單向嘲笑。  
而是：

```text
現代小說最早、最經典的人物互動結構之一。
```

---

## 九、理想與現實：他之所以滑稽，是因為他執意活在“世界應該是什麼”裡

整部作品  
都建立在一個核心衝突上：

```text
世界是什麼？
世界應該是什麼？
```

堂吉訶德  
一直活在第二個問題裡。  

因此，  
他看起來滑稽。  
卻也顯得崇高。  

這句話非常重要。  

因為他不是為了利益  
才自欺。  
他不是想欺騙別人。  
而是誠懇地相信  
世界應該容得下：

- 正義
- 榮譽
- 英雄性
- 崇高行動

因此堂吉訶德  
雖然總是錯看現實，  
但他的錯  
帶著一種非常特殊的尊嚴。  

也正因如此，  
讀者才不會只把他  
當笑話來看。

## 十、為什麼讀者會同情他？因為他錯得可笑，卻追求得高尚

讀者之所以  
常常同情堂吉訶德，  
原因很簡單：  

```text
他雖然看錯世界，
卻並不邪惡。
```

他追求的  
是很高的東西：

- 正義
- 榮譽
- 勇氣
- 理想

這些價值  
本身都沒有錯。  

錯的是：

```text
他把小說世界當成現實世界。
```

這句話非常重要。  

這就使《堂吉訶德》  
和單純諷刺作品  
很不一樣。  
它不是只在嘲笑一個  
荒唐的人。  

它也在問：

```text
如果世界失去了理想，
那麼所謂“正常”是否也有某種貧乏？
```

因此堂吉訶德  
才會顯得又可笑、  
又令人敬重。

## 十一、現代小說的誕生：作品不只講故事，也開始討論“故事如何作用於人”

《堂吉訶德》  
之所以重要，  
原因之一就在於：

```text
它開始反思故事本身。
```

作品不只是講故事。  
它還在討論：

```text
故事如何影響人。
```

這是現代小說  
的重要起點之一。  

這句話非常重要。  

也就是說，  
塞萬提斯不是只想  
編出一個有趣情節。  
他還在問：

- 為什麼人會相信故事？
- 故事如何塑造身份？
- 故事如何改變現實感？
- 文學與生活之間的界線在哪裡？

從這裡開始，  
小說變得不只是  
敘事工具。  
它也開始具有：

```text
自我意識。
```

---

## 十二、文學開始照鏡子：從塞萬提斯開始，文學也開始思考自己是什麼

從塞萬提斯開始，  
文學發生一個  
非常新的變化。  

它不只描寫世界。  
也開始思考：

```text
文學本身是什麼？
```

這句話非常重要。  

這正是《堂吉訶德》  
極其現代的地方。  
它讓文學不只是  
透明地指向外部現實。  

它也會反過來看自己：

- 自己怎麼講故事
- 自己如何塑造讀者
- 自己與其他文本的關係
- 自己是否也在製造幻象

這條路  
後來會一路延伸到：

- 博爾赫斯（Jorge Luis Borges）
- 卡爾維諾（Italo Calvino）
- 後設小說（Metafiction）
- 後現代文學（Postmodern Literature）

因此可以說：

```text
塞萬提斯讓文學開始照鏡子。
```

## 十三、堂吉訶德真的失敗了嗎？現實上失敗，但精神上未必

這是作品  
最耐人尋味的地方之一。  

現實上，  
堂吉訶德幾乎一直失敗：

- 被嘲笑
- 被打敗
- 被誤解
- 被視為荒唐人物

但如果從精神層面看，  
問題就沒有那麼簡單。  

許多讀者認為，  
他保留了一種  
現代社會逐漸失去的東西：

```text
對理想的信念。
```

這句話非常重要。  

也就是說，  
《堂吉訶德》  
不只是讓人笑一個失敗者。  
它也讓人反過來問：

```text
如果一個完全現實、完全務實、完全不再相信理想的人才叫正常，
那麼這種正常是否也失去了某些重要的人性？
```

這正是它  
最深的複雜性。

## 十四、第一個現代小說主角：不是半神，也不是國王，而是一個普通人

為什麼堂吉訶德  
會被視為  
第一個現代小說主角之一？  

因為他不是：

- 神話英雄
- 國王
- 半神
- 史詩式偉人

他其實只是一個：

```text
普通人。
```

這句話非常重要。  

而小說  
正是普通人的文學形式。  

也就是說，  
從《堂吉訶德》開始，  
文學越來越相信：

```text
一個平凡人的幻想、錯誤、願望與遭遇，
也足以構成偉大的文學。
```

這是非常大的轉變。  

因為這意味著，  
文學不再只屬於  
英雄與神。  
它也屬於：

- 普通人
- 日常生活
- 誤解
- 內心投射
- 與現實不合拍的人

這就是小說  
真正展開的地方。

## 十五、與莎士比亞的差別：一個重心理，一個重文本與現實的關係

可以簡單比較：

### 莎士比亞（Shakespeare）
```text
更集中地打開人的內心世界。
```

### 塞萬提斯（Cervantes）
```text
更集中地打開人與故事、文本、現實之間的關係。
```

這句話非常重要。  

兩者當然都很複雜，  
不能切得太死。  
但作為課程主線，  
這個區分非常有幫助。  

也因此，  
他們常常可以被看成  
現代文學史上的“雙子星”：  

- 一位讓人看見心理
- 一位讓人看見小說

這兩條路  
共同開啟了  
後來的現代文學。

## 十六、第四部分主線中的位置：人的命運，也受到自己相信的故事影響

回到第四部分  
一路追問的核心問題：

```text
人是自己命運的主人嗎？
```

《堂吉訶德》  
給出了一個  
非常新的答案：

```text
人的命運，
很大程度上受到自己相信的故事影響。
```

這句話非常重要。  

也就是說，  
人不只是被外部事件塑造。  
也不只是被內心欲望塑造。  
人還會被：

- 自己讀過的書
- 自己相信的敘事
- 自己採用的世界理解框架

所塑造。  

因此：

```text
理解世界，
其實也包含理解自己所閱讀的東西。
```

這正是《堂吉訶德》  
對現代文學與現代人的  
一個極其深刻的提醒。

```text
與莎士比亞（Shakespeare）主要打開人的內心不同，
塞萬提斯更集中地打開人與故事、文本與現實的關係；
兩者共同構成了現代文學最重要的起點之一。
```

## 十八、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果《堂吉訶德》真正揭示的，
不是“有一個人把風車看成巨人”這麼簡單，
而是“人會被自己相信的故事所塑造，
甚至用故事來理解現實、扭曲現實、拯救現實或逃避現實”，
那麼接下來的現代文學，
是否會越來越把“人如何活在自己讀過與相信的文本之中”
當成理解人的核心方式之一？
```

這個問題現在不用急著回答。  
但它會直接把我們帶進：

- 為什麼小說會逐漸成為現代最重要的文學形式
- 為什麼“閱讀”本身會成為文學主題
- 為什麼後來的作家會越來越喜歡讓文學反思文學
- 為什麼從塞萬提斯開始，文學不只描寫世界，也描寫故事如何介入世界

因為從今天開始，  
我們已經不只是在認識一部名著。  

更準確地說：

```text
從今天開始，
正式看見現代小說如何誕生——
它不只描寫人活在世界中，
也描寫人活在自己相信的故事之中。
```
