# Day 050｜莎士比亞導論
## Shakespeare Introduction

今天來到  
文藝復興最重要的中心人物之一。  

從 Day 048 到 Day 049，  
已經看到：  
文藝復興最大的改變  
不是藝術風格，  
也不是建築形式。  

真正的改變是：

```text
人開始成為世界的中心問題。
```

接下來，  
要認識那位最徹底描寫“人”的作家：

```text
威廉·莎士比亞（William Shakespeare）
```

如果說：

- 荷馬（Homer）代表古典世界
- 但丁（Dante Alighieri）代表中世紀世界

那麼莎士比亞  
就是文藝復興與現代世界  
最重要的象徵之一。  

這裡先補一個重要精度。  
把莎士比亞概括成  
“發現人的內在世界”，  
作為課程理解非常有力。  
但若更嚴格，  
最好理解為：

```text
他不是第一個描寫內心的人，
而是把內在衝突、意識分裂、動機不透明與自我誤解，
以前所未有的戲劇強度推到舞台中央。
```

同樣，  
獨白（Soliloquy）  
也不是他發明的。  
但在他手裡，  
它成了展示人物意識  
最有力量的工具之一。  

這樣後面進入  
《哈姆雷特》（Hamlet）、  
《麥克白》（Macbeth）、  
《李爾王》（King Lear）  
會更準確。  

---

## 一、為什麼莎士比亞如此重要？因為他讓人的內心本身變成戲劇現場

很多作家會寫故事。  
很多作家也會塑造人物。  

但莎士比亞  
之所以特殊，  
就在於他似乎發現了一件事：

```text
人的內心本身就是一個宇宙。
```

這句話非常重要。  

在他之前，  
文學當然也會描寫：

- 英雄
- 國王
- 聖人
- 愛人
- 叛徒

但在他之後，  
文學越來越不只是描寫  
人物做了什麼。  
也開始深入描寫：

- 猶豫（Hesitation）
- 欲望（Desire）
- 自欺（Self-deception）
- 焦慮（Anxiety）
- 矛盾（Contradiction）

也就是說：

```text
人的內在世界，
開始真正成為文學中心。
```

這不是說莎士比亞  
突然發明了“心理”。  
而是說他把人的內心  
寫得如此有戲劇張力，  
以至於後來的文學  
幾乎無法再回到  
只寫外在行動的階段。  

---

## 二、我們對莎士比亞知道多少？其實不算很多，但足夠知道他站在劇場中心

有趣的是，  
關於莎士比亞本人的生平，  
其實知道得不算特別多。  

可以確定的是，  
他出生於：

```text
斯特拉特福鎮
（Stratford-upon-Avon）
```

後來在：

```text
倫敦（London）
```

從事戲劇工作。  

他既是：

- 劇作家（Playwright）
- 詩人（Poet）
- 演員（Actor）

這些基本資料  
當然重要。  
但真正更重要的，  
並不是生平細節本身。  
而是：

```text
他的作品如何改變了我們理解“人”的方式。
```

這句話非常重要。  

也就是說，  
莎士比亞的傳記  
雖然有空白，  
但他的劇作  
卻讓他成為  
文學史上最清晰的存在之一。  

---

## 三、為什麼戲劇特別重要？因為人物不是被描述，而是直接說話給你聽

在古代史詩裡，  
人物常常透過敘述  
被描寫。  

但在戲劇裡，  
人物是直接說話的。  

這帶來一個  
非常重大的效果：  

```text
我們能聽見角色的思想。
```

也能看見：

- 角色如何做決定
- 如何改變主意
- 如何說服自己
- 如何掩飾自己
- 如何在言語中暴露自己

因此戲劇中的人物  
往往會顯得更立體。  

因為他不只是“被介紹”。  
他是直接在舞台上  
把自己活出來。  

這一點對莎士比亞  
尤其重要。  
因為他最厲害的地方之一，  
就是讓人物在說話的過程中  
同時完成三件事：

- 推進情節
- 揭露性格
- 暴露內心裂縫

因此戲劇  
成了最適合他探索人性的形式。  

---

## 四、莎士比亞最偉大的發現之一：人不一定了解自己

今天一定要記住一句話：

```text
人不一定了解自己。
```

這句話聽起來  
好像很普通。  
但在文學史上，  
這其實是一個  
非常巨大的突破。  

因為它意味著：

```text
人的行動與人的理解，
可能並不一致。
```

這句話極其重要。  

也就是說，  
一個人可能：

- 自以為知道自己在做什麼
- 其實根本不知道自己的真正動機
- 自以為在追求某種善
- 實際上卻在走向毀滅
- 自以為認識自己
- 卻只是在維持一個幻想的自我形象

這種人物理解方式  
會徹底改變文學。  

因為從這裡開始，  
人物不再只是  
善與惡的標誌。  
他也變成：

```text
一個對自己都不透明的存在。
```

這正是現代人物  
真正開始誕生的地方。

## 五、《哈姆雷特》：思考與行動之間第一次出現巨大裂縫

例如：

```text
《哈姆雷特》（Hamlet）
```

中的哈姆雷特  
就是這種新人物的代表。  

他知道自己  
應該行動。  
卻無法真正行動。  

於是，  
在他身上就產生了一條  
非常著名的裂縫：

```text
思考與行動之間的裂縫。
```

這句話非常重要。  

在古典文學裡，  
人物當然也會猶豫。  
但像哈姆雷特這樣，  
把整個劇作的力量  
都建立在：

- 自我反省
- 反覆推遲
- 思考過度
- 對行動正當性的懷疑
- 對自身意志的不信任

這樣的結構上，  
其實並不常見。  

因此哈姆雷特  
不只是一個王子形象。  
他更像是一種  
現代意識的開端：

```text
知道很多，
卻因此更難行動。
```

---

## 六、《麥克白》：悲劇不只是壞人作惡，而是普通人如何變成另一個自己

再看：

```text
《麥克白》（Macbeth）
```

中的麥克白。  

他並不是  
一開始就被寫成  
純粹天生邪惡的人。  
相反，  
他更可怕的地方正在於：

```text
他原本看起來並不是怪物。
```

他是在：

- 野心（Ambition）
- 誘惑（Temptation）
- 自我說服（Self-justification）
- 恐懼（Fear）

的推動下  
逐漸墮落。  

因此莎士比亞關心的  
不是單純的：

```text
壞人做壞事。
```

而是：

```text
普通人如何變成另一個自己。
```

這句話非常重要。  

因為這使悲劇  
不再只是道德標本展示。  
它變成了一種更可怕的問題：

```text
如果條件成立，
每個人是否都可能滑向自己不敢承認的樣子？
```

這正是《麥克白》  
真正讓人不安的地方。  

---

## 七、《李爾王》：悲劇很多時候來自認知錯誤，而不是單純邪惡

再看：

```text
《李爾王》（King Lear）
```

中的李爾王。  

他並不是單純暴君。  
他更像是一個：

```text
誤解自己的人。
```

這句話非常重要。  

他一開始  
以為自己知道：

- 誰愛他
- 自己是誰
- 權力如何運作
- 父愛與尊嚴意味著什麼

但事實證明，  
他錯了。  

而且他不是被別人完全擊倒而已。  
更深層地說，  
他是被自己的錯誤認知  
一步步帶入崩潰。  

因此《李爾王》的悲劇性  
很大程度來自：

```text
認知錯誤。
```

這就是莎士比亞  
非常現代的一點。  
他讓悲劇不只來自命運、  
陰謀或惡人。  
也來自：

- 看錯
- 聽錯
- 以為自己了解
- 實際上完全不了解

這種“認識上的錯誤”  
會成為後來現代小說  
極其重要的母題。  

---

## 八、人的矛盾：莎士比亞人物幾乎都同時活在衝突中

莎士比亞的人物  
幾乎都有一個共同特徵：

```text
他們同時相信兩件互相衝突的事情。
```

這一點非常重要。  

他們往往：

- 想要愛，又害怕受傷
- 想要權力，又害怕責任
- 想要真理，又害怕真理
- 想保持道德，又想滿足欲望
- 想忠於自己，又想逃離自己

這種矛盾  
使他的角色  
不再是扁平類型。  
而變成：

```text
充滿自我衝突的人。
```

這也是為什麼  
莎士比亞至今仍然現代。  

因為現代人的處境  
恰恰就是這樣：  

```text
我們不是不知道要什麼，
而是常常同時想要互相衝突的東西。
```

而莎士比亞  
幾乎比任何更早的作家  
都更早看清了這件事。  

---

## 九、內心第一次成為舞台：真正的衝突常常發生在角色心裡

在莎士比亞之前，  
舞台當然也會呈現思想。  
但整體上，  
舞台更常被理解為  
事件發生的地方。  

到了莎士比亞這裡，  
情況變了。  

可以說：

```text
舞台開始真正呈現思想。
```

因此：

```text
真正的衝突常發生在角色心裡。
```

這句話極其重要。  

也就是說，  
觀眾看到的  
不只是外在衝突：

- 誰殺了誰
- 誰奪權
- 誰被背叛
- 誰愛上誰

更重要的是  
內在衝突：

- 他為什麼遲疑
- 他怎麼說服自己犯罪
- 他如何在良知與欲望間掙扎
- 他如何慢慢變成自己不認識的人

這確實是一場  
文學革命。  

因為從這裡開始，  
舞台不只呈現事件。  
它也開始呈現：

```text
意識本身的運動。
```

## 十、著名的獨白：獨白不是台詞堆砌，而是意識被直接打開

莎士比亞最有名的技巧之一，  
就是：

```text
獨白（Soliloquy）
```

角色獨自說話。  
觀眾因此聽見  
他的內心。  

例如《哈姆雷特》裡  
最著名的：

```text
To be, or not to be
```

這不只是名句。  
它真正重要的地方在於：

```text
它把意識本身呈現在舞台上。
```

這句話非常重要。  

獨白的力量  
不只在於抒情。  
它更在於讓角色  
在眾目睽睽之下  
暴露出自己最深的：

- 迷惘
- 判斷
- 自我質疑
- 欲望
- 恐懼

因此觀眾不只是  
“聽到好台詞”。  
而是直接進入  
角色的心靈內部。  

這也是莎士比亞  
對後世最深遠的技術遺產之一。  

---

## 十一、為什麼四百年後仍然重要？因為人的基本情感結構沒有改變

莎士比亞四百多年後  
仍然重要，  
一個根本原因是：  

```text
時代改變了，
但人沒有停止成為人。
```

政治改變了。  
科技改變了。  
社會形式也改變了。  

但人依然會：

- 猶豫
- 愛
- 嫉妒
- 恐懼
- 渴望被理解
- 誤解自己
- 做出錯誤選擇

因此：

```text
莎士比亞沒有過時。
```

這句話非常重要。  

他的作品之所以常讀常新，  
不是因為它提供  
歷史資訊。  
而是因為它不斷讓讀者發現：

```text
人的內在結構，
其實遠比時代更長久。
```

這就是他至今仍然  
具有穿透力的原因。

## 十二、莎士比亞與現代小說：後來對心理與意識的探索，很難繞過他

今天必須理解一點：

```text
沒有莎士比亞，
就很難有後來的心理小說。
```

這句話作為課程判斷  
非常有力量。  

如果更嚴格地說，  
應該理解為：

```text
後來小說對心理、意識與自我的深度探索，
很難繞過莎士比亞所開啟的道路。
```

因為他開啟了一條路：  

```text
描寫人的內在世界。
```

這條路  
最終會通往：

- 陀思妥耶夫斯基（Fyodor Dostoevsky）
- 普魯斯特（Marcel Proust）
- 弗吉尼亞·伍爾夫（Virginia Woolf）

也就是說，  
從戲劇到小說，  
從舞台獨白到意識流，  
中間當然還有很多轉折。  
但有一條深層線索  
非常清楚：

```text
莎士比亞讓文學學會了
如何把人的內部世界當作主要現實來描寫。
```

## 十三、為什麼他是第四部分的核心？因為他最完整地呈現了自由與局限同時存在的人

第四部分的核心問題是：

```text
人是自己命運的主人嗎？
```

莎士比亞的回答  
非常複雜。  

他既相信人的自由。  
也看到人的局限。  

因此他的作品  
總是同時充滿：

- 選擇（Choice）
- 後果（Consequence）
- 自由（Freedom）
- 誤判（Misjudgment）
- 責任（Responsibility）
- 無法控制的局勢

也就是說，  
他既不是單純樂觀的人文主義者，  
也不是單純宿命論者。  

他真正看見的是：

```text
人有自由，
但人的自由從來不是透明、純粹、完全可控的。
```

這一點非常重要。  

因為這正是  
現代人的核心處境：  

- 我得選
- 但我不一定知道自己在選什麼
- 我必須負責
- 但我不一定完全理解自己的動機

莎士比亞  
正是把這種複雜性  
第一次大規模寫進文學的人之一。  

---

## 十四、從但丁到莎士比亞：文學焦點正式從外在秩序轉向內在意識

如果要用最簡單的方式  
比較但丁與莎士比亞，  
可以這樣說：

### 但丁（Dante） 的世界：
```text
宇宙有秩序。
```

### 莎士比亞（Shakespeare） 的世界：
```text
人的內心充滿衝突。
```

這就是文學主線  
最關鍵的轉折之一。  

因此，  
從這裡開始，  
文學的焦點正式從：

```text
外在秩序
```

轉向：

```text
內在意識
```

這句話非常重要。  

在但丁那裡，  
最根本的問題是：

- 世界如何被安排
- 靈魂如何走向真理
- 人如何在宇宙秩序中找到自己的位置

而在莎士比亞這裡，  
最根本的問題變成：

- 人是否理解自己
- 人如何被欲望與恐懼撕裂
- 人如何在選擇中暴露自己的真相

這正是  
第四部分真正展開的地方。

## 十六、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果莎士比亞真正改變的，
不是“故事怎麼寫”，
而是“人如何在文學中被理解”，
那麼接下來閱讀《哈姆雷特》（Hamlet）、《麥克白》（Macbeth）與《李爾王》（King Lear）時，
我們究竟是在看悲劇情節，
還是在看現代自我如何第一次被完整地放上舞台？
```

這個問題現在不用急著回答。  
但它會直接把我們帶進：
