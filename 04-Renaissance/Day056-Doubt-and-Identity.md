# Day 056｜懷疑與身份
## Doubt and Identity

從內在衝突（Internal Conflict）  
再往前一步，  
今天正式走向  
一個更困難的問題：

```text
真正的我是誰？
```

昨天已經看到，  
人並不是單一而透明的存在。  
在我們內部，  
常常有很多聲音彼此競爭：

- 理性與情感
- 慾望與道德
- 勇氣與恐懼

但當這些聲音越來越多時，  
新的問題就出現了：

```text
那麼，真正的我是誰？
```

這就是今天的主題。  

這裡先補一個重要精度。  
把文藝復興概括成  
“身份不再由命運或神決定，而需要個體自己建構”，  
作為主線非常有效。  
但若更嚴格，  
最好補上：

```text
文藝復興並不是突然讓身份完全自由。
```

更準確地說，  
它一方面提高了個體  
自我塑造（self-fashioning）的可能；  
另一方面也讓身份  
更強地受到以下因素影響：

- 社會角色
- 語言
- 權力
- 表演
- 他人眼光

同樣，  
莎士比亞（William Shakespeare）  
的重要性，  
也不只是讓人物“懷疑自己”。  
更是讓：

```text
“自我”本身變成需要被解釋、被扮演、被驗證的對象。
```

抓住這個精度，  
今天的問題就會變得更深：  

```text
現代人的危機，
不是沒有身份，
而是身份不再完全穩定。
```

---

## 一、古典世界的身份：身份首先是一種清晰的位置

回到第一部分。  

在《伊利亞德》（Iliad）中，  

```text
阿基里斯（Achilles）
```

知道自己是誰。  

他是：

- 英雄
- 戰士
- 榮耀的承擔者

他的身份  
主要來自戰場與榮譽。  
非常清晰。  

同樣地，  
《奧德賽》（Odyssey）中的：

```text
奧德修斯（Odysseus）
```

也知道自己是誰。  

他是：

- 國王
- 戰士
- 旅人
- 回家者

這種身份  
雖然會受遭遇影響，  
但基本上仍然：

```text
穩定而明確。
```

這句話非常重要。  

因為在古典世界裡，  
身份更多像是一個  
被世界清楚承認的位置。  

也就是說，  
一個人要回答“我是誰”，  
往往可以訴諸：

- 家族
- 角色
- 榮譽
- 義務
- 社會位置

因此自我  
還沒有被全面問題化。  

---

## 二、中世紀的身份：身份由靈魂在神聖秩序中的位置決定

到了《神曲》（Divine Comedy），  
人的身份  
主要來自：

```text
靈魂在宇宙中的位置。
```

你是：

- 罪人（Sinner）
- 悔改者（Penitent）
- 得救者（Saved Soul）

你的位置  
由神聖秩序（Divine Order）  
所決定。  

因此：

```text
身份依然穩定。
```

這句話非常重要。  

在中世紀，  
人當然也會掙扎、迷失、犯罪。  
但無論如何，  
“我是誰”這個問題  
仍然被放在一個  
比個體更大的秩序中回答。  

也就是說，  
身份的最終來源  
不是自己發明自己。  
而是：

```text
你在上帝（God）面前的位置。
```

這就是中世紀身份觀  
和文藝復興之後  
最不同的地方之一。  

---

## 三、文藝復興的改變：當人可以塑造自己，身份就開始變成問題

到了文藝復興（Renaissance），  
事情開始不同。  

人文主義（Humanism）說：

```text
人可以塑造自己。
```

這聽起來非常美好。  
它意味著：

- 人不只是被安排
- 人也可以成長
- 可以學習
- 可以建構自己的形象與人生

但它也帶來一個  
深刻的副作用：  

```text
如果我可以成為很多種人，
那麼我究竟是哪一種人？
```

這句話非常重要。  

也就是說，  
文藝復興把人  
從固定位置中部分解放出來，  
同時也把人  
丟進了一個新的不安之中：  

```text
身份不再完全被給定，
而要被建構。
```

而一旦需要建構，  
就意味著：

- 可能失敗
- 可能表演
- 可能偽裝
- 可能反覆修改
- 可能永遠不確定自己是否真的成為了那個人

這就是現代身份問題  
真正開始的地方。  

---

## 四、哈姆雷特的問題：他不只不知道要做什麼，也不知道自己該成為什麼樣的人

《哈姆雷特》（Hamlet）  
其實一直在問：

```text
我該成為什麼樣的人？
```

是復仇者嗎？  
是王子嗎？  
是哲學家嗎？  
是兒子嗎？  

這些身份  
彼此並不完全一致。  

因此：

```text
哈姆雷特無法找到穩定的自己。
```

這句話非常重要。  

他的危機  
不只是“要不要報仇”。  
更深層地說，  
是他不知道  
哪一種身份  
才真正能整合自己。  

也就是說，  
哈姆雷特不是只有行動困難。  
他也有：

```text
身份困難。
```

因為每一種角色  
都會要求他  
成為不同的人：  

- 作為兒子，要報父仇
- 作為王子，要維護國家秩序
- 作為反思者，要懷疑證據與正當性
- 作為有良知的人，要警惕自己成為暴力者

這就是他  
無法穩定的原因之一。  

---

## 五、懷疑的力量：現代人不只懷疑世界，也開始懷疑自己

過去的人  
當然也會懷疑世界。  

但到了現代性的起點，  
一件更重要的事開始發生：

```text
人開始懷疑自己。
```

這是一個極大的轉折。  

因為從這裡開始：

```text
自我不再是答案，
自我變成問題。
```

這句話非常重要。  

也就是說，  
“我是誰”  
不再能被快速回答。  
它開始變成一個  
持續被追問的問題。  

而一旦自我  
成為需要被解釋的對象，  
人就會開始：

- 觀察自己
- 分析自己
- 不信任自己的動機
- 質疑自己的敘述
- 懷疑自己是否只是在扮演某個角色

這正是現代主體  
最深的焦慮之一。  

---

## 六、Identity 是什麼？身份不是只有名字與職業，而是對自己的整體敘述

Identity（身份）  
不只是名字。  
也不只是職業。  

更深層地說，  
身份其實是：

```text
我如何理解自己。
```

因此：

```text
身份是一種敘事。
```

是我們對自己  
講述的故事。  

這句話非常重要。  

因為當人回答  
“我是誰”時，  
很少只是列清單。  
我們通常會說出  
一個關於自己的故事：  

- 我是怎樣的人
- 我重視什麼
- 我來自哪裡
- 我曾經歷過什麼
- 我未來想成為誰

也就是說，  
身份不是靜止標籤。  
它更像是一種  
持續編排的自我敘述。  

而這也解釋了  
為什麼身份會不穩。  
因為只要故事變了，  
自我就可能跟著搖動。  

---

## 七、如果故事改變呢？身份的危機常常發生在自我敘述崩塌的時候

假設你一直認為自己是：

```text
成功的人。
```

突然失敗。  

或者一直認為自己是：

```text
善良的人。
```

卻做出傷害別人的事。  

那麼原本的身份  
就開始動搖。  

這句話非常重要。  

因為很多時候，  
身份危機  
不是來自外在名稱改變。  
而是來自：

```text
你原本用來理解自己的故事，
突然不再成立。
```

這就是為什麼  
懷疑會如此痛苦。  

因為它不只是  
對某個選擇產生疑問。  
它更可能意味著：

- 我對自己的理解錯了
- 我一直活在某種幻象裡
- 我不是我以為的那種人

這正是現代身份危機  
最深的地方。  

---

## 八、李爾王的崩解：當“我是國王”失效時，自我也跟著崩解

《李爾王》（King Lear）  
正是如此。  

他一直認為：

```text
我是國王。
```

但當王位消失後，  
問題立刻出現：

```text
如果不是國王，
我是誰？
```

這種身份崩解  
正是現代人的經驗之一。  

這句話非常重要。  

因為李爾王的悲劇  
不只是失去權力。  
更深層地說，  
是他失去了那個  
一直支撐自我理解的中心敘事。  

也就是說，  
當“國王”這個角色  
不再能穩固地定義他時，  
他就必須第一次面對：

- 沒有權力的自己
- 沒有讚美的自己
- 沒有中心位置的自己

這也是為什麼  
《李爾王》的崩潰  
如此現代。  
因為它不是單純政治失敗。  
它更是：

```text
身份失效之後的自我崩解。
```

---

## 九、面具與真實：如果每個角色都是一種表演，那麼真正的我在哪裡？

莎士比亞反覆提出  
一個極其重要的問題：

```text
我們是不是都戴著面具？
```

例如：

- 國王的面具
- 父親的面具
- 戀人的面具
- 朋友的面具
- 忠臣的面具

那麼：

```text
面具後面是誰？
```

這句話非常重要。  

這裡的“面具”  
不是只指虛偽。  
它也指：

- 社會角色
- 公共身份
- 被期待扮演的自己
- 用來與世界互動的形象

而文藝復興之後，  
這個問題變得特別尖銳。  

因為一旦身份  
可以被建構、被塑造、被表演，  
那麼人就會開始懷疑：

```text
我現在的樣子，
到底是真實的我，
還是我學會扮演的我？
```

這正是  
現代身份問題的核心之一。

## 十、自我觀察的誕生：人不只活著，也開始觀看自己如何活著

今天的人  
很習慣：

- 反省自己
- 分析自己
- 懷疑自己

但這其實不是  
永遠如此自然的事情。  
它是一種歷史產物。  

而莎士比亞  
正是最早大量描寫  
這種現象的作家之一。  

這句話非常重要。  

因為從這裡開始，  
人不只是生活。  
也開始：

```text
觀看自己如何生活。
```

這種自我觀察  
會帶來很大力量。  
它讓人成熟、反思、校正自己。  

但它也帶來痛苦。  
因為一旦你開始  
長時間觀察自己，  
就很難再完全天真地  
相信自己原本的身份敘述。  

這也就是為什麼  
現代主體  
總是和懷疑相連。  

---

## 十一、為什麼懷疑會帶來痛苦？因為穩定感來自確定“我是誰”

懷疑之所以痛苦，  
原因之一是：

```text
穩定感來自確定性。
```

當你知道自己是誰，  
世界就比較容易理解。  

但當身份開始搖動，  
焦慮就出現了。  

於是：

```text
自由與不安一起成長。
```

這句話非常重要。  

因為文藝復興  
提高了個體的自由。  
但自由的代價之一  
就是：

- 沒有人替你最終定義你
- 你必須自己講述自己
- 你也必須承受這個敘述可能崩塌

這就是現代人的處境。  

不是沒有身份。  
而是：

```text
身份需要被不斷維持、修正與說服。
```

這也是為什麼  
自我懷疑會成為  
長期存在的精神狀態。  

---

## 十二、現代主體的誕生：主體不是單純存在，而是會意識到自己存在

今天要學一個  
重要概念：

```text
主體（Subject）
```

可以先簡單理解成：

- 能夠意識到自己存在
- 能夠反思自己
- 能夠質疑自己
- 能夠把自己當成認識對象的人

這就是所謂  
現代主體。  

這句話非常重要。  

因為從這裡開始，  
“我”不再只是  
活著的那個人。  
也是：

```text
會對自己發問的那個人。
```

而這種主體  
本身就包含緊張。  

因為只要你能反思自己，  
你就可能：

- 肯定自己
- 修正自己
- 懷疑自己
- 拆解自己

因此現代主體  
從一開始  
就不是安穩的。  
它是：

```text
帶著自我問題活著的主體。
```

## 十三、從英雄到自我：我是誰，開始變成一個需要自己回答的問題

可以做一個  
非常清楚的比較：

### 古典英雄
```text
我是誰 → 很清楚。
```

### 中世紀信徒
```text
我是誰 → 由神決定。
```

### 文藝復興個體
```text
我是誰 → 我必須自己回答。
```

這就是現代性的開始。  

這句話非常重要。  

因為從這裡開始，  
身份不再只是：

- 一個角色
- 一個位置
- 一個秩序中的固定點

它也變成：

```text
一個需要被個體持續建構、辯護與維持的答案。
```

而一旦答案  
需要自己給出，  
那麼懷疑就幾乎不可避免。  

---

## 十四、文學為什麼重要？因為它讓人看見自己也在經歷同樣的懷疑

文學之所以重要，  
正是因為它提供了一個空間，  
讓人觀察自己。  

當讀《哈姆雷特》（Hamlet）時，  
其實也在觀察：

- 自己的猶豫
- 自己的懷疑
- 自己的矛盾
- 自己的拖延
- 自己對身份的困惑

因此：

```text
文學開始成為理解自我的工具。
```

這句話非常重要。  

也就是說，  
文學不只是娛樂、  
不是只講別人的故事。  
它之所以能穿越時代，  
就在於它會反過來照亮讀者自己。  

這也是莎士比亞  
至今仍然重要的原因之一。  
因為他讓人物  
不只代表一個時代。  
也代表：

```text
每個會懷疑自己的人。
```

## 十五、通往現代主義的道路：穩定自我的問題，會在後面變得更激烈

今天的課  
其實埋下了一顆  
非常重要的種子。  

到了第七部分：

```text
現代主義（Modernism）
```

作家們將會更激烈地追問：

```text
是否真的存在穩定的自我？
```

而那時，  
這個問題會被推得更遠。  

但它的起點之一，  
就在莎士比亞。  

這句話非常重要。  

因為一旦莎士比亞  
讓人物開始懷疑自己、  
扮演自己、  
解釋自己、  
無法穩定自己，  
那麼後來的文學  
就必然會繼續追問：

- 自我是本質嗎？
- 還是敘述？
- 是中心嗎？
- 還是流動？
- 是統一體嗎？
- 還是碎片的暫時組合？

這些問題  
都會一路延伸下去。

## 十六、第四部分主線中的位置：在掌握命運之前，人甚至得先回答“我是誰”

回到第四部分  
到目前為止的核心問題：

```text
人是自己命運的主人嗎？
```

今天發現，  
這個問題其實還可以再往前推一步：  

```text
在掌握命運之前，
人甚至必須先回答：
我是誰？
```

而這個問題，  
遠比想像困難。  

這句話非常重要。  

因為如果一個人  
連自己的身份、動機、角色與自我敘述  
都不穩定，  
那麼“掌握命運”  
就不可能只是意志強不強的問題。  

它還會取決於：

- 你怎麼理解自己
- 你是否活在自我幻象中
- 你是不是只是在扮演一個被期待的角色
- 你有沒有能力承受對自己的懷疑

這正是第四部分  
越來越深的地方。

```text
這使身份危機成為一種極其現代的經驗。
```

```text
第六，
現代主體（Subject）的重要特徵之一，
就是能夠意識到自己、反思自己、質疑自己；
因此現代人不再只是活著，
也開始觀看自己如何活著。
```

```text
第七，
現代人的重要特徵之一，
就是把“我是誰”當成終身問題；
而這條從莎士比亞打開的道路，
最終會在現代主義（Modernism）中變得更加激烈。
```

## 十八、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果文藝復興真正改變的，
不只是“人能選擇什麼”，
而是“人必須自己回答自己是誰”，
而且這個答案還會受到角色、語言、表演、權力與自我懷疑不斷動搖，
那麼接下來的現代文學，
是否會越來越把“身份”看成一種永遠無法完全穩固的建構？
```

這個問題現在不用急著回答。  
但它會直接把我們帶進：

- 為什麼現代小說會越來越關心身份危機而不是單純情節發展
- 為什麼“自我敘事”會成為理解人物的重要方法
- 為什麼從莎士比亞開始，文學不只描寫人做了什麼，也描寫人如何為自己編出“我是誰”的故事
- 為什麼到了現代主義，“穩定自我”會成為最值得懷疑的觀念之一

因為從今天開始，  
我們已經不只是在問人物為什麼猶豫。  

更準確地說：

```text
從今天開始，
正式看見現代文學如何把“我是誰”變成一個永遠無法輕易回答、
卻又不得不反覆回答的核心問題。
```
