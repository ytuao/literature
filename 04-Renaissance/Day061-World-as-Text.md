# Day 061｜世界作為文本
## World as Text

從《堂吉訶德》（Don Quixote）  
一路走到今天，  
問題開始被推得更遠。  

前幾天看到的是：

- 故事影響現實
- 閱讀塑造世界觀
- 想像力改變人生

而今天要問的，  
已經不只是：

```text
故事影響世界。
```

而是：

```text
我們是否根本就是透過故事理解世界？
```

這裡先補一個重要精度。  
把 “World as Text”  
理解成“世界就像一本可以被閱讀的書”，  
這很適合作為課程推進。  
但若更嚴格，  
最好補上：

```text
這不是說世界真的等於文本，
也不是說物質現實不存在。
```

更準確地說，  
它的意思是：

```text
人類接觸世界時，
總是透過符號、語言、敘事與解釋活動來賦予它意義。
```

因此今天真正的重點  
不在於否定世界。  
而在於指出：

```text
人從來不是赤裸地面對世界，
而總是在詮釋世界。
```

抓住這個精度，  
今天的主題就會更清楚：

```text
所謂“世界作為文本”，
其實是在說：
人理解世界的方式，很像閱讀。
```

---

## 一、什麼叫「世界作為文本」？意思不是世界是紙，而是世界總要被解讀

今天的標題是：

```text
World as Text
```

直譯可以說：

```text
世界作為一本書。
```

當然，  
世界不是紙張做的。  
也不是一本真正的小說。  

這句話真正的意思是：

```text
人理解世界的方式，
很像閱讀文本。
```

這句話非常重要。  

因為文本之所以成立，  
不只是因為有字。  
而是因為有人：

- 辨認符號
- 連接關係
- 賦予意義
- 做出解釋

世界也是如此。  
人並不只是看見東西。  
人還會不斷問：

- 這是什麼？
- 這代表什麼？
- 這意味著什麼？
- 這和我有什麼關係？

也就是說，  
人和世界之間，  
總是存在一層：

```text
詮釋（Interpretation）
```

---

## 二、閱讀是什麼？閱讀的本質不是看字，而是把符號變成意義

當讀一本書時，  
看到的其實只是符號。  

例如：

```text
「英雄」
```

這兩個字，  
本身不是英雄。  
它只是符號。  

但大腦會賦予意義。  
於是：

```text
故事誕生。
```

這句話非常重要。  

也就是說，  
閱讀從來不只是  
眼睛掃過字形。  
閱讀真正發生在：

- 理解
- 聯想
- 解碼
- 意義生成

的時候。  

因此文本  
不是天然就有意義。  
它的意義  
是在讀者的理解中  
被啟動的。  

這點非常重要，  
因為它說明：

```text
世界之所以像文本，
正是因為人也總是在把眼前事物變成“有意義的東西”。
```

---

## 三、看世界也是如此：人不是只看到物體，而是立刻把物體讀成意義

當看世界時，  
其實也在不斷解釋。  

看到某人微笑，  
會判斷：

```text
他高興。
```

看到國旗，  
會想到：

```text
國家。
```

看到戒指，  
會想到：

```text
婚姻。
```

這些都不是  
物體本身自然說出來的。  
而是人替它們  
賦予的意義。  

這句話非常重要。  

也就是說，  
看到的從來不只是  
顏色、形狀、材料。  
人總是立刻把眼前東西  
讀成某種社會、情感、文化意義。  

因此“看世界”  
本身就已經包含：

```text
閱讀世界。
```

這正是  
今天整課  
最重要的轉折。  

---

## 四、人活在意義裡：我們的世界不只是物理世界，更是被故事充滿的意義世界

今天一定要記住  
一句最重要的話：

```text
人不是活在物體的世界裡，
而是活在意義的世界裡。
```

石頭只是石頭。  
但墓碑不是。  
因為墓碑承載故事。  

照片只是紙張。  
但家族照片不是。  
因為照片承載記憶。  

這句話非常重要。  

也就是說，  
對人來說，  
世界真正重要的部分，  
往往不是物理材料本身，  
而是它所承載的：

- 記憶
- 情感
- 故事
- 身份
- 社會關係

這就說明了  
為什麼“世界作為文本”  
不是空洞比喻。  
因為人類生活  
本來就深深依賴  
意義的編織。

## 五、世界充滿符號：語言、金錢、法律、國家與宗教都依賴解讀

每天接觸的很多東西，  
其實都依賴  
符號系統（Symbolic Systems）。  

例如：

- 語言（Language）
- 金錢（Money）
- 法律（Law）
- 國家（Nation）
- 宗教（Religion）

這些東西  
都不是只靠物質  
就能成立。  

因此可以說：

```text
人類社會其實是一個巨大的解讀網絡。
```

這句話非常重要。  

因為這些制度與實體  
之所以能運作，  
是因為人們共同理解：

- 某個標誌代表什麼
- 某段文字具何效力
- 某種儀式意味什麼
- 某種身分如何被承認

也就是說，  
社會不是只有物理互動。  
它更深層地建立在：

```text
共享意義的能力之上。
```

這正是“世界作為文本”  
最具體的一層。

## 六、塞萬提斯的重要發現：人不是直接接觸世界，而是透過敘事接觸世界

《堂吉訶德》  
的革命性就在這裡。  

塞萬提斯發現：

```text
人不是直接接觸世界，
而是透過敘事接觸世界。
```

這句話非常重要。  

也就是說，  
堂吉訶德的問題  
不只是眼睛看錯。  
更是他的整套  
理解世界的敘事框架  
出了偏差。  

因此：

```text
不同故事，
會產生不同現實。
```

這裡的“現實”  
不是指物理事實本身改變。  
而是指：

```text
人經驗到的世界、
判斷到的世界、
相信的世界，
會隨敘事框架而改變。
```

這就是塞萬提斯  
真正現代的地方。

## 七、騎士小說塑造了堂吉訶德：他不是先看見世界，再理解世界，而是先有故事，再看世界

堂吉訶德看見風車，  
卻想到巨人。  

為什麼？  

因為他閱讀過  
大量騎士小說  
（Chivalric Romances）。  

換句話說：

```text
他不是先看到世界，
再理解世界。
```

而是：

```text
先有故事，
再看見世界。
```

這句話非常重要。  

這正是《堂吉訶德》  
最深刻的洞察之一。  

因為它說明：  
人的觀看  
從來不是純粹被動接收。  
觀看本身  
常常已經被：

- 既有想像
- 敘事模板
- 文化形式
- 願望投射

所預先塑形。  

因此堂吉訶德  
才會成為  
現代閱讀理論的  
先驅性人物。

## 八、我們其實也一樣：同樣的世界，不同的人會讀出不同人生答案

例如：

```text
成功是什麼？
```

有人認為：

- 財富
- 地位
- 可見成就

有人認為：

- 自由
- 平靜
- 家庭幸福

為什麼答案不同？  

因為每個人  
接受的故事不同。  

這句話非常重要。  

也就是說，  
人對人生問題的回答，  
通常不是從零開始。  
它們總是建立在：

- 家庭敘事
- 社會價值
- 文化榜樣
- 宗教觀念
- 小說、影視與媒體故事

之上。  

因此就算今天  
沒有把風車看成巨人，  
也仍然在：

```text
用某種故事讀自己的生活。
```

這正是  
堂吉訶德與我們  
真正相連的地方。

## 九、人生是一個敘事：我們不只是生活，也不斷把生活講成故事

很多現代學者認為：

```text
人生本身就是一種敘事。
```

這句話非常重要。  

因為人總是在不斷講述：

- 自己的過去
- 自己的選擇
- 自己的失敗
- 自己的夢想
- 自己如何走到今天

這些故事  
不只是回憶。  
它們其實構成了：

```text
身份（Identity）
```

也就是說，  
人不是先有一個  
固定自我，  
然後再講故事。  
很多時候，  
正是透過講故事，  
人才逐漸形成  
“我是誰”的感覺。  

這就讓“世界作為文本”  
不只是一種外部理論。  
它也直接進入  
我們對自己的理解。

## 十、身份也是文本：你如何描述自己，就如何理解自己

回到 Day056，  
曾經討論過：

```text
身份（Identity）
```

今天可以更進一步說：

```text
身份不是固定物體，
而是一個故事。
```

你如何描述自己，  
就如何理解自己。  

這句話非常重要。  

例如：

- 我是受害者
- 我是倖存者
- 我是失敗者
- 我是正在學習的人

這些說法  
描述的是同一個人生，  
卻會產生  
非常不同的自我理解。  

因此身份  
不是單純被發現。  
它也經常被：

- 敘述
- 編排
- 修訂
- 重寫

這也就是  
現代主體  
為什麼如此不穩定。  
因為身份  
本身就具有文本性。

## 十一、文學的新階段：從命運、救贖與自我，走到“理解方法”的問題

可以把前面  
一路走來的問題  
串起來看：

- 古典文學問：命運是什麼？
- 中世紀文學問：救贖是什麼？
- 莎士比亞問：我是誰？

而塞萬提斯  
則開始問：

```text
我們理解世界的方法是什麼？
```

這句話非常重要。  

這是一個  
非常深的轉折。  
因為從這裡開始，  
文學不再只處理：

- 世界裡發生了什麼
- 人物做了什麼
- 命運如何安排人

它還開始處理：

```text
人是用什麼方式把世界理解成那個樣子的？
```

這正是現代文學  
方法論意識的起點之一。

## 十二、現代世界的開始：文學不再只討論內容，也開始討論形式與講述方式

這個問題  
非常重要。  

因為從此以後，  
文學不再只討論內容。  
也開始討論：

```text
形式（Form）
```

不只問：

```text
故事講什麼？
```

還問：

```text
故事如何講述？
```

這句話非常重要。  

也就是說，  
現代文學  
越來越有自我意識。  
它會開始注意：

- 誰在說故事
- 怎麼說
- 用什麼視角說
- 哪些東西被省略
- 哪些意義是被建構出來的

這條路  
後來會深深影響：

- 小說結構
- 敘事理論
- 後設小說
- 後現代文學

而它的重要起點之一，  
就正在塞萬提斯這裡。

## 十三、世界可以被閱讀嗎？從中世紀的神聖文字，到現代的世俗符號網絡

中世紀曾經認為：

```text
自然界是上帝的文字。
```

也就是說，  
世界中的萬物  
都帶有神聖意義。  

到了現代，  
這個觀念  
逐漸世俗化。  

人開始相信：

```text
世界本身是一套可以解讀的符號。
```

這句話非常重要。  

也就是說，  
就算不再訴諸上帝，  
人仍然保留了一種  
基本姿態：

```text
世界不是純粹沉默的東西，
而是需要被閱讀、被解釋、被賦義的東西。
```

這正是  
“世界作為文本”  
在現代版本中的核心。

## 十四、通往後現代：從塞萬提斯出發，最後會走到“如果世界也是文本，那真實還在哪裡？”

今天這一課  
非常重要，  
因為它會一路  
延伸到課程最後。  

到了第九部分，  
例如：

- 豪爾赫·路易斯·博爾赫斯（Jorge Luis Borges）
- 翁貝托·艾柯（Umberto Eco）

這些作家  
會把這個想法  
推向更極端的地方。  

他們甚至會問：

```text
如果世界本身就是文本，
那麼真實還存在嗎？
```

這句話非常重要。  

當然，  
今天還不用急著回答。  
但一定要看到：  

```text
塞萬提斯打開的，
不是一個關於堂吉訶德個人錯覺的小問題，
而是一條關於閱讀、符號、意義與現實的現代思想道路。
```

這就是  
這一課真正的分量。

## 十五、文學開始研究閱讀：小說不只讓人讀，也開始把“讀”本身變成主題

這正是  
《堂吉訶德》的偉大之處。  

它不只是一本小說。  
它也是最早一批研究：

```text
讀者如何閱讀小說
```

的小說。  

因此：

```text
現代文學的自我意識開始出現。
```

這句話非常重要。  

也就是說，  
小說從這裡開始  
不只是讓人沉浸。  
它也會讓人停下來思考：

- 我在怎麼讀？
- 這個故事怎麼影響我？
- 我是不是也在誤讀世界？
- 我是否也像堂吉訶德一樣，被某種敘事塑造？

這正是  
現代閱讀經驗  
真正開始複雜化的地方。

## 十六、第四部分主線中的位置：命運不只與選擇有關，也與閱讀世界的方式有關

回到第四部分  
一路追問的核心問題：

```text
人是自己命運的主人嗎？
```

今天得到一個  
新的答案：

```text
人在做選擇之前，
必須先理解世界。
```

而理解世界的方式，  
來自：

- 故事
- 敘事
- 符號
- 解釋框架

因此：

```text
命運與閱讀開始相連。
```

這句話非常重要。  

也就是說，  
人不是先有  
完全中立的世界圖像，  
再在其上自由選擇。  
相反地，  
人總是先用某種方式  
把世界“讀成”某個樣子，  
然後才可能行動。  

因此第四部分  
到這裡已經更深入了。  
問題不再只是：

- 我能不能做主？

而是：

- 我是如何理解世界的？
- 我的閱讀框架從哪裡來？
- 我是否正在用錯誤的故事讀人生？

這正是  
《堂吉訶德》  
對整個主線  
最深的新貢獻。

## 十八、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果人從來不是赤裸地面對世界，
而總是透過符號、語言、故事與解釋來理解它，
```

這個問題現在不用急著回答。  
但它會直接把我們帶進：

- 為什麼後來的小說會越來越重視敘事結構與視角
- 為什麼閱讀、誤讀與詮釋會成為現代文學核心主題
- 為什麼從塞萬提斯開始，文學不只描寫世界，也描寫人如何讀世界
- 為什麼到了博爾赫斯（Borges）、艾柯（Eco）與後現代，文本與真實的界線會被推到幾乎無法穩定的位置

因為從今天開始，  
我們已經不只是在說  
故事影響現實。  

更準確地說：

```text
從今天開始，
正式看見現代文學的一個深層前提——
人之所以活在世界中，
其實也總是活在對世界的閱讀之中。
```
