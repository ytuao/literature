# Day 052｜《馬克白》
## Macbeth

從《哈姆雷特》（Hamlet）  
走到《馬克白》（Macbeth），  
幾乎像是從一個極端  
走到另一個極端。  

昨天遇見的是一個  
無法行動的人：

```text
哈姆雷特（Hamlet）
```

他的問題是：

```text
思考過度。
```

今天遇見的  
則是另一種完全相反的人：

```text
馬克白（Macbeth）
```

他的問題是：

```text
欲望過度。
```

這裡先補一個重要精度。  
把《馬克白》概括成  
“悲劇並非來自命運，而是來自選擇”，  
作為主線非常清楚。  
但若更嚴格，  
最好補上：

```text
全劇真正有力的地方，
正在於“預言、誘惑、想像、野心與選擇”彼此纏繞。
```

女巫（Witches）  
並沒有直接命令馬克白殺人。  
但她們確實：

- 激發了他
- 放大了他
- 讓他開始想像原本未被明說的可能性

也就是說，  
悲劇既不是純命運，  
也不是毫無外力介入的單純個人選擇。  
它更像是一場：

```text
外部誘惑與內部野心互相點燃的過程。
```

同樣，  
“人生不過是一個行走的影子”  
那段著名獨白，  
確實充滿虛無感。  
但也應該放回  
他在罪、恐懼與失落中  
被徹底耗盡的狀態來讀。  

這樣整部《馬克白》  
會更準確。  

---

## 一、一個英雄的開始：悲劇不是從邪惡開始，而是從誘惑開始

故事開始時，  
馬克白其實不是壞人。  

他是蘇格蘭的勇敢將軍，  
戰功卓著，  
受到國王信任。  

換句話說：

```text
他原本已經擁有榮譽。
```

這一點非常重要。  

因為《馬克白》的可怕之處  
恰恰在於：  

```text
悲劇不是從一個純粹邪惡的人開始。
```

而是從一個  
本來具有榮譽、能力與前途的人開始。  

因此這部作品  
真正要追問的  
不是“壞人為什麼做壞事”。  
而是：

```text
一個本來並不必然邪惡的人，
是如何一步步被誘惑、被想像、被野心改變的？
```

這就是它真正令人不安的地方。  

---

## 二、三個女巫：她們沒有命令殺人，卻點燃了想像

某天，  
馬克白遇見三位女巫：

```text
Witches（女巫）
```

她們預言：

```text
你將成為國王。
```

這句話改變了一切。  

但一定要注意一點：

```text
女巫沒有命令他殺人。
```

她們只是說出預言。  

這句話非常重要。  
因為《馬克白》的核心  
並不是“邪惡外力控制一切”。  
真正的問題在於：

```text
馬克白如何回應這個預言。
```

這也是全劇  
最深的一個問題之一。  

也就是說，  
女巫的作用  
不是直接代替他做決定。  
而是讓他開始：

- 想像本來沒有明說的未來
- 把可能性轉化成慾望
- 把慾望轉化成計畫

因此她們更像是  
悲劇的引爆點，  
而不是全部原因。  

---

## 三、命運還是選擇？預言只提供可能，選擇才把可能變成罪

這是全劇  
第一個重要問題：

```text
如果預言說你會成功，
你是否應該主動推動它發生？
```

或者，  
應該等待它自然發生？  

馬克白選擇了前者。  

於是：

```text
命運開始與意志糾纏在一起。
```

這句話非常重要。  

《馬克白》之所以深刻，  
就在於它不讓問題  
停在簡單二選一：  

- 不是完全由命運決定
- 也不是完全沒有命運誘惑

更準確地說，  
它呈現的是：

```text
命運提供想像，
意志決定回應方式。
```

也因此，  
真正造成悲劇的  
不是“預言存在”本身，  
而是：

```text
馬克白決定用暴力去把預言變成現實。
```

這一步，  
才是道德墮落  
真正開始的地方。  

---

## 四、野心的誕生：不是擁有太少，而是開始無法滿足於已有的一切

在遇見女巫之前，  
馬克白並沒有明確想當國王。  

但預言讓一個念頭  
突然誕生：

```text
如果我真的可以擁有更多呢？
```

這就是野心的起點。  

這句話非常重要。  

因為野心  
並不一定來自絕對匱乏。  
很多時候，  
它恰恰來自：

- 已經擁有很多
- 卻仍然無法滿足
- 一旦看到更高位置
- 就再也回不去原本的安定

因此野心最危險的地方  
不只是它想要更多。  
而是：

```text
它讓人開始否定“已經足夠”這件事。
```

而一旦這一步出現，  
原本的榮譽、忠誠與分寸  
就都可能被重新評估。  

這也是《馬克白》  
真正現代的一點：  

```text
悲劇往往不是從缺少開始，
而是從不再能滿足開始。
```

---

## 五、馬克白夫人：她不是原因的全部，但她讓內在野心快速成形

另一位極其重要的人物是：

```text
麥克白夫人（Lady Macbeth）
```

她讀到丈夫的信後，  
立刻支持奪權計畫。  

她認為：

```text
如果機會出現，
就應該抓住。
```

因此她成為  
推動悲劇的重要力量。  

這句話沒有錯，  
但也要補一個精度。  
麥克白夫人  
不是全部原因。  
她不是把一個完全無辜的人  
硬變成惡人。  

更準確地說，  
她的作用是：

- 催化
- 鼓動
- 去除猶豫
- 逼馬克白把模糊的慾望化成具體行動

也就是說，  
她讓馬克白內在  
原本就已被點燃的野心，  
迅速取得了行動形式。  

因此她是悲劇的重要推手，  
但悲劇的根  
仍然在馬克白自己身上。  

---

## 六、第一個罪：悲劇根源不是無知，而是明知故犯

馬克白最大的轉折點  
就是：

```text
刺殺國王鄧肯（Duncan）
```

這一刻極其重要。  

因為馬克白知道  
這是不對的。  
他不是不知道。  
他是：

```text
清楚知道，
卻仍然選擇去做。
```

這句話非常重要。  

因此《馬克白》的悲劇根源  
不是無知。  
而是：

```text
明知故犯。
```

這和很多單純的墮落故事  
很不一樣。  
因為莎士比亞關心的  
不是一個完全沒有道德感的人。  
而是一個：

- 理解善惡
- 知道界線
- 感受到罪的重量
- 卻還是跨過界線的人

這就使悲劇  
變得更加可怕。  
因為問題不再是：

```text
人不知道什麼是對的。
```

而是：

```text
人明明知道什麼是對的，
卻仍然可能背叛它。
```

---

## 七、犯罪之後：得到王位不是結束，而是恐懼的開始

很多人會以為：  
既然馬克白要的是王位，  
那麼得到王位之後，  
故事就該結束了。  

其實不是。  
真正的悲劇  
反而從這裡才開始。  

因為馬克白很快發現：

```text
權力無法消除恐懼。
```

反而會：

```text
放大恐懼。
```

這句話非常重要。  

一旦靠暴力取得權力，  
權力本身就會變得不穩。  
於是他開始害怕：

- 別人知道真相
- 別人奪走王位
- 預言有別的含義
- 自己永遠無法真正安全

因此他得到的  
不是安穩，  
而是：

```text
一個必須持續被暴力保護的位置。
```

這就是罪惡王權  
真正的邏輯。  

---

## 八、罪惡的邏輯：一個罪往往需要另一個罪來維持

《馬克白》中  
最殘酷的地方之一在於：

```text
一個罪，
往往需要另一個罪來掩蓋。
```

於是：

- 謀殺之後，還要繼續謀殺
- 欺騙之後，還要繼續欺騙
- 奪權之後，還要清除更多“威脅”

悲劇因此開始失控。  

這句話非常重要。  

因為罪在這裡  
不是單一事件。  
它是一種會自我繁殖的邏輯。  

一旦一個人  
決定用暴力與謊言  
來達成目的，  
那麼他通常就無法  
停在第一步。  

因此《馬克白》  
真正描寫的  
不只是一次犯罪。  
而是：

```text
人如何在第一次越界之後，
被迫進入越來越深的自我毀滅循環。
```

---

## 九、血的意象：真正洗不掉的不是血跡，而是罪與良知

《馬克白》中  
最重要的象徵之一，  
就是：

```text
血（Blood）
```

血代表：

- 罪惡（Crime）
- 良知（Conscience）
- 無法抹去的記憶（Unforgettable Memory）
- 持續回來的道德污染

馬克白殺死國王之後，  
不停想到  
自己手上的血。  

因為真正無法洗去的，  
不是血跡本身。  
而是：

```text
罪。
```

這句話非常重要。  

血之所以強大，  
就在於它讓“內在的罪”  
變得像“外在的污痕”一樣  
可見又可怕。  

也就是說，  
莎士比亞在這裡  
不是只用血來增加暴力感。  
他更是在說：

```text
道德行為會留下痕跡，
而這種痕跡未必能被外表清理掉。
```

因此血  
既是現場證據，  
也是心靈證據。  

---

## 十、良知沒有消失：真正可怕的不是無良知，而是帶著良知墮落

這是一個  
非常關鍵的點。  

馬克白不是毫無良知。  
恰恰相反，  
他一直知道  
自己做錯了。  

因此：

```text
他持續受到折磨。
```

這句話非常重要。  

莎士比亞真正關心的，  
不是單純的惡人。  
而是：

```text
一個有良知的人如何墮落。
```

這使得《馬克白》  
比一般道德寓言  
更深刻。  

因為如果角色  
只是純粹邪惡，  
那麼他不會讓人真正害怕。  
真正可怕的是：

- 他知道自己在做什麼
- 他感到罪
- 他甚至曾經想停下來
- 但最後還是一步步走下去

這就讓悲劇  
變成一種更深的問題：  

```text
良知本身，
並不保證人一定能阻止自己作惡。
```

---

## 十一、權力的悖論：得到王冠，卻失去平靜與自己

馬克白得到權力後，  
卻沒有真正得到  
他以為自己想要的東西。  

他：

- 得到王冠，卻失去平靜
- 得到地位，卻失去安全感
- 得到權力，卻失去自己

因此：

```text
權力沒有帶來自由。
```

反而帶來：

```text
囚禁。
```

這句話非常重要。  

因為在《馬克白》裡，  
權力不是一個完成狀態。  
它是一個需要時刻防守、  
時刻懷疑、  
時刻害怕被奪走的狀態。  

因此馬克白  
不是成為了主人。  
他反而越來越像：

```text
被自己奪來的權力反過來統治的人。
```

這就是權力悲劇  
最深的一層。  

---

## 十二、馬克白與哈姆雷特：一個做不到，一個不該做卻還是做了

把兩位主角  
放在一起比較，  
會看得更清楚。

### 哈姆雷特（Hamlet）
```text
知道該做什麼，
卻做不到。
```

### 馬克白（Macbeth）
```text
知道不該做什麼，
卻還是去做。
```

兩者都悲劇。  
但方向完全相反。  

這個對照  
非常重要。  

因為它讓人看見  
莎士比亞不是只會寫  
一種類型的內在衝突。  
他同時能寫：

- 思考過多導致的停滯
- 欲望過強導致的越界

也就是說，  
《哈姆雷特》與《馬克白》  
共同構成了一個  
非常完整的現代圖景：

```text
人既可能因為反思而無法行動，
也可能因為慾望而摧毀行動的道德界線。
```

這就是第四部分  
極其關鍵的一步。  

---

## 十三、世界開始失去秩序：弒君不只是一樁罪，也是整體失衡

在中世紀文學裡，  
宇宙通常有明確秩序。  

而在《馬克白》中，  
這種秩序被破壞。  

- 國王被殺
- 忠誠被背叛
- 王權正當性崩解
- 自然界也出現異象

彷彿整個世界  
都失衡了。  

這句話非常重要。  

因為弒君（Regicide）  
在這部作品裡  
不只是政治事件。  
它同時意味著：

```text
道德秩序與自然秩序一起被撕裂。
```

因此《馬克白》  
雖然已經非常現代，  
仍然保留著一種  
強烈的舊世界觀痕跡：  

```text
人的罪不只影響自己，
也會擾亂整個共同世界。
```

這就使它  
既是心理悲劇，  
也是秩序悲劇。  

---

## 十四、最著名的一段話：當一切都得到了，意義卻徹底崩塌

在接近結尾時，  
馬克白說出一段  
極其著名的獨白：

```text
人生不過是一個行走的影子。
```

這段話透露出  
非常深的虛無感。  

但這裡要讀得更準確。  
這不只是抽象哲學感嘆。  
它是來自一個  
已經在罪、恐懼、損失與孤立中  
被徹底耗盡的人。  

因此這段獨白的力量  
正在於：

```text
他得到了一切，
卻發現一切失去意義。
```

這句話非常重要。  

也就是說，  
虛無感在《馬克白》中  
不是起點。  
而是：

```text
野心走到盡頭之後的結果。
```

這就讓它格外可怕。  
因為它不是窮困者的虛無，  
而是成功者的虛無。  

---

## 十五、真正的悲劇：不是死亡，而是活著時逐漸失去自己的靈魂

很多人會以為  
馬克白的悲劇  
就是最後死去。  

其實不是。  
真正的悲劇在於：

```text
他在活著的時候，
逐漸失去自己的靈魂。
```

這句話非常重要。  

他不是在最後一刻  
才突然毀滅。  
他的毀滅  
早就在每一次選擇中  
慢慢發生了。  

因此《馬克白》  
本質上是一部：

```text
關於腐化（Corruption）的作品。
```

也就是說，  
它最可怕的不是  
“壞事發生了”。  
而是：

```text
一個人如何在一次次選擇中，
把自己變成自己原本不會想成為的人。
```

這才是它真正的悲劇性。  

---

## 十六、第四部分主線中的位置：自由存在，但自由完全可能被用來毀掉自己

回到第四部分  
到目前為止的核心問題：

```text
人是自己命運的主人嗎？
```

《馬克白》  
給出的回答是：

```text
是。
```

但接下來  
它立刻補上一個  
更可怕的條件：

```text
你可能利用自由毀掉自己。
```

這句話非常重要。  

因此《馬克白》提醒讀者：

- 自由不保證幸福
- 能選擇不代表會選對
- 有行動力不代表有道德方向
- 人完全可能主動走向自我毀滅

這就是它和《哈姆雷特》  
一起構成的深層對照：  

- 一個人因無法行動而悲劇
- 一個人因過度行動而悲劇

兩者都在說明：

```text
現代人的自由，
既是尊嚴，
也是危險。
```

## 十七、今日核心總結（Day 052）

今天請記住這幾句話：

```text
第一，
《馬克白》（Macbeth）探討的是野心、權力、罪、良知與道德墮落；
更準確地說，
它最有力之處在於呈現預言、誘惑、想像、野心與選擇如何彼此纏繞。
```

```text
第二，
故事一開始的馬克白不是純粹惡人，
而是一位有榮譽、受信任的勇敢將軍；
因此悲劇不是從天生邪惡開始，
而是從被點燃的可能性與被放大的慾望開始。
```

```text
第三，
女巫（Witches）並沒有直接命令馬克白殺人，
她們只是用預言激發並放大了他內在的野心；

是馬克白自己的回應與選擇。
```

```text
第四，
刺殺國王鄧肯（Duncan）是全劇的決定性越界；
悲劇根源不是無知，
而是明知故犯——
馬克白知道那是不對的，
卻仍然選擇跨過道德界線。
```

```text
第五，
得到王位之後，
悲劇才真正開始；
因為權力無法消除恐懼，
反而使人陷入更多謀殺、更多掩蓋與更深的囚禁。
```

```text
第六，
血（Blood）是全劇最重要的意象之一，
象徵罪惡、良知與無法洗去的記憶；
真正抹不掉的不是血跡，
而是道德行為在心靈中留下的痕跡。
```

```text
第七，
《馬克白》最終顯示：
人是可以主動選擇自己命運的，
但也正因如此，
人完全可能利用自由與野心一步步毀掉自己，
最後在權力、恐懼與虛無中失去靈魂。
```

## 十八、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果《馬克白》真正可怕的，
不是一個壞人成功奪權，
而是一個本來有榮譽、有良知的人，
如何在預言、想像、野心與選擇的糾纏中一步步腐化自己，
那麼接下來的現代文學，
是否會越來越把“惡”理解成一個過程——
不是天生本質，
而是人如何逐漸說服自己跨越界線的過程？
```

這個問題現在不用急著回答。  
但它會直接把我們帶進：

- 為什麼莎士比亞悲劇裡最可怕的往往不是壞人，而是會墮落的人
- 權力與自由為什麼會同時帶來恐懼與自我毀滅
- 為什麼現代文學越來越關心罪、良知與心理腐化
- 《李爾王》（King Lear）又將如何把“失去自己”推向另一個方向

因為從今天開始，  
我們看到的已經不只是一場王位陰謀。  

更準確地說：

```text
從今天開始，
正式看見文學如何把“人如何一步步說服自己作惡”
寫成一部關於自由、權力與靈魂腐化的悲劇。
```
