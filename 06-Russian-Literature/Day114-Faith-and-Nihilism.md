# Day 114｜信仰與虛無主義
## Faith and Nihilism

今天要討論：

```text
Faith and Nihilism
（信仰與虛無主義）
```

---

## 一、當舊答案開始失效：十九世紀的思想危機從這裡開始

在古代世界，  
很多人生問題  
都有現成答案。  

例如：

- 為什麼活著？因為神創造了世界
- 什麼是善？因為宗教告訴我們
- 如何生活？因為傳統已經規定

然而到了十九世紀，  
這些答案開始動搖。  

因為：

- 科學快速發展
- 工業革命改變社會
- 傳統宗教權威下降
- 舊有秩序逐漸瓦解

於是人們開始問：

```text
如果舊答案不再可信，
那麼新的答案在哪裡？
```

這句話非常重要。  

因為從這裡開始，  
問題不再只是  
“哪一個答案比較好”。  
而是：

```text
是否還有答案？
```

---

## 二、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
虛無主義最可怕的地方，
不是相信錯誤的答案，
而是不再相信任何答案。
```

這就是十九世紀  
思想危機的核心。  

但若更嚴格，  
最好補上：

```text
虛無主義真正危險之處，
不只是否定既有信仰，
而是在否定終極意義、穩定價值與道德根據之後，
讓人失去判斷、行動、承受痛苦與繼續生活的內在理由。
```

這句話非常重要。  

也就是說，  
虛無主義真正可怕的  
不只是“我不信這個”。  
而是一步步走到：

- 沒有什麼值得相信
- 沒有什麼值得追求
- 沒有什麼真正重要
- 沒有什麼能替人生提供方向

從這裡開始，  
問題就不再只是  
哲學觀點。  
而是：

```text
生存問題。
```

## 三、什麼是虛無主義？它看起來像自由，實際上可能導向崩解

簡單來說：

```text
Nihilism
（虛無主義）
```

認為：

- 沒有絕對真理
- 沒有終極意義
- 沒有固定價值
- 沒有神聖基礎

乍聽之下，  
這似乎代表  
完全自由：

- 沒有束縛
- 沒有權威
- 沒有規則

但問題  
很快就出現。  

如果什麼都不相信，  
那麼：

- 為什麼做好事？
- 為什麼遵守道德？
- 為什麼幫助別人？
- 為什麼活下去？

這句話非常重要。  

也就是說，  
虛無主義之所以  
不是單純“思想解放”，  
就在於它會動搖  
人最基本的生活結構。  

因為人不是  
只靠揭穿幻覺  
就能活下去。  
人還需要：

```text
方向。
```

## 四、陀思妥耶夫斯基的警告：現代人失去的不只是信仰，而是意義

```text
費奧多爾·陀思妥耶夫斯基
（Fyodor Dostoevsky）
```

是最早看見  
這場危機的人之一。  

他發現：

```text
現代人不再只是失去信仰，
而是失去意義。
```

這句話非常重要。  

也就是說，  
問題不只是  
“我還信不信上帝”。  
而是：

- 如果世界沒有最後根據，我要靠什麼活？
- 如果善惡不再穩固，我要怎麼判斷？
- 如果痛苦沒有意義，我要怎麼承受？

這正是  
陀思妥耶夫斯基  
真正比許多人更早看見的事。  

他看到的是：  

```text
現代危機的核心，
不是知識不足，
而是意義崩塌。
```

## 五、伊凡的問題：懷疑不是問題，徹底否定才是問題

在：

```text
《卡拉馬助夫兄弟》
（The Brothers Karamazov）
```

裡，

```text
伊凡·卡拉馬助夫
（Ivan Karamazov）
```

不斷質疑：

- 如果世界充滿苦難，我為什麼要相信善？
- 如果無辜的人受苦，正義真的存在嗎？

這句話非常重要。  

但值得注意的是，  
陀思妥耶夫斯基  
並不反對懷疑。  
事實上，  
他認為：

```text
真正的信仰必須經過懷疑。
```

盲目相信，  
並不是成熟的信仰。  

問題在於  
另一件事：  

```text
當一個人開始認為，
一切都沒有意義，
所有價值都只是幻覺，
所有理想都是欺騙，
那麼虛無主義就出現了。
```

也就是說，  
懷疑本身  
未必摧毀信仰。  
但徹底否定  
會摧毀人繼續活下去的理由。

## 六、為什麼虛無主義吸引人？因為它看起來比空洞樂觀更誠實

虛無主義  
之所以會吸引人，  
有一個很重要的原因。  

它看起來  
非常誠實。  

它會拒絕：

- 自我欺騙
- 空洞口號
- 沒有證據的信念
- 表面正能量式的安慰

因此，  
許多聰明人  
會被它吸引。  

這句話非常重要。  

因為虛無主義  
並不總是以  
粗暴形式出現。  
它有時反而帶著  
某種冷靜、清醒與批判性。  

但它真正的危險  
就在於：  

如果一切都沒有意義，  
那麼：

- 善與惡有差別嗎？
- 愛與恨有差別嗎？
- 創造與毀滅有差別嗎？

從這裡開始，  
虛無主義就不再只是  
“拒絕假答案”。  
它可能進一步變成：

```text
拒絕一切價值。
```

## 七、陀思妥耶夫斯基的觀察：人無法長期活在完全虛無之中

陀思妥耶夫斯基  
有一個非常深刻的觀察：

```text
人不可能長期活在完全虛無之中。
```

因為人需要：

- 意義
- 方向
- 價值

這句話非常重要。  

也就是說，  
人也許可以短暫地  
拆毀一切。  
但如果長期  
什麼都不相信，  
那麼他很可能會：

- 陷入冷漠
- 陷入絕望
- 陷入自我毀滅
- 陷入對他人與世界的斷裂

因此陀思妥耶夫斯基  
真正擔心的  
不是單一的思想錯誤。  
而是：

```text
當人失去所有意義支點時，
靈魂會往哪裡掉下去。
```

## 八、信仰是什麼？它不只是宗教，也是一種對價值與方向的承認

在這裡，  

```text
信仰
（Faith）
```

不一定只指宗教。  

更廣泛地說，  
信仰是：

- 相信某些事值得追求
- 相信某些價值重要
- 相信人生具有方向

這句話非常重要。  

但若再說得更準確一些，  
在俄羅斯文學的語境裡，  
信仰常常不是  
幼稚地拒絕懷疑。  
而是：

```text
在經過懷疑、痛苦與意義危機之後，
仍然選擇承認某些價值值得忠誠、
某些生活值得承擔、
某些善值得相信。
```

也就是說，  
信仰的反面  
不只是“不信宗教”。  
更深的反面  
其實是：

```text
再也不相信任何事值得認真活。
```

## 九、托爾斯泰的危機：當擁有成功仍然感到人生沒有意義

```text
列夫·托爾斯泰
（Leo Tolstoy）
```

晚年曾陷入  
深刻絕望。  

他擁有：

- 財富
- 名聲
- 成功

卻仍然問：

```text
人生有什麼意義？
```

這句話非常重要。  

托爾斯泰發現：  

```text
如果生命最終只是死亡，
那麼許多努力似乎毫無意義。
```

這是一種  
非常典型的存在危機。  

也就是說，  
托爾斯泰的問題  
不是“我怎樣更成功”。  
而是：

- 如果終點都是死亡，努力還有什麼價值？
- 如果一切都會消失，我為什麼要活？
- 如果理性說不出最後答案，人還能靠什麼站住？

這使他與  
陀思妥耶夫斯基  
在另一條路上  
相遇了。  

---

## 十、他找到的答案：意義不一定來自偉大理論，而可能來自可活出的善

後來，  
托爾斯泰逐漸相信：

```text
意義不一定來自偉大理論。
```

而可能來自：

- 愛
- 勞動
- 家庭
- 善待他人

這句話非常重要。  

也就是說，  
托爾斯泰後來  
不再把人生意義  
寄託在抽象成就上。  
他越來越相信  
真正能支撐生命的，  
常常是那些  
看起來不宏大  
卻能真正活出的東西。  

因此俄羅斯文學  
對“信仰”的理解  
並不總是只停在教義。  
它也包括：

```text
一種能讓人繼續生活的價值實踐。
```

## 十一、信仰與理性：理性重要，但它不能獨自回答所有問題

俄羅斯文學  
有一個非常重要的特點。  

它並不認為：

```text
理性毫無價值。
```

但它也不認為：

```text
理性能回答所有問題。
```

這句話非常重要。  

因為有些問題  
確實超越純粹邏輯。  
例如：

- 愛
- 犧牲
- 希望

這並不是說  
理性應該被丟棄。  
而是說：

```text
人生最深的問題，
常常不是只靠推理就能解決。
```

也因此，  
信仰與虛無主義的衝突，  
從來不只是  
“宗教對反宗教”的爭論。  
更是：

```text
人能不能在沒有最終證明的情況下，
仍然忠於某些價值。
```

## 十二、現代人的處境：資訊更多了，但人生意義未必更清楚

今天仍然面對  
同樣的問題。  

現在：

- 傳統價值持續受到挑戰
- 資訊越來越多
- 選擇越來越多

但人生意義  
未必更清楚。  

這句話非常重要。  

也就是說，  
現代處境  
並不只是“知道得更多”。  
而是：

- 標準更分散
- 權威更鬆動
- 選擇更多
- 但方向未必更明確

因此：

```text
信仰與虛無主義的衝突，
今天仍然存在。
```

如果把整個課程  
一路回看，  
從神話開始，  
人類一直都在  
尋找意義。  

而到了俄羅斯文學，  
問題被推到最尖銳的形式：

```text
如果舊意義崩潰，
新的意義從哪裡來？
```

這正是  
Day 114  
在整個課程中的位置。

## 十四、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果虛無主義真正可怕的地方，
不是“答案錯了”，
而是“再也不相信任何答案值得信”，
那麼接下來閱讀俄羅斯文學時，
是否最重要的能力之一，
就是不只問“這個角色信不信上帝”，
而要問“當他失去舊答案之後，
還有沒有什麼能支撐他繼續判斷、繼續愛、繼續承受痛苦、繼續活下去”，
以及“真正成熟的信仰，
是否其實不是逃避懷疑，
而是在穿過懷疑之後，
仍然願意對某些價值保持忠誠”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看角色有沒有宗教信仰
- 不只看他說了什麼理論
- 不只看他是否反叛傳統

而是進一步去看：

- 他失去了什麼意義支點
- 他如何面對“沒有答案”的狀態
- 他的懷疑會導向更深信仰還是虛無
- 他最後靠什麼繼續活下去
- 他是否仍承認某些事值得相信與承擔

因為從今天開始，  
文學不再只是  
描寫“人相信什麼教義”。  

更準確地說：

```text
從今天開始，
正式把“信仰與虛無主義”理解成現代文學面對意義危機時最核心的一場對決——
在這裡，
真正的問題不只是宗教是否成立，
而是當舊答案崩潰之後，
人是否還能找到足以支撐判斷、道德、愛與生命方向的內在根據；
而俄羅斯文學之所以偉大，
正因為它比許多文學更早、更深地看見：
人最可怕的危機，
往往不是痛苦本身，
而是痛苦發生時，
再也不知道它是否仍有意義。
```
