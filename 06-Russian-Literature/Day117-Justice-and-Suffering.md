# Day 117｜正義與苦難
## Justice and Suffering

今天要討論：

```text
Justice and Suffering
（正義與苦難）
```

## 一、世界真的公平嗎？這是幾乎所有文明都會反覆追問的問題

這是一個  
幾乎所有文明  
都會問的問題：

- 為什麼壞事會發生在好人身上？
- 為什麼無辜的人會受苦？
- 如果世界是有秩序的，為什麼看起來如此不公平？

在日常生活中，  
人很早就會發現：

- 努力不一定有回報
- 善良不一定被獎勵
- 邪惡不一定被懲罰

這讓：

```text
正義
```

變得非常複雜。  

這句話非常重要。  

因為從這裡開始，  
正義就不再只是  
法律制度問題。  
它也變成：

```text
存在問題。
```

---

## 二、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
苦難之所以成為哲學問題，
是因為它挑戰了我們對正義的基本信念。
```

這是俄羅斯文學  
最核心的問題之一。  

但若更嚴格，  
最好補上：

```text
苦難之所以構成真正的哲學與文學問題，
不只是因為它讓人痛，
而是因為它迫使人追問：
如果善惡報應並不對稱、
無辜者仍會受害、
世界秩序看起來並不保障公平，
那麼正義究竟還有沒有根基。
```

這句話非常重要。  

也就是說，  
真正令人震動的  
不只是“世界上有痛苦”。  
而是：

- 痛苦似乎常常落在不該承受的人身上
- 世界似乎不按道德直覺運作
- 正義似乎沒有自動實現

從這裡開始，  
苦難就不只是  
個人經驗。  
它同時也是：

```text
對世界秩序的質疑。
```

## 三、陀思妥耶夫斯基的起點：如果世界不公正，人還能相信什麼？

```text
費奧多爾·陀思妥耶夫斯基
（Fyodor Dostoevsky）
```

幾乎所有重要作品，  
都圍繞著一個  
非常核心的問題：

```text
如果世界不公正，
人還能相信什麼？
```

這句話非常重要。  

這不只是宗教問題。  
也不只是哲學問題。  
它同時關係到：

- 道德是否還有基礎
- 人是否還能相信善
- 苦難是否會徹底摧毀信仰
- 當正義不出現時，人會變成什麼樣子

也就是說，  
陀思妥耶夫斯基  
真正關心的，  
不只是痛苦本身。  
而是：

```text
痛苦如何動搖整個人的信念世界。
```

## 四、伊凡的挑戰：無辜者的痛苦為何不能被輕易合理化？

在：

```text
《卡拉馬助夫兄弟》
（The Brothers Karamazov）
```

裡，

```text
伊凡·卡拉馬助夫
（Ivan Karamazov）
```

提出最尖銳的質疑。  

他不是單純否定  
上帝存在。  
他真正質疑的是：

```text
如果上帝存在，
為什麼允許無辜的苦難？
```

尤其是：

```text
孩子的痛苦。
```

這句話非常重要。  

伊凡之所以  
特別強調孩子，  
是因為這種痛苦  
最難被合理化。  

因為孩子：

- 沒有罪
- 沒有選擇
- 沒有能力理解自己為何受苦

很多哲學  
會試圖說：

- 苦難是為了更大的善
- 最終一切會被平衡

但伊凡拒絕  
這種回答。  
因為在他看來：

```text
沒有任何未來的幸福，
可以合理化無辜的痛苦。
```

這正是  
他質問世界與上帝  
最可怕的地方。

## 五、苦難若無法被解釋，會不會導向道德崩潰？

如果苦難  
無法被合理解釋，  
那麼會發生什麼？  

一條非常危險的路  
就可能出現：

- 沒有正義
- 沒有道德基礎
- 一切都只是力量問題

這句話非常重要。  

也就是說，  
當人開始相信  
世界從根本上  
毫無公平可言時，  
很容易進一步認為：

```text
那麼善惡也不過如此。
```

這正是  
虛無主義  
與苦難問題  
會碰在一起的地方。  

因為若無辜者的痛苦  
完全沒有意義，  
人可能會問：

- 為什麼還要相信正義？
- 為什麼還要選擇善？
- 為什麼不乾脆承認一切只是殘酷現實？

這就是  
苦難問題  
最危險的一面。

## 六、但俄羅斯文學沒有停在絕望：即使世界不公平，人仍可以選擇善

不過，  
俄羅斯文學  
並沒有停在絕望。  

它同時提出  
另一種可能：

```text
即使世界不公平，
人仍然可以選擇善。
```

這句話非常重要。  

這裡的意思  
不是說世界  
終將自動變公平。  
而是說：

```text
正義未必是世界保證的結果，
卻仍可以是人主動實踐的選擇。
```

也就是說，  
俄羅斯文學  
常把正義  
從“外部必然會實現的秩序”，  
轉化為：

- 一種行動方式
- 一種道德姿態
- 一種在苦難面前仍不放棄的責任

這個轉變  
非常關鍵。

## 七、托爾斯泰的視角：苦難不只摧毀人，也可能打破幻覺

```text
列夫·托爾斯泰
（Leo Tolstoy）
```

在：

```text
《戰爭與和平》
（War and Peace）
```

與

```text
《安娜·卡列尼娜》
（Anna Karenina）
```

中，  
並不迴避苦難。  

但他關心的是：

```text
人在苦難中如何生活？
```

這句話非常重要。  

托爾斯泰觀察到：

```text
苦難常常讓人更接近真實。
```

因為它會打破  
很多原本的幻想：

- 生命並不穩定
- 計畫並不可靠
- 名聲未必重要
- 成功未必能支撐靈魂

也就是說，  
苦難在托爾斯泰那裡  
雖然令人痛苦，  
卻也可能成為  
一種清醒力量。  

它讓人重新看見：

```text
生命真正重要的是什麼。
```

## 八、道德的基礎在哪裡？不是因為一定有回報，而是因為人仍然有良知

這裡就來到  
最核心的問題：

```text
如果世界不保證公平，
那麼我們為什麼還要做正確的事？
```

俄羅斯文學  
通常不會回答：

```text
因為你一定會得到回報。
```

相反地，  
它更常回答：

```text
因為這是人的選擇。
```

這句話非常重要。  

也就是說，  
道德的基礎  
未必來自  
外在獎懲系統。  
它也可能來自  
人的內在世界。  

即使沒有外在回報，  
人仍然會感到：

- 同情
- 愧疚
- 責任

而這些  
正構成了道德的  
內在基礎。  

因此正義  
在這裡不是  
“做對事就一定贏”。  
而是：

```text
即使不保證贏，
仍然不放棄做對的事。
```

## 九、拉斯柯爾尼科夫的崩潰：人自身就是道德的見證者

在：

```text
《罪與罰》
（Crime and Punishment）
```

中，

```text
拉斯柯爾尼科夫
（Rodion Raskolnikov）
```

試圖證明：

```text
某些人可以超越道德。
```

但最終，  
他的內心  
無法接受這個理論。  

即使法律  
一時無法懲罰他，  
他的內心  
仍然崩潰。  

這句話非常重要。  

這顯示出一個  
極重要的現象：

```text
人自身就是道德的見證者。
```

也就是說，  
即使世界外部  
沒有立刻實現正義，  
人的內心  
仍可能保存一種  
更深的審判能力。  

這也是俄羅斯文學  
對道德最深的信心之一：  

- 世界可能不公平
- 制度可能失效
- 報應可能不即時

但良知  
仍可能存在。

## 十、苦難的雙重性：它既可能摧毀人，也可能喚醒人

俄羅斯文學  
同時看到兩件事：

- 苦難可以摧毀人
- 苦難也可以喚醒人

這句話非常重要。  

也就是說，  
苦難本身  
並不自動高尚。  
它不是天然  
就能帶來成長。  
有時它確實會：

- 使人絕望
- 使人憤怒
- 使人冷漠
- 使人崩潰

但另一方面，  
若人能以某種方式  
回應苦難，  
它也可能：

- 打破虛假自我
- 喚醒同情
- 促成懺悔
- 逼出更真實的道德選擇

因此真正關鍵的  
不是“苦難本身好不好”。  
而是：

```text
人如何回應苦難。
```

## 十一、正義不是結果，而是一種態度與責任

如果世界本身  
不保證正義，  
那麼：

```text
正義
```

就不再只是  
某個最終會出現的結果。  

它更變成：

- 一種行動方式
- 一種選擇
- 一種責任

這句話非常重要。  

也就是說，  
俄羅斯文學  
在這裡通常不是  
保證人一定得到公平。  
而是要求人：

- 不因世界的不公而放棄善
- 不因苦難的無解而放棄同情
- 不因報應不對稱而放棄良知

這使正義  
從一個外部制度目標，  
深化為：

```text
一種存在姿態。
```

因此放回整個課程來看，  
就會明白：

- 從命運到自由
- 從倫理到信仰

一路走到這裡，  
終於看見：

```text
正義不只是制度問題，
也是存在問題。
```

## 十二、現代延伸：這個問題今天仍然沒有消失

這些問題  
到今天仍然存在。  

例如：

- 社會不平等
- 戰爭與衝突
- 個人不幸
- 無辜者承受代價

人仍然會問：

- 世界是否應該更公平？
- 人應該如何回應不公平？

這句話非常重要。  

也就是說，  
Day 117  
並不是在談  
十九世紀才有的問題。  
它其實是在談  
一個直到現在  
都沒有消失的核心困境：

```text
當世界不符合正義，
人是否仍能活出正義？
```

這正是  
今天這一課  
最重要的地方。

## 十四、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果苦難真正可怕的地方，
不是“它讓人痛”這麼簡單，
而是“它讓人開始懷疑世界是否真的有正義”，
那麼接下來閱讀俄羅斯文學時，
是否最重要的能力之一，
就是不只問“角色受了多少苦”，
而要問“這些苦難如何動搖了他的信念、正義感與對世界秩序的理解”，
以及“當世界沒有保證公平時，
他究竟會因此滑向冷漠、犬儒與虛無，
還是仍然選擇以良知、責任與同情回應這個不公平的世界”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看苦難有多慘烈
- 不只看角色是否受害
- 不只看情節是否悲傷

而是進一步去看：

- 苦難如何挑戰他的正義信念
- 他是否試圖合理化無辜者的痛苦
- 他是否因此失去道德方向
- 他是否仍能維持良知與同情
- 他最後把正義理解成什麼

因為從今天開始，  
文學不再只是  
描寫“誰遭遇了不幸”。  

更準確地說：

```text
從今天開始，
正式把“正義與苦難”理解成俄羅斯文學對世界秩序的一次最深審問——
在這裡，
最關鍵的已不只是苦難有沒有發生，
而是當苦難落在無辜者身上時，
人是否還能相信正義、如何重新理解道德，
以及在世界不保證公平的情況下，
是否仍然願意把自己活成正義的一部分；
而這也正是俄羅斯文學最震撼人的地方之一：
它不輕易替苦難找藉口，
卻仍然逼人去思考，
在不公平面前，
人究竟還能選擇成為什麼樣的人。
```
