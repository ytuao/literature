# Day 157｜媒介與現實
## Media and Reality

今天我們進入  
後現代一個非常現實、  
也非常當代的問題：

```text
Media and Reality
（媒介與現實）
```

如果前面我們已經接受：

- 身份是建構的
- 歷史是文本
- 真理依賴敘事
- 世界透過語言被理解

那現在要問的  
就是更直接的一步：

```text
如果我們接觸世界
總要透過媒介，
那麼我們所謂的“現實”
是否早已不是未加工的世界，
而是被轉譯、被編排、被格式化之後的版本？
```

---

## 一、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
我們所謂的“現實”，
很大程度是透過媒介
被製造出來的現實感。
```

這句話非常重要。  

但若更嚴格，  
最好補上：

```text
後現代媒介理論
通常不是簡單說物理世界不存在，
而是指出：
在媒介高度滲透的社會裡，
人們對“現實”的理解、
重要性排序與記憶形狀，
越來越依賴媒介的選擇、格式、
可見性機制與再現方式。
```

這句話非常重要。  

同時也最好再補一句：

```text
“在媒介社會中，
存在越來越等同於被呈現”
這句話最好理解成
公共現實層面的“存在”；
也就是一件事若沒有被報導、
記錄、轉發或影像化，
它未必在物理上不存在，
但它很可能在公共認知、
集體記憶與社會注意力中
近乎等於沒有發生；
這也是媒介權力如此核心的原因。
```

也就是說，  
今天真正要問的  
不是“世界有沒有存在”。  
而是：

```text
世界是如何透過媒介
變成我們所感到的樣子。
```

## 二、傳統觀點：媒介被當成透明通道，只負責把現實送到你面前

你原文中  
這個傳統模型很清楚：

```text
現實 → 媒介 → 你
```

並且假設：

- 媒介只是傳遞工具
- 不改變內容本質

這句話非常重要。  

這其實是很多人  
直到今天  
仍然會直覺採用的想法。  

也就是說，  
現實好像先完整在那裡，  
媒介只是一條管道，  
負責把它送到你眼前。  

在這種理解裡，  
報紙、電視、手機、社群平台  
都只是不同材質的容器。  
它們不應該  
根本改變內容。  

但後現代真正要挑戰的，  
就是這種  
“媒介透明論”。

## 三、後現代的轉折：媒介不是透明的，它本身會改變現實的樣子

你原文裡  
這一句非常關鍵：

```text
後現代認為：
媒介本身會改變“現實的樣子”。
```

這句話非常重要。  

這幾乎就是  
Day 157 的起點。  

因為只要媒介  
不是透明的，  
那麼問題就不只是  
它有沒有把事情報導正確。  
更深一層的問題會變成：

```text
媒介用什麼形式
讓某種東西看起來像現實？
```

也就是說，  
媒介不只是  
把現實傳過來。  
它也會透過：

- 取景
- 剪輯
- 節奏
- 版面
- 標題
- 演算法排序
- 視覺格式

去塑造  
現實被感知的方式。  

因此現實在媒介裡  
不是原封不動地出現。  
它總是已經被處理。

## 四、你看到的不是世界本身，而是被格式化的世界

你原文中  
這一點非常重要：

```text
你看到的不是世界，
而是“被格式化的世界”
```

你也列得很好：

- 新聞標題
- 社群貼文
- 影像剪輯
- 演算法推薦

這句話非常值得保留。  

因為今天的現實感  
很大一部分  
就是由格式構成的。  

也就是說，  
同一件事情  
一旦被做成：

- 短影片
- 熱搜詞條
- 聳動標題
- 資訊圖卡
- 演算法推播

它就不只是  
換了一個包裝。  
它連“被理解的樣子”  
都變了。  

所以我們接觸的  
往往不是世界本身。  
而是世界被平台格式  
切割、加速、  
情緒化、可分享化之後的版本。

## 五、現實開始依賴可見性：在媒介社會裡，存在越來越等於被看見

你原文裡  
這一句非常有力：

```text
在媒介社會中：
存在 = 被看見
```

這句話非常重要。  

但一定要理解成  
公共存在，  
而不是物理存在。  

也就是說，  
一件事情  
若沒有進入可見系統，  
它未必不存在。  
但它很難進入：

- 公共討論
- 社會記憶
- 集體情緒
- 政治優先級
- 現實感排序

因此“被看見”  
不只是曝光。  
它也等於  
獲得現實地位。  

這就是為什麼  
可見性在媒介社會中  
會變成權力問題。

## 六、沒有被呈現，就近乎在公共意義上消失

你原文中  
這一步非常關鍵：

```text
如果某件事：
沒有被報導
沒有被分享
沒有被記錄

它在公共意義上就“消失”。
```

這句話非常重要。  

這裡的“消失”  
不是說事情沒發生。  
而是說：

```text
它沒有被納入
社會共同感知的現實。
```

也就是說，  
很多事件的社會重量  
並不只取決於  
事件本身多大。  
還取決於它是否被：

- 命名
- 拍到
- 報導
- 推送
- 轉發
- 反覆再現

因此媒介  
不只是分配資訊。  
它也在分配：

```text
什麼算一件“有社會存在感”的事。
```

## 七、媒介不是鏡子，而是濾鏡：它在選擇、剪輯與排序現實

你原文裡  
這一句非常關鍵：

```text
媒介不只是反映世界，
而是：
選擇、剪輯、排序現實。
```

這句話非常值得保留。  

這幾乎可以說是  
Day 157 最實用的一句話。  

因為只要理解這一點，  
你就能立刻看見  
媒介的三層作用：

- 選擇什麼進來
- 決定什麼放大
- 安排你先看到什麼、後看到什麼

也就是說，  
媒介不是站在世界前面  
舉一面鏡子。  
它更像一個  
帶有偏向的編輯系統。  

而“偏向”  
不一定總是陰謀。  
它也可能只是  
格式限制、商業邏輯、  
平台規則與注意力競爭  
共同造成的結果。

## 八、現實變成可編輯內容：世界被轉化為片段、影像、敘事與標題

你原文中  
這一點非常重要：

```text
世界被轉化為：
片段
影像
敘事
標題
```

這句話非常值得保留。  

這正是媒介化現實  
最日常也最深刻的一個改變。  

因為在媒介環境裡，  
世界越來越不是  
以完整連續的形式  
向我們出現。  
它更常是：

- 一段剪好的畫面
- 一句可以轉發的話

- 一個立場鮮明的摘要

也就是說，  
現實要進入媒介，  
往往先要被內容化。  

而一旦內容化，  
它就同時被：

- 壓縮
- 格式化
- 可消費化
- 可流通化

這會直接改變  
我們對現實的感覺方式。

## 十、注意力變成資源：誰獲得注意力，誰就獲得現實權重

你原文中  
這一句非常關鍵：

```text
在媒介環境中：
注意力 = 現實權重
```

這句話非常重要。  

這可以說是  
媒介社會最冷酷的一條規則。  

因為事件的“重要性”  
越來越不只是  
由內在意義決定。  
它也由：

- 點擊量
- 轉發量
- 停留時間
- 推薦權重
- 討論熱度
- 影像衝擊力

來決定。  

也就是說，  
注意力不是中性資源。  
它是現實排序機制。  

一件事情  
得到越多注意力，  
它就越像“真正重要的現實”。  
反過來，  
沒有注意力的事  
即使非常重要，  
也可能被公共世界當成不存在。

在媒介時代，

```text
現實如何被社會想像。
```

## 十三、事件不再完整出現：我們永遠只看到被選擇與裁切後的版本

```text
我們永遠看到：
選擇後的版本
```

媒介現實的根本條件。

媒介的問題  
不只是偶爾失真。  
而是：

```text
它的呈現本質
就不可能完整。
```

所以後現代關心的  
不是如何回到  
“完全未中介的真實”。  
而是要讓你看見：

```text
你眼前這個版本
是怎麼被做出來的。
```


這句話非常值得保留。  

但這裡最好理解成  
公共真實感，


這句話非常重要。  

也就是說，  
格式本身  
正在參與真實感的生產。  

這會讓我們誤以為：  
看得到、能截圖、  
能傳播、能量化的，  
才比較真。  
而那些無法被格式捕捉的經驗，  
則更容易被邊緣化。

```text
人們對“現實”的理解、
重要性排序與記憶形狀，
越來越依賴媒介的選擇、格式、
可見性機制與再現方式。
```

```text
第二，
媒介不只是傳遞信息，
而是在塑造什麼被視為現實；
它不只是鏡子，
更是濾鏡、編輯器與排序機制，
會透過選擇、剪輯、放大與重新定義
改變世界在我們眼前出現的方式。
```

```text
第三，
在媒介社會中，
“存在”越來越等同於“被呈現”；
但這裡最好理解成公共現實層面的“存在”——
也就是一件事若沒有被報導、
記錄、轉發或影像化，
它未必在物理上不存在，
但它很可能在公共認知、
集體記憶與社會注意力中
近乎等於沒有發生。
```

```text
第四，
媒介社會最重要的變化之一，
是現實開始依賴可見性與注意力；
誰能獲得曝光、
誰能被格式化成可流通內容、
誰能進入熱點與記憶，
誰就更可能被當成“真正的現實”。
```

```text
第五，
我們越來越生活在“二次現實”中；
也就是除了物理世界之外，
還有一個被媒介轉譯、
剪輯、平台化與演算法排序過的世界，
而這個世界往往更接近我們的公共經驗。
```

```text
第六，
後現代關於媒介的核心提醒，
不是要我們否認世界，
而是要我們看見：
所謂“現實感”
本身就是一種可以被生產、放大與管理的效果。
```

```text
第七，
在整個課程中，
Day 157 代表後現代問題正式進入媒介理論層面：
從存在主義的“人在世界中創造意義”，
走到後現代的“世界透過語言被建構”，
再走到“現實透過媒介被生成”的核心轉折；
```

## 二十一、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果後現代真正要指出的，
不是“世界根本不存在”這麼簡單，
而是“我們對世界的公共理解、
重要性排序與集體記憶，
越來越依賴媒介的選擇、格式與可見性機制”，
那麼接下來閱讀媒介理論時，
是否最重要的能力之一，
就是不再只問“發生了什麼”，
而要問“這件事是以什麼形式被呈現、
為什麼是這個版本被看見、
它又是如何被平台、影像、標題與分享邏輯
塑造成現實感的”；
以及，
如果公共現實越來越離不開媒介，
那麼“看見世界”
是否也必須被重新理解成：
在無數被格式化、被排序、被轉譯的版本之間
不斷辨識、比較與拆解的過程？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看媒介是不是在傳遞信息
- 不只看畫面是否忠實呈現事件
- 不只看真實是否還有單一來源

而是進一步去看：

- 媒介如何生產現實感
- 可見性如何變成公共存在條件
- 注意力如何成為現實排序機制
- 為什麼記憶會越來越依賴媒介格式
- 影像、敘事與事件為何開始彼此難分

因為從今天開始，  
後現代已經不再只是  
在談文本、歷史與身份如何被建構。  

更準確地說：

```text
從今天開始，
正式把“媒介與現實”理解成後現代對當代世界感的一次深層揭示——
在這裡，
真正重要的已不再只是語言如何建構世界、
敘事如何競爭真理、
歷史如何被文本化、
身份如何被社會生成，
而是當我們開始意識到：
連“現實本身”作為公共經驗的樣子，
都越來越不是直接呈現給我們的，
而是透過新聞格式、社群平台、影像剪輯、
演算法排序、標題邏輯與可見性機制
被選擇、被放大、被壓縮、
被重新定義之後才進入我們的感知時，
那麼“現實”
就不再能被理解成一個單純等待被觀看的對象；
它更像是一種在媒介環境中
被不斷轉譯、編輯、再現與記憶化的效果；
而後現代之所以重要，
正因為它讓我們看到：
我們面對的，
常常不是未被觸碰的世界，
而是被媒介製造成可感知版本的世界。
```
