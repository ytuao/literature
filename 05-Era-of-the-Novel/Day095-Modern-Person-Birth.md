# Day 095｜現代人的誕生
## Modern Person Birth

今天要討論：

```text
Modern Person Birth
（現代人的誕生）
```

每天都會使用：

```text
現代人
```

這個詞。  
但其實很少停下來思考：

- 什麼叫現代人？
- 現代人與古代人，真的只是生活在不同時代嗎？
- 還是有更深層的差異？

這裡先補一個重要精度。  
把“現代人的誕生，來自於『自我』成為問題”  
作為核心句，  
非常準確。  
但若更嚴格，  
最好補上：

```text
所謂“自我成為問題”，
不是因為現代人第一次擁有內心，
而是因為在傳統角色、宗教秩序與固定身份逐漸鬆動之後，
個體必須更頻繁地自己定義自己、敘述自己、懷疑自己，
並對“我是誰”承擔回答的責任。
```

同時，  
若說：

```text
現代小說的重要任務，
是探索人的身份與自我形成。
```

也最好再補一句：

```text
小說不只是描寫一個已經存在的自我，
而是展示自我是如何在記憶、欲望、社會期待、自由選擇與自我誤解中
被慢慢形成的。
```

抓住這個精度，  
今天的主題就會更清楚：

```text
所謂現代人的誕生，
不是一夜之間出現了“新的人種”，
而是人開始越來越把自己當成一個需要被理解、被建構、被選擇、
也可能被懷疑的存在。
```

---

## 一、古代世界與現代世界的差異：身份從相對明確，走向逐漸需要自己回答

在古代世界裡，  
一個人的身份  
通常比較明確。  

例如：

- 你出生在哪個家族
- 屬於哪個階級
- 信仰哪種宗教
- 承擔什麼社會角色

這些事情  
往往已經大致決定了  
人生方向。  

這句話非常重要。  

也就是說，  
古代人當然也有  
個性、情感與內心。  
但他通常不需要  
像現代人那樣，  
持續把自己當成  
一個未完成的問題。  

因此現代與古代  
最深的差異之一  
不只是制度不同。  
而是：

```text
身份不再只是被給定，
而越來越需要被個體自己回答。
```

---

## 二、最重要的一句話：現代人的誕生，來自於“自我”成為問題

今天一定要記住  
最重要的一句話：

```text
現代人的誕生，
來自於“自我”成為問題。
```

這句話非常重要。  

但要更精確地理解。  
這不是說  
古代人沒有自我。  
而是說古代人  
較少不停追問：

- 我是誰？
- 我真正想要什麼？
- 我應該成為什麼樣的人？
- 我的人生是否屬於我自己？

這些問題  
在現代社會中  
變得越來越尖銳。  

原因就在於：  

- 傳統角色鬆動了
- 宗教秩序不再自動提供完整答案
- 固定身份開始流動
- 個體被要求為自己的人生負責

也就是說，  
現代人的誕生  
不是因為第一次有了“內心”。  
而是因為：

```text
內心第一次變成一個必須被不斷追問、建構與解釋的空間。
```

---

## 三、從角色到個體：人在傳統社會裡首先是角色，在現代社會裡則越來越像一個專案

在傳統社會裡，  
人首先常常是某種：

```text
角色。
```

例如：

- 兒子
- 父親
- 農民
- 士兵
- 神職人員

身份相對固定。  
社會期待也比較清楚。  

但到了近代，  
社會流動開始增加。  
人們越來越可能：

- 改變職業
- 改變居住地
- 改變生活方式
- 改變社會關係網

於是人生  
不再完全被出身決定。  

這句話非常重要。  

也就是說，  
現代社會  
把人從既定角色裡  
部分釋放出來。  
但同時也讓人  
開始像一個必須持續經營的：

```text
自我專案
（project of the self）
```

這就意味著，  
人不再只是活著。  
還得持續問：

- 我要成為誰？
- 我是否活成了自己？
- 我的人生故事應該如何被講述？

---

## 四、自由帶來的新問題：道路變多之後，“哪一條才屬於我”成了現代焦慮的核心

從表面看，  
近代社會的變化  
當然像一種進步。  
因為人生  
不再那麼徹底地  
被出身決定。  

但自由一增加，  
新的問題也立刻出現：

```text
如果道路很多，
哪一條才屬於我？
```

這句話非常重要。  

也就是說，  
現代人的問題  
不只是“我能不能選”。  
而是：

- 我應該怎麼選？
- 哪個選擇才像真正的我？
- 如果選錯了，我還是不是我自己？
- 是否存在一個“真正的自我”等著被找到？

這使現代自由  
很快就轉化成  
身份問題。  

從這裡開始，  
人生不再只是  
走完一條既定道路。  
而是：

```text
不斷在選擇中塑造自己。
```

## 五、從文藝復興到現實主義：自我意識如何一步步增強

回顧之前的課程。  

在文藝復興時期，  
已經看到：

```text
威廉·莎士比亞
（William Shakespeare）
```

筆下人物  
開始展現出  
強烈個性與反省能力。  

尤其是：

```text
哈姆雷特
（Hamlet）
```

他不停思考、  
不停懷疑、  
不停反省。  

這可以說是：

```text
現代人的雛形。
```

到了十九世紀，  
小說又把這條路  
往前推得更遠。  
人物不只行動，  
還開始：

- 分析自己
- 懷疑自己
- 解釋自己
- 觀看自己的內心

於是：

```text
自我意識變得越來越重要。
```

這句話非常重要。  

也就是說，  
現代人的形成  
不是突然發生的。  
而是從文藝復興的個體覺醒，  
一路經過現實主義  
對社會與內心的深入描寫，  
才逐漸成熟。

## 六、艾瑪·包法利：不滿足現狀、反覆追問幸福的人，是現代自我意識的重要範例

```text
艾瑪·包法利
（Emma Bovary）
```

就是非常典型的例子。  

她不滿足於現狀，  
而且不停在問：

- 人生應該是這樣嗎？
- 幸福到底在哪裡？
- 我為什麼不快樂？
- 為什麼我活成了現在這樣？

這種持續的  
自我檢視與不安，  
正是現代人的  
重要特徵之一。  

這句話非常重要。  

也就是說，  
艾瑪不只是  
一個追求浪漫的人。  
她也是一個  
無法停止把自己  
當成問題的人。  

因此她的悲劇  
不只來自婚姻或社會。  
更來自：

```text
她不斷把人生拿來和某個理想自我相比，
卻始終無法達到。
```

這正是  
現代自我意識  
最痛苦的一面。

## 七、現代人活在內心裡：外部冒險逐漸轉成內心冒險、身份探索與自我觀察

古代文學  
常常描寫：

- 旅行
- 戰爭
- 英雄事業
- 外部冒險

但到了現代文學，  
關注重心  
越來越轉向：

- 內心冒險
- 思想衝突
- 身份探索
- 自我理解
- 自我誤解

這句話非常重要。  

也就是說，  
現代人一個非常鮮明的特徵，  
就是他不只生活。  
他還會：

- 思考自己的生活
- 分析自己的感受
- 觀察自己的動機
- 懷疑自己的決定

這種能力  
使人擁有更深的  
自我反省。  
但也使人  
更容易陷入：

```text
過度意識
（excessive self-consciousness）
```

這就把自由  
與焦慮緊緊綁在一起。  

---

## 八、自我意識的代價：思考越多，懷疑與孤獨也可能越多

自我意識  
當然不是壞事。  

它使人能：

- 反省自己
- 擺脫盲從
- 為自己的人生負責
- 追求更真實的生活

但它同時也有代價。  

因為：

- 思考越多，懷疑也越多
- 自由越多，焦慮也越多
- 當每個人都被視為獨特個體，也可能更孤獨

這句話非常重要。  

因為到了最後，  
沒有人能完全  
替你決定人生。  
你必須自己承擔：

- 選擇
- 錯誤
- 方向
- 意義

也就是說，  
現代人的孤獨  
不只是社交上的孤獨。  
更是：

```text
存在上的孤獨。
```

你必須自己回答  
“我是誰”，  
而這個答案  
沒有誰能替你保證正確。  

---

## 九、俄羅斯文學的重要性：它特別敏銳地抓住了過度自我意識與內心搏鬥的人

俄羅斯小說家  
特別關心這個問題。  

例如：

```text
費奧多爾·陀思妥耶夫斯基
（Fyodor Dostoevsky）
```

就經常描寫：

- 過度思考的人
- 陷入內心衝突的人
- 與自己搏鬥的人
- 被自由與良心折磨的人

這句話非常重要。  

也就是說，  
俄羅斯小說  
之所以適合接續  
前面的課程，  
正因為它把：

```text
現代人內心的分裂
```

寫到了極深處。  

在這裡，  
人不再只是  
社會中的個體。  
他更是：

- 自己的法官
- 自己的敵人
- 自己的迷宮

這正是  
現代人誕生之後  
最戲劇化也最痛苦的結果之一。

## 十、人為什麼變成謎題？因為現代文學發現，人連自己也未必真正了解

古代社會  
常常比較傾向相信：

```text
人有固定本性。
```

但現代文學  
逐漸發現：

```text
人其實很難理解。
```

甚至：

```text
最不了解自己的人，
可能正是自己。
```

這句話非常重要。  

也就是說，  
現代人之所以成為問題，  
不只是因為世界變複雜。  
也因為：

- 動機常常混雜
- 記憶會重寫自己
- 欲望會偽裝成理想
- 自我敘述常常帶有欺騙性

所以小說的新任務  
就不只是寫“人物做了什麼”。  
而是要問：

```text
他以為自己是誰？
他其實又是誰？
```

這一步，  
讓小說真正變成  
探索自我的藝術。

## 十一、身份開始流動：人生不再是一條已經寫好的劇本，而是一個開放過程

另一個重大改變是：

```text
身份不再固定。
```

一個人可能：

- 改變職業
- 改變信念
- 改變價值觀
- 改變人生方向
- 重新解釋自己的過去

因此：

```text
人生變成開放過程。
```

這句話非常重要。  

也就是說，  
現代人的人生  
不再像古代角色那樣  
只是在既定秩序裡完成位置。  
而更像一個  
持續生成、  
持續修改的敘事。  

這也解釋了  
為什麼小說  
會成為現代最重要的文學形式之一。  
因為小說本身  
就是最適合描寫：

- 長期形成
- 反覆改變
- 自我重寫
- 身份流動

的形式。

## 十二、小說的新任務：它不只描述世界，而是研究人如何形成自己、理解自己、迷失自己

因此小說的工作  
也跟著改變了。  

它不再只是：

- 描述世界
- 講述事件
- 記錄社會生活

它還要研究：

- 人如何形成自己
- 如何理解自己
- 如何迷失自己
- 如何在自由中建構身份
- 如何在社會壓力與欲望中誤認自己

這句話非常重要。  

也就是說，  
小說不只是  
把“自我”當成主題。  
它更進一步展示：

```text
自我是如何被形成的。
```

而這種形成  
從來不是純內在的。  
它總是發生在：

- 記憶之中
- 欲望之中
- 社會期待之中
- 自由選擇之中
- 自我誤解之中

因此，  
現代小說  
其實是在替現代人  
發明一種理解自己的方式。

## 十三、在整個課程中的位置：從英雄面對命運，到個體開始面對自己

回顧整條主線，  
可以暫時整理成：

- 史詩時代 → 英雄面對命運
- 中世紀 → 靈魂面對上帝
- 文藝復興 → 個體開始覺醒
- 現實主義 → 個體面對社會
- 現在 → 個體開始面對自己

這句話非常重要。  

也就是說，  
今天這一課  
在整個課程中的意義，  
就是把前面所有發展  
往內再推一步。  

當文學已經學會  
描寫：

- 社會
- 城市
- 婚姻
- 階級
- 慾望
- 自由與焦慮

下一個問題  
自然就是：

```text
當一個人擁有自由、
又必須自己決定自己是誰時，
他該如何與自己相處？
```

這正是  
現代人的真正誕生點。

## 十五、為接下來的閱讀留下真正關鍵的一個問題

今天最後先留下這個問題：

```text
如果現代人的誕生，
真正不是因為他活在較新的時代，
而是因為他開始必須自己回答“我是誰”、
自己承擔“我應該成為誰”，
並且在自由、欲望、社會期待與自我懷疑之間
持續建構與重寫自己，
那麼接下來閱讀俄羅斯小說時，
是否最重要的能力之一，
就是不只問“這個人物做了什麼”，
還要問“他究竟正在如何成為他自己，
又是如何一步步失去他自己”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看人物行動
- 不只看社會壓力
- 不只看道德對錯

而是進一步去看：

- 這個人物如何理解自己
- 他的自我敘述是否可靠
- 他真正想要的是什麼
- 他是如何在自由與焦慮中慢慢塑造身份
- 他又是如何在自我誤解中偏離自己

因為從今天開始，  
文學不再只是  
描寫“人生活在世界裡”。  

更準確地說：

```text
從今天開始，
正式把“現代人的誕生”理解成文學史上一個真正深刻的轉折——
在這裡，
人第一次大規模地不再只是某種角色，
而成為一個必須自己定義、自己敘述、自己懷疑、
也可能永遠無法完全弄清的存在；
而小說真正要探索的，
正是這個自我如何被形成、如何分裂、如何追問自己，
以及如何在自由與不安中活下去。
```
