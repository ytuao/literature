# Day 105｜轉向俄羅斯 II
## Transition to Russia II

今天要討論：

```text
Transition to Russia II
（轉向俄羅斯 II）
```

從第67天開始，  
進入了小說時代。  

一路走過：

- 簡·奧斯汀
- 巴爾扎克
- 狄更斯
- 福樓拜
- 現實主義傳統

也一路看見了：

- 現代城市
- 社會階級
- 婚姻制度
- 個體與社會
- 資產階級世界

這些共同構成了  
十九世紀歐洲小說  
非常重要的內容。  

但接下來，  
文學將進入  
另一個高度。  

這裡先補一個重要精度。  
把“西歐小說主要研究社會，俄羅斯小說開始研究靈魂”  
作為核心句，  
非常準確。  
但若更嚴格，  
最好補上：

```text
這並不是說西歐小說沒有靈魂、俄羅斯小說沒有社會，
而是說俄羅斯小說在繼承現實主義社會觀察的同時，
進一步把人物推向道德、信仰、罪責、自由、救贖與生命意義等
更劇烈的精神問題中心。
```

同時，  
若說：

```text
俄羅斯小說是連接十九世紀現實主義與二十世紀現代思想的重要橋樑。
```

也最好再補一句：

```text
它的重要性不只在題材更沉重，
而在於它把小說變成一種思想形式，
使文學不再只是再現社會，
而能直接承載哲學、宗教、心理與存在問題的衝突。
```

抓住這個精度，  
今天的主題就會更清楚：

```text
所謂“轉向俄羅斯”，
真正轉向的不是國別而已，
而是小說重心的轉變——
從社會結構、階級流動與現代生活的外部觀察，
逐步進入人的靈魂、良心、自由、罪與生命意義的深層探問。
```

---

## 一、為什麼是俄羅斯？因為小說在這裡突然變得更沉重、更激烈、更接近根本問題

很多人第一次接觸  
俄羅斯小說時，  
都會有一種感覺：

- 它更沉重
- 更深刻
- 更激烈
- 更關心人生的根本問題

這不是錯覺。  

這句話非常重要。  

因為在俄羅斯小說裡，  
人物面對的  
往往不只是：

- 愛情成敗
- 社會流動
- 都市競爭
- 階級位置

而是更根本的問題：

- 人為什麼活著？
- 自由是什麼？
- 善與惡是什麼？
- 生命有沒有意義？

也就是說，  
小說到了這裡，  
不再只是描寫社會。  
它開始逼近：

```text
人的存在本身。
```

---

## 二、最重要的一句話：西歐小說主要研究社會，俄羅斯小說開始研究靈魂

今天一定要記住  
最重要的一句話：

```text
西歐小說主要研究社會，
俄羅斯小說開始研究靈魂。
```

這是兩者  
最大的差異之一。  

但要更精確地理解。  
這並不是說：

- 西歐小說沒有靈魂
- 俄羅斯小說沒有社會

而是說  
俄羅斯小說在繼承  
現實主義社會觀察的同時，  
進一步把人物  
推向以下問題的中心：

- 道德
- 信仰
- 罪責
- 自由
- 救贖
- 生命意義

這句話非常重要。  

也就是說，  
如果說巴爾扎克、福樓拜  
非常擅長描寫  
現代世界如何運作，  
那麼托爾斯泰、陀思妥耶夫斯基  
則更常逼問：

```text
人在這個世界裡，
究竟該怎麼活？
```

這就使小說  
從社會分析  
進一步變成  
靈魂探索。  

---

## 三、西歐現實主義的成就：它為現代社會畫出巨大地圖

以：

```text
奧諾雷·德·巴爾扎克
（Honoré de Balzac）
```

為例。  

他最擅長的是：

- 描寫整個社會
- 金融
- 政治
- 階級
- 野心
- 巴黎生活

彷彿替十九世紀法國  
建立一幅巨大地圖。  

而：

```text
居斯塔夫·福樓拜
（Gustave Flaubert）
```

則更深入人物心理。  
但他的關注點  
仍然與社會緊密相關。  

例如：

- 欲望如何形成
- 幻想如何破滅
- 社會如何塑造人生

這句話非常重要。  

也就是說，  
西歐現實主義  
最大的成就之一，  
是讓人第一次  
如此清楚地看見：

```text
現代社會如何運作，
又如何塑造個體。
```

但接下來  
俄羅斯小說  
會把問題  
再往深處推一步。  

---

## 四、俄羅斯作家提出的新問題：不只問社會如何運作，而是問生命本身如何成立

到了俄羅斯，  
問題開始改變。  

不再只是：

- 社會如何運作？
- 階級如何影響人生？
- 都市如何塑造欲望？

而是轉向：

- 人為什麼活著？
- 自由是什麼？
- 善與惡是什麼？
- 生命有意義嗎？

這句話非常重要。  

也就是說，  
在俄羅斯小說裡，  
真正的主角  
常常不是事件，  
甚至不是情節。  
而是：

```text
人的內心。
```

從這裡開始，  
小說的張力  
就不只來自  
外部衝突。  
更來自：

- 靈魂的分裂
- 良心的拷問
- 信仰與懷疑的碰撞
- 自由與責任的對立
- 生之意義的追問

這正是  
“轉向俄羅斯”  
最關鍵的變化。

## 五、托爾斯泰與陀思妥耶夫斯基：一個追問人生應如何度過，一個追問靈魂為何會墮落又如何得救

```text
列夫·托爾斯泰
（Leo Tolstoy）
```

關心的問題包括：

- 幸福是什麼？
- 家庭是什麼？
- 歷史如何推動人？
- 人生應該如何度過？

他的小說  
經常像一種  
哲學探索。  

而：

```text
費奧多爾·陀思妥耶夫斯基
（Fyodor Dostoevsky）
```

則更進一步。  
他關心：

- 罪惡
- 自由
- 信仰
- 良心
- 自我毀滅
- 救贖

這句話非常重要。  

也就是說，  
如果說托爾斯泰  
更擅長描寫  
人生如何展開、  
歷史如何包圍人、  
幸福如何被誤解；  
那麼陀思妥耶夫斯基  
則更像在靈魂內部  
搭起一座法庭。  

在那裡，  
每個人都必須回答：

```text
我究竟為何這樣活？
我究竟應否為自己負責？
```

## 六、人的內在裂縫：理性已不足以解釋人類

陀思妥耶夫斯基  
反覆追問一個問題：

```text
人為什麼會做錯事？
```

人明明知道  
某件事不好，  
卻仍然去做。  
為什麼？  

這句話非常重要。  

也就是說，  
俄羅斯小說  
逐漸發現：  

```text
理性似乎不足以解釋人類。
```

西歐現實主義  
已經發現  
人會受到社會影響。  
而俄羅斯小說  
則更進一步發現：

```text
人內部本身就充滿衝突。
```

例如：

- 理性與欲望
- 信仰與懷疑
- 自由與責任
- 良知與自我辯護

同時存在。  

從這裡開始，  
小說不只是  
研究社會行為。  
它也開始研究：

```text
靈魂為何會分裂。
```

---

## 七、焦慮被推向哲學層面：從現代焦慮走向存在焦慮

前幾天已經談過  
現代焦慮。  

但俄羅斯小說  
會把這種焦慮  
推向更深一層。  

不只是：

- 工作焦慮
- 成功焦慮
- 時間焦慮
- 身份焦慮

而是：

```text
存在焦慮。
```

也就是：

- 我為什麼活著？
- 生命有什麼意義？
- 如果沒有答案怎麼辦？
- 如果自由意味著責任，那我能承受嗎？

這句話非常重要。  

也就是說，  
俄羅斯小說  
讓焦慮不再只是  
社會情緒。  
它變成：

```text
一種哲學與存在問題。
```

而這一步  
將深刻影響  
整個二十世紀思想。

## 八、為什麼偏偏是俄羅斯？因為它處在傳統與現代、宗教與理性之間的強烈斷層中

十九世紀俄羅斯  
處在一個  
非常特殊的位置。  

它同時面對：

- 傳統與現代
- 宗教與理性
- 東方與西方

因此思想衝突  
特別強烈。  

這句話非常重要。  

也就是說，  
俄羅斯小說之所以  
如此劇烈，  
不只是因為作家個人才華。  
也因為整個文化處境  
本身就充滿裂縫。  

在這樣的背景下，  
小說自然更容易成為：

- 價值衝突的場所
- 信仰危機的場所
- 歷史轉型的場所
- 靈魂問題被極端放大的場所

這也解釋了  
為什麼俄羅斯小說  
總讓人感到  
重量特別大。  

---

## 九、小說變成思想實驗室：人物不只活著，也在替不同觀念彼此辯論

俄羅斯作家  
並不把小說  
只當成娛樂。  

他們更把小說  
當成一種：

```text
思想實驗室。
```

在這裡：

- 人物成為思想的載體
- 不同觀點互相碰撞
- 道德與信仰問題被具體化
- 抽象觀念進入活生生的人生處境

這句話非常重要。  

尤其在陀思妥耶夫斯基作品中，  
常出現許多不同聲音。  

- 沒有人完全正確
- 也沒有人完全錯誤
- 每種立場都被推到極端
- 讀者必須自己思考

這就使他的小說  
具有很強的：

```text
多聲部
（polyphony）
```

特徵。  

也就是說，  
小說不再只是  
作者替角色說話。  
而是讓不同靈魂、  
不同思想、  
不同價值觀  
同時在作品中  
彼此對抗。

## 十、俄羅斯小說為何成為通往二十世紀的橋樑？因為它提前打開了現代思想的核心問題

因此，  
俄羅斯小說的重要性  
不只屬於十九世紀。  

它還預告了：

- 存在主義
- 心理分析
- 現代主義文學
- 二十世紀哲學

這句話非常重要。  

但更精確地說，  
它的重要性  
不只在題材更沉重。  
而在於它把小說  
變成一種思想形式，  
使文學不再只是  
再現社會，  
而能直接承載：

- 哲學問題
- 宗教問題
- 心理問題
- 存在問題

的衝突。  

這就是  
俄羅斯小說  
真正的歷史位置。  

也就是說，  
從這裡開始，  
文學不再只是  
描述一個時代。  
它還能成為：

```text
思考人類存在本身的場所。
```

## 十一、我們即將進入的新世界：接下來的重心將逐步轉向內心、自由與生命意義

接下來的課程，  
焦點將逐漸轉向：

- 人的內心
- 道德選擇
- 自由意志
- 信仰危機
- 生命意義

這些最根本的問題。  

這句話非常重要。  

也就是說，  
從今天開始，  
閱讀方式也需要改變。  
不只看：

- 人物做了什麼
- 情節如何推進
- 社會背景如何運作

還要更進一步看：

- 人物如何審判自己
- 他如何面對自由
- 他如何理解罪與責任
- 他如何在沒有穩定答案時活下去

從這裡開始，  
小說就不再只是  
“社會的鏡子”。  
它也變成：

```text
靈魂的顯影術。
```

## 十三、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果“轉向俄羅斯”真正轉向的，
不是國別本身，
而是小說從社會地圖走向靈魂深處，
從描寫現代生活的外部形式
走向自由、罪、良心、信仰與生命意義的內部問題，
那麼接下來閱讀托爾斯泰與陀思妥耶夫斯基時，
是否最重要的能力之一，
就是不只問“這個人物發生了什麼事”，
而要問“他的靈魂正在經歷什麼審判”，
以及“他所承受的問題，
是否其實就是現代人至今仍無法徹底逃開的問題”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看社會背景
- 不只看情節推進
- 不只看人物對錯

而是進一步去看：

- 他內心最深的衝突是什麼
- 他如何理解自由與責任
- 他是否相信救贖可能
- 他如何面對懷疑、罪與良心
- 他提出的是否是整個現代世界仍在面對的問題

因為從今天開始，  
文學不再只是  
描寫“現代社會如何運作”。  

更準確地說：

```text
從今天開始，
正式把“轉向俄羅斯”理解成整個課程的一次深層換軌——
在這裡，
小說不再滿足於替現代世界畫出外部地圖，
而開始深入人的靈魂、良心與存在本身；
而托爾斯泰與陀思妥耶夫斯基之所以偉大，
正因為他們讓小說成為一個可以同時容納社會、心理、哲學、宗教與生命問題的巨大空間，
使文學第一次如此完整地承擔起“思考人為何而活”的任務。
```
