# Day 100｜人生作為敘事
## Life as Narrative

今天是課程的第100天。  

從最早的神話與史詩開始，  
一路走過：

- 古希臘史詩
- 悲劇
- 中世紀文學
- 文藝復興
- 莎士比亞
- 現實主義小說
- 現代人的誕生

現在，  
來到一個總結性的主題：

```text
Life as Narrative
（人生作為敘事）
```

這個觀念  
不只屬於文學。  
它也深刻影響  
現代人理解自己的方式。  

這裡先補一個重要精度。  
把“人生的意義，往往不是來自發生了什麼，而是來自我們如何講述它”  
作為核心句，  
非常準確。  
但若更嚴格，  
最好補上：

```text
敘事不是憑空編造意義，
而是在既有經歷之上進行選擇、排列、強調、解釋與重寫，
使零散事件形成可理解的生命結構。
```

同時，  
若說：

```text
自我認同是在敘事中形成的，
而非固定不變的本質。
```

也最好再補一句：

```text
這不是否定人格、記憶與歷史經驗的真實性，
而是強調個體之所以能把自己看成“同一個人”，
往往正是因為他把過去、現在與想像中的未來
連結成了一個持續修訂的生命故事。
```

抓住這個精度，  
今天的主題就會更清楚：

```text
所謂“人生作為敘事”，
不是把人生想成虛構小說，
而是理解人如何透過講述、回顧、整理與重寫自己的經歷，
使生命從零散事件變成一個可以被理解、承受與繼續活下去的整體。
```

---

## 一、人生只是一連串事件嗎？如果只是事件排列，人生其實還沒有意義

從客觀角度看，  
人生似乎只是  
事件的集合：

- 出生
- 成長
- 學習
- 工作
- 戀愛
- 失敗
- 成功
- 死亡

但如果只是  
事件排列，  
人生其實還沒有意義。  

因為真正重要的，  
不是只有  
“發生了什麼”。  
而是：

```text
我們如何理解這些事件。
```

這句話非常重要。  

也就是說，  
意義不是自動  
附著在事件上。  
它往往要透過：

- 選擇哪些事件重要
- 排列它們的關係
- 說明它們如何改變了自己
- 給它們一個整體方向

才會出現。  

因此人生  
不只是經歷。  
它也必須被：

```text
敘事化。
```

---

## 二、最重要的一句話：人生的意義，往往不是來自發生了什麼，而是來自我們如何講述它

今天一定要記住  
最重要的一句話：

```text
人生的意義，
往往不是來自發生了什麼，
而是來自我們如何講述它。
```

這句話非常重要。  

但要更精確地理解。  
這不是說  
人可以隨便捏造人生意義。  
而是說敘事會在  
既有經歷之上進行：

- 選擇
- 排列
- 強調
- 解釋
- 重寫

讓零散事件  
形成一個可理解的  
生命結構。  

也就是說，  
同樣的人生經歷，  
不同人可能會給出  
完全不同的解釋。  

有人把挫折  
看成羞辱。  
有人把它  
看成轉折。  
有人把失敗  
當成人生終點。  
也有人把它  
理解為學習的開始。  

因此：

```text
人生不只是經歷，
也是敘事。
```

## 三、我們都在講自己的故事：當回答“你是誰”時，人很少只報資料

想一想。  

當別人問：

```text
你是怎樣的人？
```

通常會怎麼回答？  

很少有人會回答：

- 我的身高是多少
- 我的體重是多少
- 我的出生時間是多少

人通常會開始：

```text
說故事。
```

例如：

- 我從小喜歡閱讀
- 我曾經遭遇挫折
- 某件事改變了我
- 我一直努力追求某個目標

這些故事  
慢慢組成了：

```text
自我認同。
```

這句話非常重要。  

也就是說，  
身份不是單靠  
客觀資訊構成的。  
人真正理解“我是誰”，  
往往是透過  
那些被自己認為  
有意義的生命片段。  

因此回答“我是誰”，  
本質上常常就是在回答：

```text
我把自己講成了一個什麼樣的故事。
```

## 四、敘事如何讓人生連續？它把碎片連成一條可以辨認的線

人生其實  
充滿碎片。  

每天都發生  
許多事情。  
大部分都會被遺忘。  

敘事的作用，  
就是把這些碎片  
連接起來。  
形成一條線。  

這句話非常重要。  

也就是說，  
人之所以能覺得  
自己不是完全破碎的，  
往往正因為他能透過敘事  
把以下三者連起來：

- 過去的自己
- 現在的自己
- 未來的自己

從而感到：  

```text
雖然我一直在改變，
但我仍然是同一個人。
```

這就是  
身份感的一個重要來源。  

因此敘事  
不只是講述已經發生過的事。  
它還在替生命  
建立：

- 連續性
- 可理解性
- 方向感
- 自我一致感

## 五、小說與人生為什麼相似？因為理解一個人，關鍵不在資料，而在生命故事

小說中的人物  
也是如此。  

讀者之所以理解角色，  
並不是因為知道  
他的所有資料。  
而是因為知道：

- 他經歷了什麼
- 他如何理解那些經歷
- 他怎樣被某些事改變
- 他怎樣為自己的人生賦予意義

這句話非常重要。  

也就是說，  
理解一個人物，  
其實就是在理解  
他的生命故事。  

十九世紀小說  
逐漸清楚地發現：

```text
人不是固定不變的，
人是在故事中形成的。
```

因此現代小說  
之所以偉大，  
不只是因為它塑造了很多角色。  
更因為它發現：

```text
要理解一個人，
就必須理解他如何活在自己的敘事裡。
```

---

## 六、托爾斯泰與陀思妥耶夫斯基：一個寫人生如何展開，一個寫人如何解釋自己的人生

```text
列夫·托爾斯泰
（Leo Tolstoy）
```

特別擅長描寫  
人物的一生。  

讀者可以看見：

- 理想如何誕生
- 幻想如何破滅
- 信念如何改變
- 人生如何轉折

也就是說，  
在托爾斯泰那裡，  
人物不是靜止的。  
而是在時間中  
慢慢成為現在這個人。  

而：

```text
費奧多爾·陀思妥耶夫斯基
（Fyodor Dostoevsky）
```

則更關心  
另一件事：  

```text
人如何理解自己的故事。
```

有些人  
把自己看成英雄。  
有些人  
把自己看成受害者。  
有些人  
把自己看成罪人。  

於是：

```text
不同敘事產生不同人生。
```

這句話非常重要。  

也就是說，  
托爾斯泰更強於展示  
人生如何展開，  
而陀思妥耶夫斯基  
則更強於展示：

- 人如何講述自己
- 人如何誤解自己
- 人如何被自己的敘事困住或拯救

這兩種能力  
都把“人生作為敘事”  
推到了極高的層次。

## 七、記憶也是敘事：回憶不是單純儲存，而是一次又一次重寫

人通常以為：

```text
記憶只是儲存過去。
```

其實不是。  

每一次回憶，  
人都在某種程度上  
重新整理故事：

- 強調某些部分
- 忽略某些部分
- 改變某些連結
- 重新賦予意義

這句話非常重要。  

也就是說，  
記憶不是純粹被動的倉庫。  
它更像一種  
不斷被修訂的生命編輯。  

因此“過去”  
並不是固定不動地  
躺在那裡。  
它常常會因為：

- 現在的處境
- 新的理解
- 不同的情感狀態
- 對未來的期待

而被重新講述。  

這也說明了  
為什麼自我認同  
從來不是完全定型的。  
因為連記憶本身  
都帶著敘事性。  

---

## 八、人生沒有固定劇本：現代世界的自由，也意味著必須自己建構故事

古代世界  
常比較相信：

```text
命運已被安排。
```

但現代世界  
則越來越不同。  
人生充滿不確定性。  

因此：

```text
每個人都必須自己建構故事。
```

這句話非常重要。  

也就是說，  
現代人的自由  
不只是可以選擇行動。  
也是必須對自己的生命  
做出敘事性的回答。  

你不能只活著。  
你還得回答：

- 我的人生到底在往哪裡去？
- 哪些事對我最重要？
- 我怎麼理解自己的失敗與轉折？
- 我準備把自己講成一個什麼樣的人？

因此現代人的自由  
其實也伴隨著  
敘事責任。  

---

## 九、敘事與自由：自由不只在行動上，也在解釋上

這裡出現一個  
非常重要的關係：

```text
自由不只是選擇行動，
也是選擇如何理解人生。
```

同樣的失敗，  
有人視為終點。  
有人視為開始。  

同樣的受傷，  
有人把它講成  
自己永遠無法前進的原因。  
也有人把它講成  
自己看清世界、  
重新開始的契機。  

這句話非常重要。  

也就是說，  
敘事不是被動描述。  
它具有主動力量。  

人在某種程度上  
可以透過改變敘事，  
改變自己與過去的關係。  
當然，  
這不是魔法。  
它不能抹去發生過的事。  
但它能改變：

- 那件事對自己意味著什麼
- 自己將如何帶著它繼續生活
- 自己是否仍被某個舊故事困住

這就是  
敘事與自由  
最深的連結之一。

## 十、故事如何改變現實？因為敘事會塑造未來的行動方向

因此敘事  
從來不只是描述。  
它還會影響行動。  

如果一個人相信：

```text
我是一個永遠失敗的人。
```

與相信：

```text
我是一個正在學習的人。
```

未來可能會  
完全不同。  

這句話非常重要。  

也就是說，  
敘事不只是對過去的總結。  
它還會成為：

- 現在行動的依據
- 未來選擇的框架
- 自我可能性的邊界

因此人生故事  
不是寫完才有力量。  
它在寫的過程中，  
就已經在塑造現實。  

這也是為什麼  
現代小說如此重要。  
因為它讓人看見：

```text
人是如何被自己講述的人生所塑造的。
```

---

## 十一、現代小說的重要任務：它不再只講冒險，而是探索自我如何形成意義

現代小說  
不再只講冒險故事。  

它更深地探索：

- 人如何形成自我
- 如何賦予人生意義
- 如何面對不確定世界
- 如何在時間中重寫自己

這句話非常重要。  

也就是說，  
當小說走到成熟階段，  
它已經不再只是  
“發生了什麼”的藝術。  
它變成：

```text
“人生如何被理解”的藝術。
```

這就是  
從神話、史詩、悲劇  
一路走到現代小說  
最重要的歷史轉變之一。  

從這裡開始，  
小說最深的任務  
就是呈現：

- 一個人如何形成自己的生命敘事
- 一個人如何在這個敘事裡尋找意義
- 一個人如何在不確定中繼續寫下去

---

## 十二、人生像一本未完成的小說：它總還有下一章，但永遠不能一次看完全貌

這也是許多現代作家  
非常接近的看法。  

人生不像  
已完成的書。  
而像正在書寫中的小說。  

每一天，  
都在增加新的篇章。  

這句話非常重要。  

也就是說，  
人生作為敘事  
最動人的地方之一，  
就在於它永遠不是  
完全封閉的。  

你可以回顧前面。  
可以修改理解。  
可以重新命名某些轉折。  
但你永遠不能  
一次看完全部。  

因此人生的敘事性  
總是同時包含：

- 已經發生的部分
- 正在理解的部分
- 尚未展開的部分

這也讓現代人的存在  
總帶著開放性與不確定性。

```text
人生本身也具有敘事結構。
```

文學與現代自我理解

## 十五、為下一段閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果人生真正不是一堆零散事件的總和，
而是我們不斷透過講述、回憶、選擇、解釋與重寫
所形成的一個生命故事，
那麼接下來閱讀更深的現代小說時，
是否最重要的能力之一，
就是不只問“這個人物經歷了什麼”，
而要問“他把自己的生命講成了什麼樣的故事，
這個故事又是如何塑造了他未來的行動、希望、痛苦與可能性”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看情節事件
- 不只看角色資料
- 不只看道德評價

而是進一步去看：

- 角色如何講述自己
- 他如何解釋自己的過去
- 哪些記憶被強調，哪些被壓抑
- 他的生命故事如何給他力量或把他困住
- 敘事如何在理解過去的同時塑造未來

因為從今天開始，  
文學不再只是  
描寫“人生發生了什麼”。  

更準確地說：

```text
從今天開始，
正式把“人生作為敘事”理解成整個課程前100天最重要的總結之一——
在這裡，
生命不是現成完成的東西，
而是人在時間中透過記憶、選擇、解釋與講述
不斷建構出來的整體；
而小說真正深刻之處，
就在於它不只展示事件，
更展示人如何把事件變成意義、
把碎片變成生命、
把經歷變成“我是誰”的故事。
```
