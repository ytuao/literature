# Day 086｜慾望與階級
## Desire and Class

今天要討論：

```text
Desire and Class
（慾望與階級）
```

這是現實主義小說  
最深刻的主題之一。  

我們常常以為：  

```text
自己的願望來自自己。
```

但十九世紀小說家  
開始提出一個  
非常尖銳的疑問：

```text
我們的慾望，
真的完全屬於自己嗎？
```

這裡先補一個重要精度。  
把“慾望不只是心理現象，也是社會現象”  
作為核心命題，  
非常準確。  
但若更嚴格，  
最好補上：

```text
社會並不是直接命令人去渴望某樣東西，
而是透過階級位置、榜樣、文化敘事、消費符號、
教育期待與比較機制，
慢慢塑造什麼被看作值得追求。
```

同時，  
若說：

```text
階級會塑造人們認為值得追求的目標。
```

也最好再補一句：

```text
階級不只決定人“能不能得到”，
也會影響人“覺得自己應該想要什麼、
敢不敢想要什麼、以及如何想像幸福”。
```

抓住這個精度，  
今天的主題就會更清楚：

```text
慾望不是單獨長在心裡的，
它常常是在社會中被喚起、被比較、被模仿、
被合法化，最後才被個人誤以為完全出於自己。
```

---

## 一、慾望推動故事：沒有“想要”，就沒有行動，也沒有小說

回顧前一天的內容。  

人物之所以行動，  
是因為他有所渴望。  

例如：

- 有人想要愛情
- 有人想要金錢
- 有人想要地位
- 有人想要自由

如果沒有慾望，  
故事往往就  
不會真正開始。  

這句話非常重要。  

也就是說，  
慾望是人物行動的引擎。  
它不只讓人物“去做事”，  
也讓讀者開始理解：

- 他為什麼動起來
- 他為什麼願意冒險
- 他為什麼會犯錯
- 他為什麼會不肯停下

所以一部小說  
如果要真正塑造人物，  
就一定得處理：

```text
他到底想要什麼。
```

---

## 二、但慾望從哪裡來？真正深刻的問題不是“要什麼”，而是“為什麼會想要”

真正困難的問題  
其實不是：

```text
一個人物想要什麼？
```

而是：

```text
他為什麼會想要這個？
```

例如：

- 如果你想成為富人，為什麼？
- 如果你想住豪宅，為什麼？
- 如果你想進入上流社會，又是為什麼？

現實主義作家  
逐漸發現：

```text
很多願望其實來自社會。
```

這句話非常重要。  

也就是說，  
小說開始不再只描述  
人的行動。  
它開始追問：

- 願望是怎麼被製造的
- 哪些東西看起來像“自我選擇”，其實帶著時代印記
- 人如何把外部價值誤認成內在需求

這一步，  
使現實主義小說  
真正深入了  
人物動機的深處。  

---

## 三、最重要的一句話：慾望不只是心理現象，也是社會現象

今天一定要記住  
最重要的一句話：

```text
慾望不只是心理現象，
也是社會現象。
```

換句話說，  
我們想要什麼，  
往往深深受到  
周圍世界影響。  

這句話非常重要。  

但更精確地說，  
社會不是直接  
命令人去想要。  
它更像是在慢慢地：

- 提供範本
- 提供榜樣
- 提供可羨慕的生活想像
- 製造可比較的差距
- 讓某些目標看起來自然、正當、值得追求

也就是說，  
慾望常常不是  
突然出現在心裡。  
而是在社會裡  
被一點一點訓練出來的。

## 四、階級是什麼？它不只關於錢，也關於生活方式與想像能力

所謂：

```text
階級
（Class）
```

不只是收入高低。  

它同時還包括：

- 教育
- 生活方式
- 社交圈
- 文化品味
- 未來期待

因此：

```text
不同階級的人，
可能擁有不同的夢想。
```

這句話非常重要。  

也就是說，  
階級不只決定  
你現在站在哪裡。  
它也會決定：

- 你覺得哪些東西是“像你這種人”可以追求的
- 你覺得什麼叫體面
- 你是否敢想像更高的位置
- 你會不會把某些幸福形式視為理所當然

所以階級  
不只是外部位置。  
它也深入到：

```text
想像力本身。
```

## 五、巴爾札克的人物：對成功的渴望，其實是整個社會價值秩序進入個體內心的結果

在：

```text
奧諾雷·德·巴爾扎克
（Honoré de Balzac）
```

筆下，  
有許多年輕人  
來到：

```text
巴黎
（Paris）
```

追求成功。  

他們渴望：

- 財富
- 名聲
- 權力

這看起來  
像個人選擇。  
但其實也反映了  
整個社會的價值觀。  

這句話非常重要。  

也就是說，  
巴爾札克人物  
之所以總是那麼想往上爬，  
並不是因為他們天生  
就特別虛榮。  
而是因為他們活在  
一個已經把：

- 成功
- 出人頭地
- 進入更高圈層
- 被看見與被承認

當成核心價值的社會裡。  

所以他們的野心  
不只是個人心理。  
更是：

```text
社會價值內化後的行動形式。
```

## 六、艾瑪的渴望：她想要的不是單純幸福，而是一種被文化包裝過的幸福形象

回到：

```text
艾瑪·包法利
（Emma Bovary）
```

她渴望的是：

- 浪漫愛情
- 華麗生活
- 高級社交圈

她相信：  

```text
只要得到這些，
人生就會圓滿。
```

但這些願望  
並不是從真空中來的。  
它們來自：

- 小說
- 時尚文化
- 對上流社會的想像

這句話非常重要。  

也就是說，  
艾瑪真正想要的，  
其實不是某個具體物品  
或某段具體關係。  
她想要的是：

```text
一種被文化塑造成“值得羨慕的人生形式”。
```

所以她的悲劇  
非常現代。  
因為她不只是  
誤判現實。  
她其實是把  
被社會包裝好的幸福幻象，  
當成了自己最真實的願望。

## 七、模仿的慾望：我們常常不是直接想要東西，而是透過他人學會想要

後來的現代思想  
提出過一個  
非常重要的觀察：  

```text
人常常不是直接渴望某樣東西，
而是看到別人渴望，
自己也開始渴望。
```

例如：  

- 某個東西本來很普通
- 但當成功人士擁有它
- 它 suddenly 變得珍貴

這句話非常重要。  

也就是說，  
很多慾望  
其實帶著強烈的：

```text
模仿性
（mimetic desire）
```

人不只是追求物本身。  
更是在追求  
這個物所代表的：

- 地位
- 身分
- 可羨慕性
- 進入某個階級世界的門票

這正是  
慾望和階級  
如此緊密相連的原因之一。  

---

## 八、比較的力量：在階級社會裡，慾望會被不斷重新點燃

在階級社會裡，  
比較幾乎無處不在。  

例如：

- 誰住在哪裡
- 誰穿什麼衣服
- 誰和誰來往
- 誰能進入哪些圈子

因此：

```text
慾望不斷被刺激。
```

這句話非常重要。  

也就是說，  
比較不只是  
附帶現象。  
它本身就是  
慾望的燃料。  

一旦人開始  
把自己放進社會排序裡，  
他就很難只問：

- 我需要什麼？

而更容易去問：

- 我比別人多還是少？
- 我看起來像不像成功的人？
- 我是否被排除在某個世界之外？

從這裡開始，  
慾望就不再只是  
關於需要。  
它變成了：

```text
關於位置。
```

## 九、現代城市的效果：讓不同階級彼此可見，也讓欲望更容易升溫

昨天已經談過  
現代城市。  

城市的一個重要效果是：  

```text
讓不同階級的人互相看見。
```

於是：

- 窮人看見富人
- 富人看見更富的人
- 每個人都開始比較

這句話非常重要。  

也就是說，  
城市不只是  
經濟空間。  
它也是  
慾望被加速製造的空間。  

因為在城市裡，  
你不會只活在自己的世界。  
你會不斷看見：

- 更體面的生活
- 更高級的消費
- 更耀眼的成功樣板
- 更有權力的人際圈

這些可見性  
會讓願望變得更強。  
也讓不滿足  
變得更持久。

## 十、成功的想像：社會價值會慢慢變成個人的“理想”

十九世紀資產階級社會  
不斷告訴人們：

- 成功很重要
- 財富很重要
- 社會地位很重要

於是，  
這些價值就會  
逐漸被內化。  

最後變成：

```text
個人的願望。
```

這句話非常重要。  

也就是說，  
現代社會最微妙的地方  
就在這裡：  
它未必要強迫你。  
它只需要  
讓某些東西  
看起來理所當然地值得追求。  

於是人就會  
在沒有感覺被命令的情況下，  
主動去追逐  
社會最看重的東西。  

這就是  
慾望社會化  
最深的一層。

## 十一、慾望與不滿足：欲望真正可怕的地方，在於它會不斷複製自己

今天還要抓住  
另一個重要觀察：

```text
慾望經常製造新的慾望。
```

得到某樣東西後，  
人很快又會  
想要下一樣。  

因此：

- 滿足感往往短暫
- 追逐卻持續不斷

這句話非常重要。  

也就是說，  
如果慾望與社會比較  
緊密相連，  
那它就很難停止。  
因為總有：

- 更高的位置
- 更好的生活樣板
- 更體面的象徵
- 更被羨慕的版本

所以現代焦慮  
常常不來自  
完全得不到。  
而是來自：

```text
永遠覺得還不夠。
```

這也正是  
艾瑪·包法利  
如此具有現代性的原因。

## 十二、為什麼小說愛寫野心？因為野心最能暴露一個社會真正崇拜什麼

小說之所以  
那麼愛寫：

```text
野心。
```

是因為野心  
會直接揭露社會。  

一個人物  
想要什麼，  
其實往往透露：

```text
這個社會重視什麼。
```

如果人人  
都追求財富，  
就說明財富  
是重要價值。  

如果人人  
都追求名聲，  
就說明名聲  
具有力量。  

這句話非常重要。  

也就是說，  
人物的欲望  
不只是個人檔案。  
它也是社會  
價值排序的線索。  

所以當讀者  
分析人物野心時，  
其實也在讀：

- 一個時代的崇拜物
- 一套階級秩序的誘惑
- 一種被廣泛承認的成功標準

這正是  
慾望成為理解社會入口  
的原因。

## 十三、愛情也受到階級影響：最私人的感情，也可能帶著社會條件

很多人會以為：

```text
愛情最純粹。
```

但現實主義小說  
經常指出：  
事情沒有那麼簡單。  

婚姻和愛情  
常常牽涉到：

- 財產
- 教育
- 家庭背景
- 社會地位
- 生活方式是否相容

因此：

```text
愛情與階級常常交織在一起。
```

這句話非常重要。  

也就是說，  
愛情並不是  
只發生在兩顆心之間。  
它也發生在：

- 兩個家庭之間
- 兩種生活方式之間
- 兩種階級期待之間

所以現實主義  
會逼人看到：  

```text
即使是“我愛你”這句話，
有時也背著一整個社會在說。
```

## 十四、自由與慾望：選項越多，不一定越自由，也可能越迷失

現代社會  
給人更多自由。  

但同時  
也提供了更多：

- 選項
- 廣告
- 商品
- 成功範本
- 理想生活模板

於是：

```text
人反而更容易迷失。
```

這句話非常重要。  

因為問題不再只是  
“能不能選”。  
而是：

```text
我到底該選什麼，
以及那真的是我想要的嗎？
```

也就是說，  
現代自由  
並不自動帶來清晰。  
它有時反而會讓人  
陷入更深的困惑：  

- 哪個欲望屬於自己？
- 哪個欲望只是被社會植入？
- 哪個目標通向幸福，哪個只通向認可？

這正是  
現代小說  
如此擅長描寫的問題。

## 十五、我們真的知道自己想要什麼嗎？幸福與社會認可常常被混在一起

這是現實主義小說  
提出的一個  
非常尖銳的問題：

```text
我們真的知道自己想要什麼嗎？
```

很多時候，  
人以為自己  
在追求幸福。  
但實際上，  
他追求的可能只是：

```text
社會認可。
```

而這兩者  
並不一定相同。  

這句話非常重要。  

也就是說，  
現實主義  
真正深刻的地方  
不只是描述慾望。  
而是揭示：

```text
人常常會把“被羨慕”
誤以為“會幸福”。
```

這正是  
為什麼小說人物  
會那麼努力，  
卻仍然那麼不滿。

## 十六、現實主義的深刻之處：它把慾望變成理解社會價值與權力結構的入口

現實主義  
不只是描寫行動。  
它更進一步分析：

```text
行動背後的動機。
```

而人物的欲望，  
就成了理解社會的  
重要入口。  

這句話非常重要。  

也就是說，  
當讀者問：

- 這個人物到底想要什麼？
- 他為什麼偏偏想要這個？
- 他從哪裡學會覺得這樣東西值得追求？

其實就在同時問：

- 這個社會最看重什麼？
- 這個階級世界如何塑造欲望？
- 權力是如何透過吸引而不是命令來運作？

這就是  
現實主義  
真正的分析深度。

## 十七、第五部分主線中的位置：理解慾望，也是在理解自由被什麼塑形

回到第五部分  
一路追問的核心問題：

```text
個體如何在社會中追求自由、愛情與地位？
```

今天得到的一個  
非常重要的發現是：

```text
個人的願望，
往往受到階級與社會價值塑造。
```

因此：

```text
理解慾望，
也是理解社會。
```

這句話非常重要。  

也就是說，  
第五部分到今天  
又更深了一步。  
一個人是否自由，  
不只要看  
外部限制多不多。  
還要看：

- 他的願望是怎麼形成的
- 他是否把社會期待當成了自己的內心聲音
- 他追求的到底是幸福、自由，還是被承認

這使“自由”  
不再只是行動問題。  
它也成了：

```text
慾望來源的問題。
```

```text
第六，
現實主義小說透過慾望揭示社會價值與權力結構；
人物想要什麼、如何想要、為何覺得某些目標值得追求，
往往正暴露出一個社會最深的價值排序。
```

```text
第七，
從第五部分主線來看，
個體追求自由、愛情與地位時，
必須同時面對一個更深的問題：
自己的願望究竟有多少是真正出於自己，
又有多少其實是階級世界與社會價值內化後的結果。
```

## 十九、為接下來讀小說補上一個更尖銳的問題

今天最後先留下這個問題：

```text
如果現實主義小說真正高明的地方，
不是只寫人物“想要什麼”，
而是進一步追問：
這個人物為什麼會把某些東西看成值得追求、
他從哪裡學會羨慕、
又為什麼總把社會認可誤認成幸福，
那麼接下來閱讀十九世紀小說時，
是否最重要的能力之一，
就是不只問“他想要什麼”，
還要問“到底是誰教會他這樣想要”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來讀小說的方式：

- 不只看人物是否有野心
- 不只看人物追求什麼目標
- 不只看他最後成功還是失敗

而是進一步去看：

- 這個願望背後有什麼階級想像
- 這種渴望是出於需要，還是出於比較
- 這個人物是在追求幸福，還是在追求被承認
- 為什麼某些東西對他如此有吸引力，而對別人卻未必如此

因為從今天開始，  
慾望不再只是  
人物內心的一個詞。  

更準確地說：

```text
從今天開始，
正式把慾望理解成現代小說裡最深的社會透鏡之一——
在這裡，
人看似在追求自己的夢，
其實常常是在追逐一整個階級世界替他命名過的幸福圖像；
而小說的任務，
就是讓人看見這些圖像如何進入內心、變成願望，
最後又如何塑造命運。
```
