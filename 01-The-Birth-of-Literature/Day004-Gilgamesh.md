# Day 004｜吉爾伽美什史詩
## Epic of Gilgamesh

從今天開始，  
你正式進入這門課的第一部真正作品。  

如果說 Day 001 到 Day 003  
都還在建立概念地基：

- 文學是什麼
- 神話與敘事是什麼
- 人如何在故事中被理解

那麼 Day 004 開始，  
你不再只是在討論概念，  
而是直接走進人類最古老的文學現場之一。  

今天你進入的，  
不只是一個故事，  
而是：

```text
文學的起源層
```

因為《吉爾伽美什史詩》（Epic of Gilgamesh）  
幾乎可以說是我們今天仍能讀到的  
最早的重要文學作品之一。  

它的重要性不只是“很早”，  
而在於：

```text
它已經在非常早的時代，
就把文學最根本的問題提出來了：
人如何面對死亡？
```

---

## 一、吉爾伽美什是什麼？

首先要知道，  
《吉爾伽美什史詩》不是小說。  
它也不是現代意義上的單一作者作品。  
它是一部來自古代美索不達米亞（Mesopotamia）的史詩，  
最完整的現存文本主要以：

```text
阿卡德語（Akkadian）
```

形式保存在泥版上。  

如果用最簡單的方式介紹它，  
可以說它是：

- 現存最早的重要文學作品之一
- 最早的重要英雄史詩之一
- 古代近東世界最有影響力的敘事文本之一

但如果只是這樣介紹，  
還不夠。  

真正重要的是：  
它不是單純在寫一個強大的英雄做了什麼，  
而是在很早很早的文明時代，  
就已經開始思考：

- 人會死這件事意味著什麼
- 力量能不能戰勝死亡
- 友誼如何改變一個人
- 文明是否能替代個體不朽
- 人如何在有限中理解生命

所以《吉爾伽美什史詩》的地位，  
不只是文學史上的“早期作品”，  
它更像是：

```text
人類第一次大規模把死亡焦慮寫進文學
```

---

## 二、這部史詩的真正核心，不只是英雄，而是「有限的人」

一開始讀《吉爾伽美什史詩》，  
你可能會以為它是典型的英雄故事。  

確實，吉爾伽美什（Gilgamesh）  
是一位非常典型的早期英雄人物：

- 強大
- 驕傲
- 近乎神一般
- 擁有王權
- 充滿行動力

但如果只把它讀成  
“古代英雄冒險故事”，  
就會錯過它最深的地方。  

因為這部史詩真正要處理的，  
不是英雄如何征服世界，  
而是：

```text
即使是英雄，也不能逃離死亡。
```

這一點非常關鍵。  
因為它意味著：  
文學從一開始，  
就不是只在歌頌力量。  
它更在處理：

```text
力量的界限
```

而這個界限，  
就是死亡。

---

## 三、故事的三段核心結構

如果要把《吉爾伽美什史詩》的主體結構  
先簡化成三大段，  
可以這樣理解：

### （1）暴君吉爾伽美什：權力與失衡

故事一開始的吉爾伽美什，  
並不是成熟的英雄，  
而更像是一個失衡的統治者。  

他擁有巨大權力，  
但這種力量一開始不是秩序來源，  
反而變成壓迫來源。  
也就是說：

```text
力量沒有自動帶來正義，
反而可能先帶來失序。
```

這是文學在非常早期  
就已經提出的一個重要政治問題：

```text
權力是否必然導致失衡？
```

這裡很重要的一點是，  
史詩沒有把王權自動神聖化。  
相反，它先讓你看到：

- 統治可能過度
- 力量可能失控
- 王者也需要被限制
- 英雄未必一開始就是值得歌頌的人

這意味著《吉爾伽美什史詩》  
不是單純地崇拜英雄，  
而是在追問：

```text
英雄如何成為真正的人？
```

---

## 四、恩奇都（Enkidu）的出現：另一種人類可能性

為了對應吉爾伽美什的失衡，  
神創造了：

```text
恩奇都（Enkidu）
```

恩奇都非常重要。  
因為他不是普通配角，  
而是整部作品的關鍵鏡子。  

他一開始是：

- 自然中的人
- 未被文明規訓的人
- 與動物共處的人
- 不屬於城市秩序的人

如果說吉爾伽美什代表的是：

```text
文明、王權、制度化力量
```

那麼恩奇都一開始代表的就是：

```text
自然、原初生命、未馴化存在
```

這個對照非常關鍵。  
因為人類文學在這裡第一次清楚呈現出：

```text
人不只有一種存在方式
```

也就是說，  
吉爾伽美什不是唯一的人類模型。  
恩奇都的出現讓作品第一次打開了一個問題：

```text
文明真的是唯一正確的人類狀態嗎？
```

這是非常早期、  
但已經非常深的文明問題。

---

## 五、對抗變成友誼：人如何被另一個人改變

恩奇都與吉爾伽美什  
最初不是朋友，  
而是對抗者。  

但正因為他們勢均力敵，  
兩人才可能真正看見彼此。  
這種從衝突轉向友誼的過程，  
其實是整部史詩最重要的心理轉折之一。  

因為這代表：

```text
一個人第一次不只在自己裡面活著，
而是在另一個人身上看到自己的界限與可能。
```

所以恩奇都的意義  
不只是“朋友”這麼簡單。  
他更像是：

- 吉爾伽美什的鏡子
- 吉爾伽美什的限制者
- 吉爾伽美什成長的條件
- 讓他第一次變成人的他者

可以說，  
如果沒有恩奇都，  
吉爾伽美什可能永遠只是一個  
過度膨脹的強者。  

真正讓他開始改變的，  
不是戰鬥勝利，  
而是：

```text
友誼
```

這一點非常重要。  
因為它表示人類最早的偉大文學之一，  
已經在說：

```text
人成為自己，
常常不是靠孤立力量，
而是靠與另一個人的關係。
```

---

## 六、恩奇都之死：故事真正從這裡開始

如果說前半部是英雄與友誼，  
那麼整部史詩真正最深的部分，  
其實是從：

```text
恩奇都之死
```

才真正開始。  

這裡是整部作品的核心斷裂點。  
因為在恩奇都死之前，  
死亡還比較像一個抽象背景。  
但在恩奇都死之後，  
死亡第一次變成：

```text
不可逃避的個人經驗
```

吉爾伽美什在這裡第一次真正明白：

```text
不是別人會死，
而是我也會死。
```

這是一個巨大的轉折。  
因為文學在這一刻  
不再只是講英雄事蹟，  
而是開始深入人的內部。  

可以說：

```text
恩奇都的死，
把吉爾伽美什從力量世界
推進了有限生命的世界。
```

所以你原稿說得很準：  
故事真正開始，  
其實是從死亡開始。

---

## 七、吉爾伽美什真正問的是什麼？

整部《吉爾伽美什史詩》  
如果最後只濃縮成一個問題，  
那就是：

```text
人如何面對死亡？
```

但這個大問題，  
其實又可以拆成幾個層次。

### （1）人可以逃離死亡嗎？

吉爾伽美什後半部的旅程，  
幾乎都在圍繞這件事。  
他尋找永生，  
尋找不朽的秘密，  
尋找超越凡人命運的方法。  

而作品給出的答案是：

```text
不能
```

這個答案非常殘酷，  
但也非常成熟。  
因為它沒有用神話式方便解法  
把人從死亡裡救出來。  
它反而讓人正面承認：

```text
死亡是人的條件之一
```

### （2）那人能靠功業延續生命嗎？

這部作品的答案不是簡單否定。  
它其實會走向一種更微妙的答案：

```text
個體不能永生，
但人的名字、城市、記憶、文明成就，
可能以另一種方式被保存。
```

所以這裡的不朽  
已經開始從“生物不死”  
轉向：

```text
文化留存
```

### （3）那活著的意義是什麼？

作品最後並沒有給一套明確哲學公式。  
但它顯然把讀者推向一個理解：

```text
生命的意義
不在逃離有限，
而在理解有限之中的生活。
```

也就是說：

```text
活著的問題，
不是怎樣不死，
而是怎樣在必死中仍然活得像人。
```

## 八、這裡發生了文學史上的第一個巨大轉折

你原稿裡有一個很好的觀察：  
到了《吉爾伽美什史詩》，  
人類意識開始內向化。  
這個判斷非常重要。  

如果非常簡單地對照，  
可以先這樣看：

| 早期神話傾向 | 《吉爾伽美什史詩》的推進 |
|---|---|
| 神主導世界 | 人開始正面面對自己的處境 |
| 世界秩序是主要焦點 | 個體死亡成為核心問題 |
| 宇宙較偏向循環與神意 | 人的有限性被突出 |
| 人是整體秩序中的一環 | 人開始感受到「我會死」的焦慮 |

這裡發生的事情非常深：  

```text
人類第一次不只問世界怎麼運作，
還開始問：
我在這樣的世界裡該怎麼活？
```

這就是你說的“內向化”。  
也就是說，  
文學從此開始真正進入：

```text
人的內在問題
```

## 九、《吉爾伽美什史詩》與一般神話有何不同？

這裡必須非常精確。  
《吉爾伽美什史詩》當然仍然處在神話世界裡。  
它有神、夢兆、超自然旅行、洪水故事、神人混合角色。  

所以不能簡單說它“離開了神話”。  
但它確實做了一件重要的推進：

```text
它把神話材料，
轉化成對人類有限性的深度追問。
```

這是關鍵。  

在較典型的神話世界裡，  
死亡常常是宇宙秩序的一部分，  
它被放在更大的神意框架裡。  

但在《吉爾伽美什史詩》中，  
死亡第一次非常鮮明地成為：

```text
一個人必須親自承受的不可逆事件
```

所以更準確的說法不是：

```text
神話世界裡沒有死亡問題
```

而是：

```text
《吉爾伽美什史詩》讓死亡第一次被深刻地內在人化、個體化。
```

這一點，  
正是它為什麼會被後世一直視為  
人類文學起點之一的重要原因。

## 十、文學是否從死亡開始？

你原稿裡有一句非常強的話：

```text
文學不是從故事開始，
而是從死亡開始。
```

這句話作為課堂總結非常有力量。  
如果要更學術精確一些，  
可以稍微修正為：

```text
文學不只是從講故事開始，
而是從人類開始需要用故事理解死亡、有限性與失去開始。
```

因為沒有死亡意識，  
人的生命就很難獲得那種  
迫切需要被理解、被整理、被傳述的張力。  

也就是說：

- 如果生命是無限的，很多問題就不急迫
- 正因為生命有限，人才需要問意義
- 正因為會失去，人才需要記憶
- 正因為必死，人才需要文明、名字、作品與故事

所以這句話真正的意思是：

```text
死亡讓敘事變得必要，
而敘事讓有限生命得以被承受。
```

## 十一、最後回到烏魯克（Uruk）：真正的不朽是什麼？

《吉爾伽美什史詩》的結尾非常重要。  
吉爾伽美什最終沒有獲得個體永生。  
他沒能打敗死亡，  
也沒能成為例外。  

但故事最後，  
他回到：

```text
烏魯克（Uruk）
```

重新看見城牆、城市與文明成果。  

這個結尾不是失敗式結尾，  
而是一種深刻的轉向。  
因為它告訴我們：

```text
個人會死，
但文明可以留下。
```

這裡第一次出現了一個  
非常重要的文學與文明觀念：

```text
永生不再只是神話問題，
而開始變成文明問題。
```

也就是說，  
真正可留下的未必是肉身，  
而可能是：

- 城市
- 制度
- 記憶
- 名字
- 故事
- 作品
- 文化

所以《吉爾伽美什史詩》的結尾  
其實不是單純的失落，  
而是把“不朽”的概念重新定義了。  

從這裡開始，  
文學第一次提出一個極其重要的觀念：

```text
個體的不朽也許不可能，
但文化的不朽可能部分成立。
```

## 十二、為什麼《吉爾伽美什史詩》是整條課程最好的起點？

因為它幾乎一次把整條文學課  
後面會一直反覆出現的主題  
全部先種下來了：

- 英雄與權力
- 文明與自然
- 友誼與他者
- 死亡與有限性
- 失落與哀悼
- 尋找意義
- 名字、記憶與不朽
- 個人與城市
- 人如何接受自己的界限

也就是說，  
它雖然極古老，  
但它問的問題其實從未過時。  

你之後讀《伊利亞德》、  
希臘悲劇、莎士比亞、  
陀思妥耶夫斯基（Fyodor Dostoevsky）、  
卡夫卡（Franz Kafka）甚至加繆（Albert Camus）時，  
都會重新碰到這裡已經出現的問題。  

所以這部作品之所以是起點，  
不是因為它古老，  
而是因為：

```text
它已經把文學最深的問題，寫出來了。
```

## 十四、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果人無法真正獲得永生，
那麼名字、城市、作品與記憶，
是否就是人類對死亡最早的回答？
```

這個問題現在不用急著回答。  
但它會一路通往後面的：

- 《伊利亞德》中的英雄榮耀
- 希臘悲劇中的命運與毀滅
- 小說中的個體生命
- 現代文學中的虛無與存在焦慮

因為從《吉爾伽美什史詩》開始，  
文學就已經在問：

```text
人明知必死，
為什麼還要留下痕跡？
```
