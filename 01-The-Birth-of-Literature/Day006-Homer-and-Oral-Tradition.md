# Day 006｜荷馬與口述傳統
## Homer and Oral Tradition

今天我們要做一個很重要的後退。  
不是往後退，  
而是往更深的起點退。  

如果 Day 004–Day 005  
讓你接觸的是最早的重要書寫史詩之一：

```text
《吉爾伽美什史詩》（Epic of Gilgamesh）
```

那麼今天我們要再往前問一層：

```text
在文字還沒有穩定成為文學主要載體之前，
文學到底如何存在？
```

答案是：

```text
透過口述傳統（Oral Tradition）
```

也就是說，  
今天我們要暫時離開：

```text
文本
```

回到：

```text
聲音
```

因為文學最初不是先作為“紙上的字”存在，  
而是先作為：

- 記憶中的節奏
- 聲音中的公式
- 表演中的敘事
- 共同體中的反覆傳唱

來存在。  

這一天非常重要。  
因為它會徹底改變我們對文學的直覺：

```text
文學最初不是閱讀物，
而是被聽見、被記住、被表演的東西。
```

## 一、什麼是口述傳統（Oral Tradition）？

很多人一聽到「口述傳統」，  
會立刻把它理解成：

```text
沒有文字的文學
```

這種說法不能說錯，  
但還不夠準。  

更精確地說，口述傳統不是單純“缺少文字”，  
而是一種完全不同的文學生存方式。  
它的核心不是：

```text
寫下來
```

而是：

```text
被記住、被重複、被演唱、被不斷再生成
```

所以口述傳統的文學，  
其存在方式通常具有幾個特徵：

- 沒有單一固定版本
- 每次講述都可能有變化
- 依靠節奏、重複與公式來支持記憶
- 依靠表演情境，而不是書頁穩定性
- 屬於共同體流傳，而不只是個人作品

也就是說，  
在口述傳統中，  
文學不是一個封閉文本，  
而是一種：

```text
活的敘事行動
```

它不是“寫完後保存”，  
而是“講述中保存”。  

這一點非常重要。  
因為它提醒我們：  
文字固定不是文學的自然起點，  
而只是後來的一種技術條件。

---

## 二、文學最初不是文字，而是聲音技術

今天我們太容易把文學理解成：

- 文字
- 書頁
- 印刷
- 閱讀
- 作者寫作

但在荷馬（Homer）所代表的世界裡，  
情況完全不同。  

那時文學更接近：

```text
記憶 + 聲音 + 身體 + 表演
```

這意味著什麼？  
意味著文學最初不是眼睛的藝術，  
而是耳朵的藝術。  

它依賴的不是書桌，  
而是：

- 口腔
- 呼吸
- 韻律
- 重複
- 現場
- 聽眾的接受與回應

所以如果要用一句話概括今天最重要的轉向，  
可以先記住：

```text
文學最初不是文字技術，
而是聲音技術。
```

這句話不是比喻。  
它是真的在說：  
口述文學的保存與流傳，  
依靠的是聲音組織能力，  
而不是書寫穩定性。

## 三、荷馬（Homer）到底是誰？

接下來就會遇到一個經典問題：

```text
荷馬（Homer）到底是誰？
```

這個問題其實不只是人物考據問題，  
而是整個西方文學史裡  
最有名的問題之一：

```text
荷馬問題（Homeric Question）
```

這個問題之所以重要，  
是因為當我們說“荷馬”時，  
我們到底是在說：

- 一位真實存在過的詩人？
- 一位天才作者？
- 一個長期口述傳統的象徵性名字？
- 一個後來才被統一歸屬的文化標誌？

學界大致有不同理解。  
簡化地說，  
可以先看到兩種主要方向。

### （1）單一作者理解

這種看法傾向認為：

- 荷馬是一位詩人
- 《伊利亞德》（Iliad）與《奧德賽》（Odyssey）可視為他的作品
- 作品具有高度整體性與藝術統一性

### （2）口述傳統理解

這種看法更強調：

- 荷馬史詩深受口述傳統影響
- 詩句中的大量公式、重複與類型場景，不像單純書面創作
- 史詩很可能是長期流傳、累積、演唱與整合的結果
- “荷馬”不必只理解為單一寫作者，也可以理解為傳統聚合的名稱

所以更成熟的說法不是：

```text
荷馬不是一個人
```

而是：

```text
「荷馬」既可能指傳統中的詩人形象，
也必須放在口述傳統的敘事系統中理解。
```

也就是說，  
荷馬的真正重要性  
不只在個人身份，  
更在於：

```text
他所代表的詩歌世界，
是口述文化與後來書面固定之間的關鍵交界點。
```

## 四、荷馬史詩不是先被寫出來，而是先被唱出來

這裡必須抓住一個核心觀念：

```text
荷馬史詩最初不是先作為文字寫作而存在，
而是先作為口頭演唱的詩歌而存在。
```

這一點非常重要。  
因為它會徹底改變我們理解《伊利亞德》與《奧德賽》的方式。  

如果你把它們當成現代書面小說，  
就會覺得：

- 為什麼那麼多重複？
- 為什麼人物常有固定稱號？
- 為什麼場景結構反覆出現？
- 為什麼語言有那麼多程式化痕跡？

但如果你把它們放回口述傳統裡，  
這些特徵就不再是缺點，  
而會變成：

```text
表演與記憶所必需的技術。
```

因此今天要記住一個非常關鍵的觀念：

```text
文學一開始不是寫作，
而是表演。
```

這不只是形式差異，  
而是整個文學觀念的差異。  
因為在口述文化中，  
作品不是先寫好再朗讀，  
而是常常在表演過程中  
被即時組織出來。

## 五、口述史詩如何運作？三個核心機制

口述文學不是隨便記、隨便講。  
它有一套非常精密的技術。  
最核心的，可以先抓三個機制。

### （1）公式（Formula）

這是理解荷馬史詩最關鍵的概念之一。  

像這樣的固定稱號你一定很熟悉：

- 灰眼的雅典娜（grey-eyed Athena）
- 多謀的奧德修斯（resourceful / many-minded Odysseus）
- 迅足的阿基琉斯（swift-footed Achilles）
- 玫瑰色手指的黎明（rosy-fingered dawn）
- 酒色般的海（wine-dark sea）

這些不是單純裝飾性修辭。  
它們真正的重要功能之一是：

```text
作為記憶與即興作詩的工具
```

根據口頭程式化理論  
（oral-formulaic theory），  
這些公式能夠在特定韻律位置中反覆使用，  
幫助歌者迅速完成句子、維持節奏、銜接敘事。  

也就是說，  
公式不是偷懶，  
而是一種高效率的口述技術。

### （2）重複（Repetition）

口述史詩裡大量出現重複句型、  
重複場景、重複描述，  
這對今天只用安靜閱讀的人來說  
可能會覺得冗長。  

但在口述世界裡，  
重複有幾個重要作用：

- 幫助記憶
- 穩定敘事節奏
- 給聽眾辨識感
- 讓重要場景獲得儀式性
- 形成共同期待

所以重複不是技術不足，  
而是口述世界裡  
組織文學的方式之一。

### （3）節奏與格律（Meter）

荷馬史詩最典型的是：

```text
六音步長短格（dactylic hexameter）
```

這種固定格律  
不是純形式美學而已。  
它還具有非常實際的功能：

- 幫助歌者維持流暢
- 幫助語句配入固定節拍
- 讓即興表演更可控制
- 讓聽者更容易跟上敘事

也就是說，  
格律不是附加在故事上的裝飾，  
而是整個史詩得以被口頭生成的重要骨架。

## 六、Milman Parry 與口頭程式化理論

20 世紀研究荷馬最重要的轉折之一，  
來自：

```text
米爾曼・帕里（Milman Parry）
```

以及後來的：

```text
阿爾伯特・洛德（Albert Lord）
```

他們的重要發現是：  
像荷馬史詩這樣的作品，  
可以透過“口頭程式化作詩法”  
（oral-formulaic composition）來理解。  

也就是說，  
史詩歌者不是背誦一份字字不變的固定文本，  
而是掌握一個龐大的傳統材料庫，  
包括：

- 公式語句
- 類型場景（type-scenes）
- 敘事模式
- 固定角色稱號
- 韻律結構

歌者在現場表演時，  
可以調用這些現成構件，  
快速又流暢地生成長篇敘事。  

這個理論的重要性非常大。  
因為它讓我們重新理解：

```text
史詩不是只能由單一作者閉門寫成，
也可能由高度成熟的口述系統現場生成。
```

這不但改變了荷馬研究，  
也改變了整個文學研究對  
“作者、文本、原作、版本”的看法。

## 七、《伊利亞德》與《奧德賽》的本質差異

既然今天談荷馬，  
也要先抓住這兩部史詩  
最基本的氣質差異。  

### 《伊利亞德》（Iliad）

關鍵詞是：

- 戰爭
- 憤怒
- 榮耀
- 即時衝突
- 英雄的毀滅性力量

它的核心動力  
是一種高強度、近距離、爆發性的張力。  
所以你可以把它理解成：

```text
爆炸型史詩
```

它關心的是：

- 衝突如何升高
- 榮耀與死亡如何纏在一起
- 英雄如何在戰場中被定義

### 《奧德賽》（Odyssey）

關鍵詞則是：

- 回家
- 延遲
- 漂泊
- 智慧
- 試煉
- 敘事的曲折與延展

它不像《伊利亞德》那樣  
集中在單一戰爭現場的高壓張力，  
而更像一條被不斷拉長的回返路線。  
所以你可以把它理解成：

```text
旅程型史詩
```

如果非常簡單地濃縮：

| 作品 | 氣質 |
|---|---|
| 《伊利亞德》 | 爆發、戰爭、憤怒、榮耀 |
| 《奧德賽》 | 延遲、回家、機智、漂泊 |

也就是說：

```text
一個是爆炸，
一個是旅程。
```

這樣的區分很粗略，  
但作為入門非常有力。

## 八、口述文化中的記憶世界

在沒有穩定書寫系統的世界裡，  
記憶不是附屬能力，  
而是文明的基礎設施。  

這時候可以說：

- 記憶就是檔案
- 重複就是保存
- 表演就是傳播
- 共同體就是資料庫

也就是說，  
人的腦袋不只是思考器官，  
還是：

```text
圖書館 + 劇場
```

這句話不是誇張。  
因為在口述文化裡，  
知識與故事不是儲存在紙上，  
而是儲存在：

- 歌者的記憶中
- 聽眾的熟悉感中
- 儀式與場合中
- 重複性的共同表演中

所以口述世界裡的文學  
本質上是一種：

```text
記憶共同體
```

這也就是為什麼  
口述文學特別強調節奏、重複、類型場景與固定稱號。  
因為這些不是“古老的笨拙手法”，  
而是文明在書寫之前  
最精密的記憶技術之一。

## 九、從口述到書寫：文學史的一個重大轉折

今天最重要的歷史性理解之一是：

```text
文學不是從寫作開始，
而是從記憶開始。
```

這句話意味著三件大事。

### （1）文本不是原始形式

我們今天讀到的書面版本，  
通常已經是後來固定下來的形式。  
原初狀態未必是一本書，  
而可能是一連串不斷被表演、修整、再組織的敘事實踐。

### （2）變異是正常的

在口述文化裡，  
每次講述略有不同，  
不是錯誤，  
而是正常現象。  

這表示：

```text
版本差異不是對原作的污染，
而是口述生命的一部分。
```

### （3）作者未必是單一個體

當作品來自長期口述傳統時，  
作者概念就不能完全照現代個人創作理解。  

所以更精確地說：

```text
「荷馬」既可以是詩人，
也可以是傳統在某一歷史節點上的集中命名。
```

這三點都很重要。  
因為它們會讓我們重新思考：  

- 什麼叫原作？
- 什麼叫版本？
- 什麼叫作者？
- 什麼叫文本穩定？

## 十、當史詩被寫下來，發生了什麼？

當口述史詩逐漸被寫下來，  
文學史其實發生了一次非常大的轉變。  

可以簡單理解成：

```text
口述
→
書寫
```

但這條線背後，  
其實包含幾個更深的變化：

| 口述傳統 | 書寫固定 |
|---|---|
| 流動 | 穩定版本 |
| 變異正常 | 開始追求定本 |
| 表演即生成 | 作品與表演分離 |
| 集體傳承 | 更容易歸屬於作者 |
| 記憶驅動 | 文本驅動 |

也就是說，  
當史詩被寫下來，  
文學開始慢慢變成  
我們今天比較熟悉的樣子：

- 有固定文本
- 有相對穩定版本
- 有可討論的作者
- 有可保存的完整作品

但不要忘記：  
這一切不是文學本來就有的自然狀態，  
而是歷史中的一次技術轉折。  

所以我們今天讀荷馬時，  
一方面是在讀文本，  
另一方面其實也應該記得：

```text
這些文本背後，
曾經有一個比文本更古老的聲音世界。
```

那麼當它被寫下來、固定下來之後，
