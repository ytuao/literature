# Day 143｜存在先於本質
## Existence Before Essence

今天我們進入  
存在主義最著名的一句話：

```text
Existence Before Essence
（存在先於本質）
```

提出者是：

```text
讓-保羅·薩特
（Jean-Paul Sartre）
```

這句話之所以重要，  
是因為它幾乎直接改寫了：

```text
人是什麼？
```

這個最古老的哲學問題。  

---

## 一、你不是“被製造好的東西”：問題真正從哪裡開始

今天的核心問題可以先寫成：

```text
人是先“被定義”，
還是先“存在”？
```

這句話非常重要。  

因為在很多傳統思路裡，  
人常常被想像成  
某種已經被定義好的東西。  
好像我們出生時  
就已經帶著：

- 固定用途
- 固定角色
- 固定人性
- 固定人生方向

但薩特要挑戰的  
正是這個前提。  

也就是說，  
他不是先問：

```text
人的本質是什麼？
```

而是先問：

```text
如果人根本沒有先被決定好的本質，
那麼“人是什麼”這個問題
是不是要整個重寫？
```

這正是  
Day 143  
最核心的起點。  

---

## 二、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
人不是帶著本質出生的，
而是在存在中逐漸成為自己。
```

這句話直接改寫  
“人是什麼”。  

但若更嚴格，  
最好補上：

```text
薩特（Jean-Paul Sartre）所說的
“存在先於本質”
（existence precedes essence），
主要不是否認人有身體、處境、歷史條件或穩定傾向，
而是反對那種認為人一出生就帶著一個預先寫好的人生定義、
固定目的或不可更動的人性腳本的看法。
```

這句話非常重要。  

同時也最好再補一句：

```text
“你就是你的行為總和”若要更精確，
應理解為：
在薩特這裡，
身份不是藏在行為背後的一個現成本體，
而是透過反覆的選擇、承諾與行動被具體呈現出來；
因此“我是誰”不能只靠自我感覺宣稱，
而要看我如何實際地活。
```

也就是說，  
薩特不是在說  
人沒有任何條件。  
而是在說：

```text
這些條件不能直接等於你的本質。
```

你的本質  
不是先寫好、  
等你去發現。  
它是在你活的過程中  
被做出來的。

## 三、傳統觀點：本質先於存在

為了理解薩特，  
最好先看他在反對什麼。  

在很多古典或傳統思路裡，  
人們會認為：

- 石頭有石頭的本質
- 刀有刀的用途
- 桌子有桌子的功能
- 人也應該有固定“人性”

這句話非常重要。  

這種思維的邏輯是：

```text
先有定義，
再有實例。
```

也就是說，  
某個東西之所以存在，  
是因為它先已經  
被規定成某種樣子。  

例如刀被製造出來之前，  
就已經有：

- 它是什麼
- 它做什麼
- 它該如何被使用

因此它的本質  
早於它的出現。  

而傳統思想  
常常也會這樣看人：  

```text
人應該也先有某種固定本質，
然後才來到世界上。
```

薩特要顛覆的，  
正是這個模式。

## 四、存在主義的顛覆：人沒有預設說明書

薩特最著名的顛覆  
就在這裡：

```text
人沒有預設說明書。
```

這句話非常重要。  

也就是說，  
人不是像一件工具那樣  
先被設計好用途，  
然後再被製造出來。  

相反地，  
人在薩特這裡  
是先被拋入世界、  
先存在起來，  
然後才透過：

- 選擇
- 行動
- 承擔
- 對世界的回應

慢慢形成自己是誰。  

因此人不是  
“先有定義，再有生活”。  
而是：

```text
先活著，
然後才在活的過程中
形成自己的樣子。
```

這就是  
“存在先於本質”  
最核心的意思。

## 五、人是“未完成品”：出生時沒有固定目的、角色與意義

你原文裡  
這個說法很好，  
而且非常適合保留：

```text
人是“未完成品”。
```

這句話非常重要。  

意思不是說  
人不成熟。  
而是說：

- 沒有固定目的
- 沒有固定角色
- 沒有固定意義
- 沒有寫好的最終版本

一開始有的，  
只是：

```text
存在本身。
```

也就是說，  
人不是一個  
內容已經填滿的東西。  
人更像是一個  
開放的工程。  

這也是為什麼  
存在主義的“人”  
總帶著一種  
不穩定性與未完成性。  

因為如果本質不是預設的，  
那麼人就永遠不是  
一個單純被讀取的答案。  
而是一個  
持續被活出的過程。

## 六、本質是後來形成的：它不是給定，而是透過選擇慢慢建立

你原文這句  
非常重要：

```text
本質不是給定的，
而是透過選擇慢慢建立。
```

這句話應該保留。  

這裡的意思是：  
本質不再被理解為  
出生時就藏在內部的東西。  
它反而更像是：

```text
你反覆做出的選擇
在時間中沉積後
形成的樣子。
```

這句話非常重要。  

因此“你是誰”  
不是一個  
可以完全脫離生活實踐  
來回答的問題。  

它要看：

- 你如何行動
- 你如何承諾
- 你如何對待他人
- 你如何面對壓力與選擇
- 你如何持續把自己活成某種方向

也就是說，  
本質不是起點。  
它更像是一種  
被生活慢慢寫出來的結果。

## 七、人是“自我設計的過程”：你不是被定義的人，而是定義自己的人

你原文中  
這個對比很好：

- 你不是被定義的人
- 你是不断定義自己的人

這句話非常重要。  

這正是薩特  
最強烈的自由觀  
與身份觀的交會點。  

也就是說，  
人的身份  
不是一個固定名詞。  
它更像一個動詞：

```text
成為。
```

你不是先有一個  
完整的“你”，  
然後再去把它找出來。  
你是在生活中  
一點一點把自己做出來。  

因此存在主義  
不鼓勵那種  
太輕易的說法：

- 我天生就是這種人
- 我本來就註定如此
- 我只是把內在的我表現出來

薩特會說：  
事情沒有那麼簡單。  
因為你一直都在  
透過行動  
繼續定義自己。

## 八、沒有“天生的你”：存在主義拒絕命運人格、固定性格本質與人生劇本

你原文這一段  
方向非常對，  
但最好精細地理解。  

存在主義確實會拒絕  
以下這些過度固定化說法：

- 命運人格
- 固定性格本質
- 天生人生劇本

這句話非常重要。  

但這不是說  
人沒有性格差異、  
沒有既定條件、  
沒有成長背景。  
而是說：

```text
這些東西都不能最後替代
你如何活出自己。
```

也就是說，  
存在主義要否定的  
不是所有差異。  
而是那種  
把人的可能性  
過早封死的說法。  

因此“沒有天生的你”  
更精確地理解為：

```text
沒有一個完全寫好、
不可更動、
等你去驗收的“你”。
```

這一點非常關鍵。  
因為它把身份  
從命運式理解中  
解放了出來。

## 九、每個行為都是建構：你做的每件事都在更新“你是誰”

你原文中  
這一句非常重要：

```text
你做的每件事，
都在更新“你是誰”。
```

這句話非常值得保留。  

這裡的“更新”  
不是社群媒體式的更新。  
而是說：

```text
身份不是固定檔案，
而是會被行動不斷改寫。
```

這句話非常重要。  

你怎麼說話、  
怎麼愛、  
怎麼逃避、  
怎麼承擔、  
怎麼在壓力下做決定，  
這些都不只是附帶情節。  
它們就在形成你。  

因此在薩特這裡，  
“我是誰”  
不能只透過內在自我描述  
來成立。  
它要在行動中  
被證明與被活出。  

這也是為什麼  
存在主義會如此重視：

- 承諾
- 行動
- 選擇
- 責任

因為這些正是  
本質生成的場所。

## 十、自自由變得更徹底：沒有本質，就沒有“參考答案”

你原文中  
這個推論非常重要：

```text
如果沒有本質，
你無法“參考答案”。
```

這句話非常重要。  

因為如果真有  
預設本質，  
那麼人遇到問題時  
就可能說：

- 我只要照本性活
- 我只要發現真正的自己
- 我只要找到命中注定角色
- 我只要回到被設計好的目的

但薩特的存在主義  
會直接切斷這條路。  

也就是說，  
一旦“本質先於存在”  
被否定，  
你就不能再期待  
有一份隱藏的正解  
在後面等你。  

這使自由  
變得非常徹底。  

因為沒有本質  
可以替你當答案，  
也沒有“真正的你”  
可以替你先決定方向。

## 十一、但也更沉重：你不能把“我是誰”推給任何東西

自由之所以變得沉重，  
正是因為：

```text
你不能把“我是誰”
推給任何東西。
```

這句話非常重要。  

你不能最後說：

- 這就是我的天性
- 這是我的命
- 我本來就只能如此
- 真正的我其實早就決定好了

因為存在主義會追問：

```text
那你怎麼活？
你做了什麼？
你讓什麼價值透過你被實現？
```

也就是說，  
身份不再是藉口。  
它變成責任。  

這也就是  
“存在先於本質”  
之所以不只是  
解放性的命題。  
它同時也是  
沉重的命題。  

因為它把  
“我是誰”  
從命運與本性那裡拿走，  
重新放回你的選擇裡。

提到卡繆的補充，


所以這裡最好理解成：  
卡繆與薩特  
在語言與理論架構上不同，

```text
人有一份現成說明書，

```

身份在存在主義裡

## 十四、語言與自我：當你說“我是誰”，你其實也在塑造自己

你原文裡  
這一點很有啟發性：

```text
你說出的“我是誰”，
本身就在塑造你。
```

這句話非常重要。  

因為自我認同  
不只是內心感覺。  
它也透過語言  
被說出來、  
被固定、  
被反覆強化。  

也就是說，  
當你說：

- 我就是這樣的人
- 我永遠不會改變
- 我天生不適合這種生活

自我建構的一部分。

所以 Day 143  
可以更精準地理解為：

```text
不只是行為在塑造你，
你對自己的語言也在參與塑造你。
```

但存在主義仍會堅持：  
最後真正有重量的，  
不是說法本身，  
而是它如何被活出來。

## 十六、沒有“背後的你”：存在主義為什麼拒絕一個隱藏的真我

這句非常尖銳：

```text
存在主義拒絕：
有一個真正的、隱藏的“真我”。
```

但最好精準理解。  

存在主義不是說  
人沒有內在世界。  
也不是說  
人沒有自我感受。  

它真正要反對的是：

```text
把真正的自我想成一個
```

這句話非常重要。

你不能說：

```text
真正的我在裡面，

```

在薩特看來，  
這種說法很危險。  
因為它容易把行動  
和身份切開。  

而存在主義正是要說：  

```text
你不能只靠內在感覺
就保住一個“真正的你”。
真正的你必須在生活中被活出來。
```

這也是  
“沒有背後的你”  
最準確的意思。

## 十八、這帶來兩種感受：自由感與不穩定感同時出現

這個雙重效果

所以“存在先於本質”  
不是只會讓人覺得自由。  
它也會讓人感到：

```text
我一直都在生成之中，
因此我也一直都不穩定。
```

這正是  
它既解放又讓人不安的地方。

## 十九、社會與本質衝突：社會總想固定你，但存在主義說那只是暫時建構

這個觀察非常好：

- 你應該如何行動
- 你應該成為什麼樣子

這有助於秩序穩定。  
但存在主義會提醒：

```text
這些分類不能最後等於人的本質。
```

也就是說，  
角色、身份、標籤  
可以很有力量，  
但它們不是永恆定義。  
它們仍然可能被：

- 改寫
- 拒絕
- 重新承擔
- 用新的方式活出

這也是為什麼  
存在主義常常會  
和社會規訓產生摩擦。  

因為它不願意  
讓人被完全固定。

## 二十一、今日核心總結（Day143）

今天請記住這幾句話：

```text
第一，
人沒有預設本質，只有存在本身；
但更精確地說，
薩特（Jean-Paul Sartre）所說的
“存在先於本質”
（existence precedes essence），
主要不是否認人有身體、處境、歷史條件或穩定傾向，
而是反對那種認為人一出生就帶著一個預先寫好的人生定義、
固定目的或不可更動的人性腳本的看法。
```

```text
第二，
身份不是發現，而是在選擇與行動中逐步建構；
“你就是你的行為總和”若要更精確，
應理解為：
在薩特這裡，
身份不是藏在行為背後的一個現成本體，
而是透過反覆的選擇、承諾與行動被具體呈現出來；
因此“我是誰”不能只靠自我感覺宣稱，
而要看我如何實際地活。
```

```text
第三，
“我是誰”不是答案，而是一個持續進行的過程；
這意味著人不是被定義好的東西，
而是未完成的、持續生成的存在，
必須在時間中不斷透過選擇與承擔把自己活成某種樣子。
```

```text
第四，
“存在先於本質”徹底改變了身份問題的理解方式；
它不再把身份看成等待被發現的內在真相，
而是把身份理解為在有限生命中透過行動逐步形成的存在作品。
```

```text
第五，
這個命題同時帶來自由與不穩定；
正因為沒有預設本質，
人才能重新定義自己；
但也正因如此，
人失去了固定身份所提供的穩定支點，
必須在未完成與不確定中生活。
```

```text
第六，
存在主義拒絕把社會角色、標籤與既定分類直接等同於人的本質；
社會可以暫時地命名你、安置你、分類你，
但這些都不能最後取代你如何透過實際生活去形成自己。
```

```text
第七，
在整個課程中，
Day 143 代表存在主義核心結構真正收束成形：
從無意義到自由，
從自由到責任，
再從責任推進到本質消解；
也就是從“世界沒有預設答案”
走向“人本身也沒有預設定義”。
```

## 二十二、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果薩特真正要說的，
不是“人可以任意想像自己是誰”這麼簡單，
而是“人沒有一個先於生活而完成的人生定義，
因此身份只能在選擇、承諾與行動中被逐步做出來”，
那麼接下來閱讀存在主義時，
是否最重要的能力之一，
就是不只問“真正的我是誰”，
而要問“我究竟透過什麼樣的生活實踐把自己活成了什麼樣子”，
以及“如果不存在一個已經完成、藏在行為背後的真我，
那麼我是否其實一直都在用今天的行動重寫我是誰”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看身份是不是內在本質
- 不只看自由是不是抽象能力
- 不只看自我是不是等待被發現的答案

而是進一步去看：

- 為什麼本質會被存在主義消解
- 身份如何在時間中被建構
- 行動如何比自我感覺更有重量
- 社會分類如何與自我生成衝突
- 文學與哲學如何一起把“成為自己”從名詞變成動詞

因為從今天開始，  
存在主義不再只是  
說“世界沒有預設意義”或“人必須自由負責”。  

更準確地說：

```text
從今天開始，
正式把“存在先於本質”理解成存在主義對人之身份問題的一次最根本改寫——
在這裡，
真正重要的已不再只是世界是否有答案，
而是人本身也不是一個帶著固定說明書出生的存在；
他沒有先於生活而完成的人生定義、
沒有不可更動的本性腳本、
也沒有一個只需被發現的內在真我，
因此“我是誰”不再是一個等待揭示的答案，
而成為一個必須透過選擇、承諾、行動與時間
持續被建造出來的過程；
而薩特之所以如此重要，
正因為他讓我們看見：
人不是被製造好的東西，
而是活著本身就在不斷生成自己的存在。
```
