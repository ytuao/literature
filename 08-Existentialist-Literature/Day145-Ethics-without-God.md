# Day 145｜沒有上帝的倫理
## Ethics without God

今天我們進入  
存在主義最尖銳的倫理問題之一：

```text
Ethics without God
（沒有上帝的倫理）
```

如果前面我們已經接受：

- 世界沒有預設意義
- 本質不存在
- 價值不再穩固
- 虛無可以成立

那麼問題就會變成：

```text
如果沒有上帝，
“應該”還從哪裡來？
```

這正是  
存在主義倫理  
真正困難的地方。  

---

## 一、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
當沒有終極權威時，
“應該”仍然存在，
但必須由人自己承擔。
```

這句話非常重要。  

但若更嚴格，  
最好補上：

```text
在薩特（Jean-Paul Sartre）那裡，
沒有上帝（God）
並不自動推出“什麼都可以”，
而是意味著沒有任何超越性權威
能替人預先提供道德指南；
因此倫理判斷不再是服從被給定的命令，
而是人在選擇中同時為自己、
也為“人應當如何存在”的形象負責。
```

這句話非常重要。  

同時也最好再補一句：

```text
加繆（Albert Camus）
並不是訴諸傳統宗教命令來反對暴力，
而是在反抗（revolt）、
限度（limits）
與人的共同脆弱性之中尋找倫理底線，
因此他特別警惕任何以歷史、理想或未來之名
為殺戮與壓迫辯護的做法。
```

也就是說，  
Day 145 的問題  
不是“有沒有宗教信仰”這麼簡單。  
而是：

```text
當外在終極保證消失後，
倫理還能不能成立？
```

## 二、傳統倫理的結構：為什麼“應該”曾經看起來像被給予的

你原文裡  
對傳統倫理結構的整理  
非常清楚，  
可以保留：

- 上帝 → 定義善惡
- 宗教 → 提供規則
- 絕對真理 → 提供標準

所以：

```text
“應該”是被給予的。
```

這句話非常重要。  

在這種框架下，  
倫理的力量  
不是主要來自人的創造。  
而是來自一個  
比人更高、更穩、更終極的來源。  

也就是說，  
當人問：

- 我應該做什麼？
- 什麼是善？
- 什麼是惡？
- 為什麼我不能這樣做？

答案通常可以被推回到：

```text
因為更高的權威已經規定了。
```

這就讓“應該”  
具有外部來源與保證。

## 三、現代斷裂：當外部來源消失，倫理為什麼突然變得尖銳

你原文裡  
這個轉折非常重要：

```text
當這些基礎消失，
倫理不再有外部來源。
```

這句話非常重要。  

這正是現代問題  
之所以如此尖銳的原因。  
因為當：

- 上帝不再作為普遍保證
- 宗教命令不再自動成立
- 絕對真理失去原有權威

那麼倫理就失去了  
那個最熟悉的支撐架構。  

也就是說，  
問題不再只是：

```text
我是否願意服從善？
```

而變成：

```text
如果沒有最後替我背書的來源，
那我憑什麼說這是善？
```

這就使倫理  
從規則問題  
變成了根基問題。

## 四、問題真正尖銳的地方：如果沒有上帝，誰有權說“應該”？

你原文裡  
這三個問題  
非常核心：

- 誰定義善？
- 誰定義惡？
- 誰有權說“應該”？

這句話非常重要。  

因為“沒有上帝的倫理”  
真正困難的地方  
不是生活技巧，  
而是合法性。  

也就是說，  
如果再也沒有  
終極立法者，  
那麼：

- “應該”會不會只剩主觀偏好？
- 道德會不會變成任意意見？
- 誰能防止“我喜歡”直接變成“我認為正當”？

這就是  
虛無主義之後  
倫理問題最危險的裂口。  

因此存在主義  
若要不是只剩空談，  
就必須回答：

```text
沒有上帝，
是否還可能有倫理？
```

## 五、存在主義的回答：不是“沒有倫理”，而是倫理必須從人類本身出發

你原文中  
這一句非常關鍵：

```text
存在主義不說：
沒有倫理
而說：
倫理必須從人類本身出發。
```

這句話非常重要。  

這裡的意思不是說  
人可以隨便訂規則。  
而是說：

```text
既然沒有超越性權威
替人預先寫好道德手冊，
那麼倫理就必須在人自己的選擇、
關係、承擔與共同世界中被建立。
```

也就是說，  
存在主義並不想  
回到傳統外部授權模式。  
但它也拒絕說：

```text
沒有神，就沒有道德。
```

相反地，  
它要挑戰的是：

```text
倫理是否能在沒有外部保證的條件下，
仍然被嚴肅地活出來？
```

這正是  
Day 145  
真正的核心。

## 六、薩特的立場：當你選擇時，你也在定義“人應該如何存在”

你原文中  
這個薩特式命題  
非常重要：

```text
當你選擇時，
你在定義“人應該如何存在”。
```

這句話非常值得保留。  

這和前面 Day 142  
談責任時  
其實是一條線。  

薩特的意思不是說  
你一個人的每個舉動  
都真的統計式決定全人類。  
而是說：

```text
當你選擇某種活法時，
你同時也在承認這種活法
是可以被肯定的。
```

這句話非常重要。  

因此沒有上帝之後，  
“應該”並沒有消失。  
它只是從：

```text
被上方宣告
```

轉變成：

```text
在人的選擇中被生成。
```

這就使每個選擇  
都天然帶有倫理重量。

## 七、你的選擇是倫理模型：每個行為都是一個“倫理示範”

你原文這一句  
很好，而且非常關鍵：

```text
每個行為都是：
一個“倫理示範”。
```

這句話非常重要。  

因為只要你行動，  
你就不只是完成一件事。  
你也同時在示範：

- 這樣對待人是否可以
- 這樣理解責任是否可以
- 這樣面對權力是否可以
- 這樣使用自由是否可以

也就是說，  
你的選擇  
會把某種“應該”  
具體演示出來。  

這裡的關鍵在於：  
沒有上帝之後，  
倫理不再以命令形式  
先於行動存在。  
它反而會在行動之中  
被不斷製造出來。  

因此你不是  
先知道全部標準  
才開始活。  
而是：

```text
你怎麼活，
標準就怎麼逐漸被顯現。
```

## 八、沒有外部裁判：神、絕對法則與最終審判不再替你背書

你原文裡  
這個結構整理  
非常重要：

- 沒有神
- 沒有絕對法則
- 沒有最終審判
- 只有人的選擇本身

這句話非常重要。  

這裡的意思不是說  
法律與制度完全沒有作用。  
而是說：

```text
它們都不能最後取代
你自己的倫理承擔。
```

也就是說，  
你不能再躲到一句話裡：

- 這是神要我做的
- 這是永恆真理規定的
- 最後審判會證明我對

在存在主義中，  
這些最後保證  
不再可用。  

因此倫理  
會變得異常透明：  

```text
最後是你自己
讓某種價值透過你被實現。
```

這也就是  
為什麼“沒有上帝的倫理”  
比“有規則的倫理”  
沉重得多。

## 九、卡繆的補充：沒有上帝，仍然不能把暴力與壓迫合理化

你原文中  
把卡繆放進來  
非常必要。  

因為如果只說薩特，  
倫理容易顯得  
過度依賴主體選擇。  
卡繆則補上了  
另一條重要線索。  

```text
即使沒有上帝，
也不能合理化暴力或壓迫。
```

這句話非常重要。  

但更精確地說，  
卡繆不是說  
因為某個神聖命令禁止，  
所以不能這樣做。  
而是說：

```text
當人以歷史、理想、革命、
未來幸福或抽象真理之名
合理化當下的殺戮與壓迫時，
倫理就已經被背叛。
```

也就是說，  
在沒有上帝的世界裡，  
底線不會因此消失。  
反而更需要被人  
清醒地守住。  

這也是卡繆  
比很多激進思想  
更重視限度  
（limits） 的原因。

## 十、“沒有上帝”不等於“什麼都可以”：這是最常見也最危險的誤解

你原文裡  
這一句非常重要：

```text
倫理缺失 ≠ 道德消失
```

這句話應該保留。  

很多人一聽到  
“沒有上帝的倫理”，  
就會立刻以為：

```text
那不就是什麼都可以？
```

但存在主義  
恰恰反對這個結論。  

這句話非常重要。  

因為沒有上帝  
並不意味著  
你被釋放出責任。  
它只意味著：

```text
你不能再把責任
交給更高權威。
```

也就是說，  
不是道德消失了。  
而是道德不再  
以外部命令形式  
被保證。  

這會讓倫理  
變得更危險，  
但也更真實。

## 十一、倫理變成責任，而不是規則：問題不再是“我被允許做什麼”，而是“我正在創造什麼世界”

你原文裡  
這組對比非常好：

- 不是：我被允許做什麼
- 而是：我在創造什麼世界

這句話非常重要。  

這正是  
存在主義倫理  
和傳統規則倫理  
最深的差別之一。  

在這裡，  
倫理不再只是一套  
允許與禁止的條文。  
它更像是一個  
面向世界的問題：

```text
我的選擇正在讓什麼樣的世界
透過我被具體實現？
```

也就是說，  
你怎麼對人說話，  
怎麼使用權力，  
怎麼面對他者的脆弱，  
怎麼處理暴力與利益，  
這些都不只是行為細節。  
它們都在建造世界。  

因此“應該”  
在存在主義裡  
會和責任緊緊綁在一起。

## 十二、“應該”變成自我生成：它不是被發現，而是在選擇中被創造

你原文中  
這一句非常關鍵：

```text
“應該”不是發現的，
而是創造的。
```

這句話非常重要。  

這句話如果讀淺了，  
容易被誤解成：

```text
那倫理就是任意亂造。
```

但其實不是。  

更準確地說：

```text
“應該”不再是從宇宙、
神學或絕對法則中被直接讀出來，
而是在人的選擇、
關係、責任與對世界的實際回應中
被逐步生成。
```

也就是說，  
它不是現成答案。  
但也不是純粹任性。  

它會受到：

- 他人的存在
- 痛苦的事實
- 暴力的後果
- 關係的回應
- 自由與責任的結構

所約束。  

因此“自我生成”  
不是任意，  
而是：

```text
在人無法逃避的存在條件中
被迫建構。
```

## 十三、最困難的地方：沒有標準答案可依賴，道德焦慮因此出現

你原文裡  
這一點抓得非常準：

```text
道德焦慮的來源
不是因為不知道規則，
而是：
所有規則都必須自己承擔責任。
```

這句話非常重要。  

這就是  
沒有上帝的倫理  
真正讓人不安的地方。  

在傳統結構裡，  
人可以說：

- 我只是服從規則
- 我只是執行命令
- 我只是依照更高標準行動

但存在主義會說：  

```text
你不能再這樣躲。
```

因為現在  
即使你採納某條規則，  
那也是你在採納。  
即使你遵守某種倫理，  
那也是你在承擔。  

所以焦慮  
不是來自資訊不足。  
而是來自：

```text
最後沒有誰能替你保證
你這次真的完全正確。
```

這正是  
現代倫理最沉重的焦慮來源之一。

## 十四、他人變得更重要：沒有上帝之後，倫理參照為什麼更轉向他者

你原文裡  
這一句非常值得展開：

```text
沒有上帝 → 只有他人作為倫理參照。
```

這句話方向是對的，  
但最好稍微精細化。  

更準確地說：

```text
當超越性權威退場後，
倫理更直接地回到人與人之間的關係、
脆弱性、承擔與相互影響上。
```

這句話非常重要。  

也就是說，  
他人之所以變得更重要，  
不是因為他人變成新上帝。  
而是因為：

- 你的選擇會落到他人身上
- 你的行動會改變他人的世界
- 你的自由可能擴張也可能壓縮別人的存在空間
- 倫理判斷越來越不能脫離具體的人際後果

因此沒有上帝之後，  
倫理並沒有蒸發。  
它反而更直接地  
落到了：

```text
你如何對待他人的存在。
```

```text
沒有上帝（God）

而是意味著沒有任何超越性權威
能替人預先提供道德指南；
因此倫理判斷不再是服從被給定的命令，
而是人在選擇中同時為自己、
也為“人應當如何存在”的形象負責。
```

```text
第二，
“應該”不再被給予，而是透過選擇不斷生成；
這意味著“應該”不再是從宇宙、宗教或絕對法則中
被直接讀出的現成答案，
而是在人的行動、關係、承擔與對他人的回應中
被具體地活出來、建立起來。
```

```text
第三，
每個行為都成為對“什麼是人”的倫理定義；
當你行動時，
你不只是在完成一件事，
你也同時在示範
“人可以如何對待他人”
“自由可以如何被使用”
以及“什麼樣的世界可以被允許透過我發生”。
```

```text
第四，
沒有上帝不等於沒有倫理，
而是倫理失去了外部保證；
這使倫理變得更困難、
更焦慮、
也更需要由個體清醒承擔，
因為再也沒有一個更高權威
能替你最後完成判斷與背書。
```

```text
第五，
卡繆的重要補充是：
即使沒有上帝，
也不能合理化暴力與壓迫；
但更精確地說，
他不是訴諸傳統宗教命令來反對暴力，
而是在反抗（revolt）、
限度（limits）
與人的共同脆弱性之中尋找倫理底線，
因此他特別警惕任何以歷史、理想或未來之名
為殺戮與壓迫辯護的做法。
```

```text
第六，
在沒有終極權威的情況下，
倫理會更直接地回到他人；
因為你的選擇會落到他人身上，
會改變他人的存在條件，
也會塑造共同世界對“人應該如何被對待”的理解，
所以倫理不再只是規則問題，
而更是關係問題。
```

```text
第七，
在整個課程中，
Day 145 代表存在主義開始真正從危機走向倫理重建：
從虛無、無意義、自由與責任一路推進，
最終抵達“在沒有上帝的世界裡，
人是否仍然可以對善惡、暴力、他人與應該作出負責判斷”的核心試煉。
```

## 二十、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果存在主義真正要說的，
不是“既然沒有上帝，一切都由個人任意決定”這麼簡單，
而是“正因為沒有終極權威替人預先提供道德指南，
所以人反而必須更清醒地為自己的選擇、
為自己所示範的人之形象、
以及為自己讓什麼樣的世界透過行動被實現而負責”，
那麼接下來閱讀存在主義倫理時，
是否最重要的能力之一，
就是不只問“有沒有規則”，
而要問“當規則失去最終保證之後，
我是否仍然能對他人的存在負責”，
以及“如果‘應該’不再是被給予的，
那麼我是否已經在每一次選擇中
不斷地參與生成它”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看沒有上帝是否導向虛無
- 不只看倫理是否需要宗教保證
- 不只看道德是不是一套規則

而是進一步去看：

- 為什麼“應該”會轉移到人的承擔上
- 薩特如何把選擇變成倫理生成
- 卡繆如何在無超越保證下守住限度
- 他人為什麼會成為倫理最直接的場域
- 文學與哲學如何一起逼問“當沒有上帝時，人還能否對世界說應該”

因為從今天開始，  
存在主義已經不再只是  
分析危機、自由或虛無。  

更準確地說：

```text
從今天開始，
正式把“沒有上帝的倫理”理解成存在主義對現代倫理危機的一次正面回應——
在這裡，
真正重要的已不再只是上帝是否退場、
價值是否失去外部根基、
虛無是否可能成立，
而是當所有終極權威都不再替人提供道德指南時，
“應該”是否仍然能被保留下來；
而存在主義之所以深刻，
正因為它試圖指出：
倫理並不會因上帝的沉默而自動消失，
相反，
它會以更沉重的形式回到人自身，
要求人為自己所選擇的價值、
所示範的人之形象、
以及所創造的共同世界負責。
```
