# Day 026｜《變形記》與變形
## Metamorphoses and Transformation

今天我們進入羅馬文學中  
最奇特、也最深遠的一部作品：

```text
《變形記》（Metamorphoses）
```

昨天我們已經認識了：

- 奧維德（Ovid）
- 欲望（Desire）
- 情感（Emotion）

而今天，  
我們要進入他的代表作。  

這本書對整個西方文學影響極大。  
後來的：

- 但丁（Dante Alighieri）
- 莎士比亞（William Shakespeare）
- 卡夫卡（Franz Kafka）

都在不同層面上受到它的深刻影響。  

如果說《埃涅阿斯紀》（Aeneid）  
要為羅馬建立秩序、起源與歷史使命，  
那麼《變形記》則幾乎在提醒我們：

```text
世界沒有任何形式能被永遠固定。
```

這就是今天最重要的起點。

---

## 一、什麼是「變形」？真正的問題不是變成什麼，而是為什麼會改變

Metamorphosis 的基本意思，  
就是：

```text
形態改變
```

在《變形記》裡，  
你會看到許多著名場景：

- 人變成樹
- 人變成鳥
- 人變成石頭
- 人變成星星
- 人變成花
- 人變成動物
- 人變成水流或自然景物

表面看起來，  
這似乎只是一部充滿奇異神話的故事集。  
但如果只把它當成“神怪傳說”，  
就會完全低估奧維德。  

因為他真正關心的不是：

```text
一個人最後變成了什麼
```

而是：

```text
一個人為什麼會改變？
```

這句話非常重要。  
因為它把“變形”從魔法效果  
轉成了存在問題。  

也就是說，  
奧維德在問：

- 欲望如何改變人？
- 恐懼如何改變人？
- 創傷如何改變人？
- 愛情如何改變人？
- 被觀看、被追逐、被傷害之後，
  一個人還是不是原來那個人？

這才是《變形記》真正深的地方。  

---

## 二、《變形記》的世界觀：世界沒有永遠固定的形式

《變形記》整體有一個非常重要的核心假設：

```text
世界沒有永遠固定的形式。
```

這不只是書裡很多角色會變形。  
更重要的是，  
整部作品在告訴你：

- 人會改變
- 愛情會改變
- 身分會改變
- 權力會改變
- 記憶會改變
- 國家會改變
- 連神也不是永遠穩定不動的

也就是說，  
變化不是偶然事件，  
而是世界運作本身的一部分。  

這個世界觀和《埃涅阿斯紀》  
形成鮮明對照。  
在維吉爾那裡，  
文學努力把世界導向：

- 穩定
- 建立
- 秩序
- 帝國
- 可被承繼的歷史

但在奧維德這裡，  
文學提醒你：

```text
沒有任何建立，
能徹底終止變化。
```

這是羅馬文學內部  
非常深的一個反差。

---

## 三、與《埃涅阿斯紀》的對比：一個追求穩定，一個揭示流動

如果把兩部作品放在一起看，  
差異會非常清楚：

| 《埃涅阿斯紀》（Aeneid） | 《變形記》（Metamorphoses） |
|---|---|
| 穩定（Stability） | 流動（Fluidity） |
| 秩序（Order） | 變化（Change） |
| 建立帝國 | 展示不確定性 |
| 歷史使命 | 存在流變 |
| 把世界固定成敘事 | 顯示世界難以被完全固定 |

維吉爾更像在說：

```text
世界應該被建立。
```

而奧維德則像在提醒：

```text
世界永遠無法完全固定。
```

這兩種聲音都屬於羅馬。  
而正因為它們互相拉扯，  
羅馬文學才真正變得偉大。  

《埃涅阿斯紀》相信  
歷史可以把世界整合成秩序；  
《變形記》則反覆展示  
情感、欲望、暴力與時間  
如何讓一切重新流動。  

所以從今天開始，  
你要開始看到一件事：

```text
羅馬文學不只是建立文明，
它也深知文明底下存在著不穩定性。
```

## 四、最著名的故事之一：阿波羅與達芙妮

《變形記》中  
最著名的故事之一就是：

```text
Apollo（阿波羅）與 Daphne（達芙妮）
```

阿波羅愛上了達芙妮。  
但達芙妮不願意。  
她拒絕他、逃離他，  
而當她即將被追上時，  
向神祈求幫助。  

結果發生了最著名的變形之一：

```text
她變成了一棵月桂樹。
```

這個故事之所以重要，  
不只是因為它在藝術史中極其著名，  
更因為它幾乎濃縮了  
《變形記》的核心問題。  

表面上，  
這是一個神追逐少女、  
少女最終變成樹的神話。  
但更深層地看，  
它其實在問：

```text
當一個人無法保護自己時，
她還能保留什麼？
```

這才是這個故事  
真正令人震動的地方。

---

## 五、達芙妮的故事真正說什麼？變形不只是懲罰，也可能是保存

如果只從表面看，  
達芙妮失去了人形，  
看起來像是極端的不幸。  

但奧維德的高明之處在於：  
他讓變形不只是“毀掉一個人”。  
在某些情況下，  
變形也可能是一種保留。  

達芙妮確實失去了原來的身體形式，  
但某種更重要的東西  
卻仍然被保住了：

```text
她的拒絕
她的意志
她對自己邊界的堅持
```

所以在這裡，  
變形不是單純懲罰，  
也不是單純逃避。  
它更像是：

```text
另一種存在方式。
```

這句話非常重要。  
因為它說明奧維德筆下的變化  
不是“從有到無”，  
而是“從一種存在轉為另一種存在”。  

這也正是《變形記》  
非常現代的一點：  

```text
形式改變了，
但某些內在痕跡仍然留下。
```

也就是說，  
變形不是抹除，  
而是帶著殘留的改寫。

---

## 六、身分開始變得不穩定：人真的有固定身分嗎？

古典世界很多時候  
傾向假設一個人有某種相對固定的身分。  

例如：

- 英雄就是英雄
- 國王就是國王
- 神就是神
- 人就是人

當然，  
前面悲劇已經開始讓這些角色  
出現裂縫。  
但到了《變形記》，  
這種穩定性被更徹底地懷疑。  

奧維德開始逼問：

```text
身分真的固定嗎？
```

這是一個極其現代的問題。  
因為它不只是在說  
“人會換工作、換位置”。  
而是在更深的層面問：

- 你的形體改變了，你還是你嗎？
- 你的欲望改變了，你還是你嗎？
- 你的記憶、關係、角色改變了，
  身分會不會整體鬆動？
- 所謂“我”，究竟是固定本質，
  還是不斷變動中的暫時構成？

從這一點看，  
《變形記》比很多表面上更“現代”的文本  
還要激進。  

因為它直接動搖了  
“穩定身分”這個假設本身。

---

## 七、欲望導致變形：真正改變人的，不只是魔法，而是情感

在《變形記》的許多故事裡，  
變形往往都不是無緣無故發生。  
它常常來自某種強烈情感。  

例如：

- 愛情
- 嫉妒
- 憤怒
- 恐懼
- 羞辱
- 悲傷
- 被追逐或被拒絕的經驗

也就是說，  
真正推動變形的，  
往往不是單純神力本身，  
而是：

```text
情感先改變了人的存在，
神話只是把這個改變外化成形體。
```

這句話非常關鍵。  
因為它讓我們看見：

```text
變形不是單純的魔法效果，
而是情感的存在論後果。
```

也就是說，  
奧維德其實在用神話形式處理一個很現代的問題：

```text
情感如何改變一個人？
```

這也正是為什麼  
你昨天講“欲望讓人不再固定”  
會在今天得到更完整的展開。  

因為在《變形記》裡，  
情感不是附屬物，  
它真的會把“你是誰”都改掉。

---

## 八、世界是一條流動的河流：存在不是固定物，而是過程

如果一定要把奧維德的宇宙觀  
濃縮成一句話，  
那就是：

```text
沒有什麼永遠保持原樣。
```

但這句話如果再往深一點推，  
可以變成：

```text
存在不是固定物，
而是過程。
```

這是非常重要的轉向。  
因為它改變的不只是神話敘事，  
而是整個對“世界”的理解方式。  

也就是說，  
在奧維德那裡：

- 生命不是穩定物件
- 身分不是封閉盒子
- 世界不是完成品
- 一切都處於流動與生成之中

這樣的觀念  
後來會一路影響到：

- 現代哲學
- 現代心理學
- 現代小說
- 身分與主體的流動性問題

所以《變形記》真正驚人的地方，  
就在於它用最古老的神話形式，  
提前說出了非常現代的東西。

---

## 九、為什麼後世如此喜歡《變形記》？因為它提供了無限可重讀的象徵

《變形記》之所以被後世反覆閱讀，  
原因非常清楚：  

```text
它提供了無數可以被重新詮釋的象徵。
```

變形這個母題  
幾乎天然就能被不同時代重新使用。  

例如它可以代表：

- 成長
- 青春期
- 創傷
- 愛情
- 記憶
- 社會身份轉換
- 羞辱
- 死亡
- 失語
- 性別與身分流動
- 人與非人的邊界崩解

所以每個時代  
都可以重新解讀奧維德。  

這也是為什麼  
莎士比亞、  
後來的現代主義作家、  
甚至當代關於身分與身體的討論，  
都還會一次次回到《變形記》。  

因為它不是只講某些古老神話，  
而是在提供一種極有彈性的思考模型：

```text
當人改變了，
那還算不算同一個人？
```

這個問題永遠不會過時。

## 十、與卡夫卡的連結：奧維德問變成別的東西意味著什麼，卡夫卡把這問題推到極端

你前面提到卡夫卡（Franz Kafka），  
這個連結非常重要。  

未來學到《變形記》  
與卡夫卡的《變形記》（The Metamorphosis）時，  
你會立刻發現  
兩者之間其實存在一條非常清楚的文學血脈。  

奧維德問的是：

```text
人變成別的東西意味著什麼？
```

而卡夫卡把這個問題  
推到更加恐怖、更加現代的極端：  

```text
如果人突然變成怪物呢？
```

或者更準確地說：

```text
如果一個人醒來後，
已經不再能被世界承認為“人”，
那他剩下的是什麼？
```

你會發現，  
這其實仍然是奧維德的問題，  
只是被移到了現代性、  
異化、家庭關係與存在恐懼的語境裡。  

所以可以先記住：

```text
卡夫卡不是突然發明“人變形”；
他是在現代世界裡，
重寫奧維德早就提出的存在問題。
```

## 十一、《變形記》的核心思想：世界的本質不是穩定，而是變化；但變化中仍殘留某種連續性

如果一定要用一句話  
總結《變形記》的核心思想，  
你原本那句非常準：

```text
世界的本質不是穩定，而是變化。
```

但今天要再把它講得更完整一點：

```text
世界的本質不是穩定，而是變化；
然而在變化之中，
某些欲望、記憶、痛苦與意志
又常常以另一種形式延續下去。
```

這一點非常重要。  
因為它使奧維德的“變形”  
不只是炫目的形式遊戲，  
而是一種極深的存在洞見。  

也就是說：

- 形式會變
- 身分會鬆動
- 但內部痕跡不一定消失
- 變化未必等於斷裂
- 改變中可能仍有殘留的“同一性”

這就讓《變形記》  
與單純的奇幻敘事徹底區分開來。  

同時，  
它也和羅馬帝國追求穩定、秩序、固定身份的傾向  
形成尖銳對照。  

奧維德始終在提醒讀者：

```text
沒有任何制度
能徹底固定人性與存在的流動性。
```

這就是《變形記》  
最深的一層力量。

## 十三、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果一個人的形體、角色、身份甚至人生位置都會改變，
但某些欲望、痛苦與記憶又仍然殘留，
那麼真正構成「我」的，
究竟是我現在的形式，
還是那些在變化中始終沒有完全消失的東西？
```

這個問題現在不用急著回答。  
但它會直接把我們帶進：

- 奧維德（Ovid）更深的欲望與變形問題
- 莎士比亞如何把變形轉化為戲劇中的身份流動
- 卡夫卡如何把古典變形推向現代異化
- 現代文學如何越來越強烈地處理不穩定自我

因為從《變形記》開始，  
文學已經非常清楚地提出：

```text
人不是固定物，
而是一個會被世界、情感與時間不斷改寫的存在。
```
