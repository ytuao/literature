# Day 003｜故事與人
## Story and Human

今天我們要問一個  
比前兩天更深的問題：

```text
是人創造故事，
還是故事創造人？
```

前兩天我們已經知道：

- 神話（Myth）是世界的早期理解方式
- 敘事（Narrative）是把事件變成意義的工具

但今天要再往前一步。  
如果沒有故事，  
那麼：

```text
「人」還成立嗎？
```

這就是今天真正的核心。  
因為文學真正關心的，  
從來不只是世界發生了什麼，  
而是：

```text
人如何在故事中成為一個可理解的人
```

---

## 一、為什麼人離不開故事？

先看一個很簡單的例子。  
同一件事發生在兩個人身上：

### A 的說法

```text
我失敗了，
我是一個沒用的人。
```

### B 的說法

```text
我失敗了，
但我學到了東西。
```

事實可能幾乎一樣。  
但最後出現的是兩個完全不同的人。  

差別不在事件本身，  
而在於：

```text
事件被放進了什麼故事裡
```

也就是說，  
人不是單純活在事件裡，  
而是活在：

```text
對事件的解釋、
排列、
連接與命名之中
```

這些東西合起來，  
就構成了一種敘事。  

所以今天第一個要記住的觀念是：

```text
人類不是只經歷事情，
而是會把事情變成故事來活。
```

---

## 二、Self（自我）不是一塊固定物，而是一條敘事線

我們平常很容易以為：

```text
「我」是一個固定存在的東西
```

好像“我”就像桌子、石頭那樣，  
本來就在那裡，  
只是等著被認出來。  

但文學理論、敘事理論、  
甚至很多心理學與哲學討論都會提醒我們：

```text
自我（Self）
往往不是先天完整地擺在那裡，
而是在時間中被持續組織出來的。
```

如果要把這件事先簡化，  
可以先這樣理解：

```text
自我 = 一條持續被講述的故事線
```

這條故事線通常包括三個部分：

- 過去：記憶（Memory）
- 現在：經驗（Experience）
- 未來：期待、計畫、想像（Expectation / Projection）

也就是說，  
所謂「我是誰」，  
其實不是一個瞬間答案，  
而是一種把人生不同時間點  
串起來的方式。  

因此更準確地說：

```text
人不是只有時間中的經驗，
人還會把時間中的經驗
整理成「關於我是誰」的敘事。
```

## 三、Narrative Identity（敘事身份）是什麼？

這裡可以補上一個很重要的理論概念：

```text
敘事身份（Narrative Identity）
```

這個概念通常用來說明：  
人之所以能感覺自己是同一個人，  
不是因為自己永遠不變，  
而是因為能把變化中的人生  
講成某種仍然可理解的整體。  

換句話說，  
人會透過敘事來完成下面幾件事：

- 把分散的事件串連起來
- 把偶然經歷解釋成生命歷程
- 把痛苦轉化為某種可承受的意義
- 把過去、現在與未來放進同一條線上

所以“我是誰”  
很多時候其實是在回答：

```text
我的人生可以被講成一個什麼樣的故事？
```

這裡要注意，  
這並不是說人可以隨便虛構自己。  
而是說：

```text
人的自我理解，
幾乎總要透過某種敘事形式才能成立。
```

這一點非常重要。  
因為它把“人”從一個靜止物體，  
變成一個時間中的、可解釋的存在。

---

## 四、沒有敘事的人會怎樣？

如果把這個問題反過來問：

```text
如果一個人失去敘事能力，
會發生什麼？
```

那答案會非常嚴重。  

當一個人無法把經驗串起來時，  
他失去的不只是某段記憶，  
而更可能是：

```text
自我連續性
```

也就是：

- 不知道自己為什麼在這裡
- 無法理解現在和過去的關係
- 無法把事件組織成因果
- 無法把痛苦放進一條可以承受的生命線
- 無法回答「我是誰」

所以問題不只是：

```text
失去記憶
```

而更深的是：

```text
失去把自己理解為「一個人」的能力
```

這也就是為什麼  
記憶、敘事與身份會彼此連在一起。  
因為一個人若不能說出自己的故事，  
那麼“自己”就會開始鬆動。

## 五、人不是單純的生物個體，而是可被講述的存在

如果更往深一點講，  
今天的問題其實在挑戰一個很根本的想法。  
我們平常以為人首先是：

- 生物學上的個體
- 心理上的主體
- 社會中的角色

這些當然都沒有錯。  
但文學會補上一個非常重要的角度：

```text
人還是一種可被講述的存在
```

這是什麼意思？  
意思是：

- 人會替自己編織生命線
- 人會為自己的選擇找理由
- 人會回頭重寫自己的過去
- 人會預演自己的未來
- 人會把偶然經歷說成命運轉折

也就是說，  
人不是只“活著”，  
而是會把活著這件事  
敘事化。  

所以若要更精煉地說：

```text
人不是只存在，
人還會把自己的存在理解成一個故事。
```

## 六、文學如何處理「人」？

這裡就要進入文學本身。  
在文學裡，  
「人」從來不是固定不變的。  
不同文學形式，  
其實在生產不同版本的人。  

可以先粗略地這樣看：

| 文學階段 | 人的形式 |
|---|---|
| 神話 | 宿命秩序中的角色 |
| 史詩 | 行動中的英雄 |
| 悲劇 | 被命運撕裂的人 |
| 小說 | 社會中的個體 |
| 現代主義 | 破碎的意識 |
| 後現代 | 被語言與文本構造的存在 |

這張表非常重要。  
因為它告訴我們：

```text
人的定義，
會隨著敘事形式而改變
```

在神話裡，  
人比較像宇宙秩序中的一個位置。  

在史詩裡，  
人是能夠行動、征戰、承擔榮耀的英雄。  

在悲劇裡，  
人開始成為一個會分裂、會錯誤、會承受命運撕裂的存在。  

到了小說，  
人則進一步變成：

- 有家庭背景
- 有階級位置
- 有心理矛盾
- 有日常生活困境
- 有自我反思能力

再到現代主義與後現代，  
人甚至不再是完整穩定的主體，  
而可能只是：

- 記憶片段
- 意識流
- 語言效果
- 身份拼裝物

所以文學史不只是風格史，  
也是一部：

```text
「人」如何被不斷重新定義的歷史
```

## 七、故事不是描述人，而是在生成可理解的人

這一點非常關鍵。  
原稿說得很好，  
但可以再說得更深一點：

```text
故事不只是描述已經存在的人，
而是在生成一個可被理解的人。
```

這是什麼意思？  

例如《奧德賽》（The Odyssey）中的：

```text
奧德修斯（Odysseus）
```

他當然不是新聞意義上的真實人物。  
但他之所以重要，  
不是因為“是否真有其人”，  
而是因為他讓我們看到一種  
非常清晰的人類模型。  

他代表的不是單一個體，  
而是一組人類可能性：

- 智慧
- 策略
- 忍耐
- 漂泊
- 回家的慾望
- 面對誘惑時的自我維持

所以文學人物的意義  
不只是像真實人那樣“存在”，  
而是：

```text
他們讓某種人類經驗有了形狀
```

這也就是為什麼  
文學人物雖然虛構，  
卻常常比現實中的人  
更容易被理解。  

因為文學會把某種生命力量、  
某種矛盾、某種命運，  
集中、清晰、形式化地展現出來。

## 八、人為什麼需要故事？三個層次

如果把「人為什麼離不開故事」  
再整理得更清楚一些，  
可以分成三個層次。

### （1）認知層（Cognition）

故事幫助我們理解：

- 因果
- 動機
- 選擇
- 行動後果
- 人與人之間的關係

如果沒有故事，  
世界就只剩下碎片事件。  
而碎片本身很難被理解。  

所以在認知層上，  
故事是一種：

```text
把混亂經驗整理成模式的工具
```

### （2）身份層（Identity）

故事幫助我們回答：

```text
我是誰？
```

例如一個人可能會這樣理解自己：

- 我是很努力的人
- 我是跌倒後還能重來的人
- 我是被家庭影響很深的人
- 我是一路反抗過來的人
- 我是某次傷害之後改變的人

這些都不是單純事實陳述，  
而是身份敘事。  
也就是說，  
身份不是純粹被發現的，  
而是被講述出來的。

### （3）時間層（Time）

故事讓時間變成：

- 有起點
- 有轉折
- 有方向
- 有未來感
- 有回顧可能

否則時間就只是不斷流逝。  

所以故事不只整理事件，  
也整理時間。  
它讓人生不只是過去、現在、未來的堆積，  
而是一條：

```text
有結構的生命線
```

## 九、一個關鍵轉折：人是敘事動物（Narrative Animal）

從 Day 003 開始，  
可以先記住一個非常重要的定義：

```text
人是敘事動物（Narrative Animal）
```

這個說法的意思不是  
人完全沒有理性。  
而是說：

```text
人在實際生活中，
並不是主要靠抽象邏輯來理解自己，
而是靠故事。
```

我們會用故事來：

- 合理化自己的決定
- 解釋自己的失敗
- 美化自己的過去
- 想像自己的未來
- 給自己的痛苦找到位置
- 替自己的行動賦予正當性

也就是說，  
理性常常不是生活的第一語言。  
故事才是。  

所以更準確地說：

```text
人不是不用邏輯，
但人通常是先在敘事中活著，
再用邏輯去修補或說明那個敘事。
```

這也是為什麼  
文學對理解人如此重要。  
因為文學最擅長的，  
不是證明一個命題，  
而是展開一條生命敘事。

## 十、從世界故事到人故事：文學真正的轉折

如果 Day 002 處理的是：

```text
世界如何變成故事
```

那麼 Day 003 要處理的就是：

```text
人如何變成故事
```

這是一個非常關鍵的轉折。  

在神話中，  
敘事主要在回答：

- 世界怎麼來的
- 神與人如何分工
- 災難與秩序如何成立

但到了更成熟的文學階段，  
敘事開始越來越集中於：

- 一個人的選擇
- 一個人的記憶
- 一個人的錯誤
- 一個人的欲望
- 一個人的命運線

這表示文學真正的起點，  
不只是世界被敘事化，  
而是：

```text
人本身被故事化
```

也就是說，  
文學不是在世界之外  
“加上一些人物”。  
而是在做一件更深的事情：

```text
把「人」本身生產成可理解對象
```

## 十一、如果沒有故事，還有「人」嗎？

現在可以回到最開頭的問題：

```text
如果沒有故事，
「人」還成立嗎？
```

如果這裡的“人”只是指  
生物學上的 Homo sapiens，  
那當然還成立。  

但如果這裡的“人”指的是：

- 可理解的自我
- 有連續性的身份
- 能夠回顧過去與預想未來的主體
- 能為自己行動負責的存在

那麼答案很可能是：

```text
沒有故事，
就很難有一個完整可理解的人。
```

因為“人”不只是肉身活著。  
人還需要能說：

- 我從哪裡來
- 我經歷了什麼
- 我為什麼成為現在這樣
- 我要往哪裡去

而這四件事，  
本質上都是敘事問題。  

所以今天這一課  
其實已經碰到文學最深處的一個命題：

```text
故事不是附加在人身上的裝飾，
而是人之所以能成為自己的一部分條件。
```

## 十三、為後面課程留下一個問題

今天最後先留下這個問題：

```text
如果一個人的自我是由故事構成的，
那麼改變他講述自己的方式，
是否就等於改變了他自己？
```

這個問題現在不用急著回答。  
但它會一路通往：

- 史詩中的英雄
- 悲劇中的命運人物
- 小說中的社會個體
- 現代主義中的破碎意識
- 後現代中的語言主體

因為整條文學史，  
某種程度上都在回答同一件事：

```text
人不是只有生命，
人還需要一個能夠講述自己的形式。
```
