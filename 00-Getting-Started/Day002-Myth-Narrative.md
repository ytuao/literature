# Day 002｜神話與敘事
## Myth & Narrative

今天我們要處理文學真正的起點之一：

```text
為什麼人類最早不是用理論理解世界，
而是用故事理解世界？
```

如果 Day 001 討論的是：

```text
文學是什麼？
```

那麼 Day 002 要問的就是：

```text
文學為什麼從神話開始？
```

這不只是文學史問題。  
它其實同時涉及：

- 人類如何理解世界
- 人類如何組織經驗
- 人類為什麼需要故事
- 敘事為什麼比事實更早出現

要理解後面的史詩、悲劇、小說，  
就必須先理解：

```text
神話（Myth）
```

與

```text
敘事（Narrative）
```

到底在做什麼。

---

## 一、為什麼文學要從神話開始？

如果你回看文學史，  
會發現一個非常關鍵的事實：

```text
人類最早不是用理性系統理解世界，
而是用故事理解世界。
```

在科學、哲學、歷史還沒有形成之前，  
人類已經在做一件事情：

- 解釋雷電
- 解釋季節
- 解釋死亡
- 解釋災難
- 解釋戰爭
- 解釋人從哪裡來
- 解釋世界為什麼是現在這個樣子

而這些解釋  
最早的共同形式之一就是：

```text
神話（Myth）
```

這表示什麼？  
表示神話不是文學史外圍的一點背景知識，  
而是整個文學世界最古老的母體之一。  

可以先記住一句話：

```text
在思想還沒有變成理論以前，
它先變成故事。
```

---

## 二、神話（Myth）不是「假的故事」

這裡有一個最常見、  
也最需要立刻修正的誤解：

```text
神話 = 不真實的故事
```

這種理解在日常口語裡很常見，  
但在文學研究、神話學與文化研究裡，  
這樣說太粗糙。  

更準確地說，神話不是單純的“假”，  
而是：

```text
一種用故事來組織世界的方式
```

也就是說，  
神話的功能不主要是：

```text
像科學一樣描述事實
```

而更像是在做下面這些事：

- 為世界提供起源敘述
- 為混亂提供秩序模型
- 為共同體提供共享意義
- 為制度、習俗、禁忌提供正當性
- 為不可控的命運提供可理解形式

所以神話不是在問：

```text
這件事是否符合現代事實檢驗？
```

它更像是在回答：

```text
這個世界為什麼是這樣？
我們該如何理解自己在其中的位置？
```

換句話說：

```text
神話未必是事實性的真，
但它常常追求的是存在上的真、
文化上的真、
集體經驗上的真。
```

---

## 三、神話真正處理的是「世界為什麼可理解」

人類面對世界時，  
最原始的處境之一就是：

```text
世界太大、太亂、太不可控。
```

天災會來，  
人會死，  
戰爭會爆發，  
愛會失敗，  
孩子會出生，  
季節會輪替，  
王國會興亡。  

如果這一切都只是純粹偶然，  
那麼人其實很難承受。  
因此神話的一個根本功能就是：

```text
讓世界看起來不是純粹混亂，
而是有某種結構。
```

例如神話會告訴你：

- 天上有神
- 地上有人
- 地下有冥界
- 災難有原因
- 死亡不是毫無意義
- 王權不是偶然出現
- 禁忌背後有神聖秩序

也就是說，  
神話不是在增加世界的神祕感，  
而恰恰是在：

```text
降低世界的不可理解性
```

所以神話最深的功能之一，  
其實是：

```text
讓人可以居住在世界裡
```

---

## 四、Narrative（敘事）是什麼？

如果說神話是文學與文化中  
最早的一類重要故事形式，  
那麼敘事（Narrative）則是更大的概念。  

最簡單地說：

```text
敘事（Narrative）
就是把事件組織成故事的方式。
```

注意，  
重點不是“事件發生了”，  
而是：

```text
事件如何被連接、排序、解釋、賦予因果與意義。
```

例如同樣是一件事：

```text
一個人死了
```

它可以被放進完全不同的敘事中：

- 神話敘事：他被神帶走
- 宗教敘事：他的靈魂回到神的國度
- 英雄敘事：他為共同體犧牲
- 悲劇敘事：他的性格缺陷導致毀滅
- 現代醫學敘事：器官衰竭導致死亡
- 犯罪敘事：他的死是一場陰謀的一部分

事件本身可能相近，  
但敘事框架一變，  
整個世界觀就變了。  

所以敘事真正厲害的地方不只是“講故事”，  
而是：

```text
決定一件事在什麼世界中被理解。
```

## 五、人類為什麼需要敘事？

這是今天最核心的問題之一：

```text
為什麼人不能只活在事實之中？
```

因為“事實”本身  
通常不能直接告訴我們：

- 這件事為什麼發生
- 它和我有什麼關係
- 它意味著什麼
- 它在更大圖像裡佔什麼位置

現實本身往往是：

- 碎片化的
- 混亂的
- 不連續的
- 沒有天然意義保證的

但人類大腦  
通常無法長期忍受純碎片。  
我們會本能地尋找：

- 因果
- 方向
- 關聯
- 目的
- 結局
- 解釋

於是人就會自動做一件事：

```text
把事件敘事化
```

也就是把“發生過的事”，  
轉化成“可理解的故事”。  

這不只是文學現象，  
也是人類心智的基本活動。  
我們回憶自己人生時，  
其實也一直在做這件事。  

所以可以說：

```text
敘事不是文學之後才有的技巧，
而是人類理解經驗的基本方式。
```

## 六、神話與敘事的關係：不是對立，而是包含關係

你原來的講法說：

```text
Myth 是內容，
Narrative 是結構
```

這樣講對初學者有幫助，  
但如果要更準確，  
應該稍微修正。  

更好的理解方式是：

```text
敘事（Narrative）是更大的範疇，
神話（Myth）是其中一種特殊敘事。
```

也就是說：

- 不是所有敘事都是神話
- 但神話幾乎總是以敘事形式存在

神話之所以特殊，  
是因為它通常具有以下特徵：

- 關於起源或根本秩序
- 關於神、人、宇宙之間的位置
- 關於共同體如何理解自身
- 具有神聖性、權威性或集體性
- 不只是個人故事，而是世界故事

所以可以這樣理解：

| 概念 | 意義 |
|---|---|
| 神話（Myth） | 一種帶有起源、秩序、神聖性與集體意義的特殊敘事 |
| 敘事（Narrative） | 把事件組織成可理解故事的更大形式原理 |

因此，  
神話不是敘事的對立面，  
而是敘事在早期文明中  
最有力量的一種形式。

---

## 七、神話的三個核心功能

如果要把神話的功能  
先濃縮成三點，  
可以這樣理解：

### （1）世界秩序化

神話把混亂世界  
轉化成有層次、有結構的世界。  

例如：

- 天界
- 人界
- 冥界

或者：

- 神
- 英雄
- 凡人

這樣一來，  
世界不再是一團無法理解的事件，  
而是一個有位置關係的宇宙。

### （2）道德化

神話中的事件  
通常不只是發生，  
而且往往帶有評價。  

例如：

- 傲慢會招致懲罰
- 違禁會導致災難
- 忠誠會得到回報
- 僭越會引來毀滅

也就是說，  
神話會把世界寫成一個：

```text
有道德結構的宇宙
```

### （3）時間故事化

在純自然時間裡，  
時間只是流動。  

但在神話中，  
時間往往會被組織成：

- 起源（Creation / Origin）
- 衝突（Conflict）
- 轉折（Crisis）
- 解決或秩序重建（Resolution）

於是時間不再只是鐘錶流逝，  
而成為一條能夠被講述的命運線。  

這就是為什麼  
神話常常特別關心：

- 世界從哪裡來
- 人為何受苦
- 為什麼會有死亡
- 災難之後如何恢復秩序

## 八、從神話到文學：真正的起點是「世界敘事系統」

如果把這門課往前追，  
你會發現：

```text
文學不是從小說開始，
而是從神話開始。
```

原因很簡單。  
因為神話其實就是  
最早的大型世界敘事系統之一。  

它做的事情包括：

- 描寫宇宙結構
- 建立因果模式
- 提供集體記憶
- 安排人物位置
- 讓共同體共享同一套世界圖像

從這裡往後發展，  
文學史大致可以看到一條演化線：

```text
神話
↓
史詩
↓
悲劇
↓
小說
```

這條線不是說前者消失、後者取代。  
而是說文學逐漸把  
原本屬於整個共同體的“世界故事”，  
慢慢轉向更複雜的：

- 人物故事
- 城邦故事
- 家族故事
- 個人內心故事
- 現代社會故事

所以你原稿那句話其實很有力量，  
只是可以說得更準：

```text
神話是最早的世界敘事，
史詩是神話的文學擴展，
小說則是敘事的個體化與世俗化。
```

## 九、一個關鍵轉折：神話不再只是信仰，而變成文學材料

這裡有一個非常重要的轉折，  
也是整條課後面會一直用到的觀念：

```text
人類後來逐漸不再只「相信神話」，
而開始「使用神話」。
```

這是什麼意思？  

在早期社會中，  
神話往往和宗教、儀式、共同體信念連在一起。  
它不只是故事，  
而是生活世界本身的一部分。  

但後來，  
隨著哲學、歷史、科學與世俗文化的發展，  
神話開始發生變化。  
它不再只是被當作字面真理來接受，  
而越來越多地被當成：

- 文學材料
- 象徵資源
- 原型結構
- 文化記憶
- 敘事模板

也就是說：

```text
神話的信仰功能可能削弱了，
但神話的文學生命沒有消失。
```

所以更準確地說，  
不是“神話死了”，  
而是：

```text
神話從宗教真理，
轉化為文學與文化的深層語言。
```

這一點非常重要。  
因為你之後會發現：  
從希臘悲劇到現代小說，  
神話其實一直沒有離開文學。

## 十、敘事為什麼一直活著？

即使在現代社會，  
我們不再普遍用神話解釋雷電或瘟疫，  
敘事依然沒有消失。  
因為人類仍然需要把事件  
轉化成可理解的形式。  

今天我們依然活在各種敘事中：

- 國家敘事
- 家庭敘事
- 戀愛敘事
- 成功敘事
- 受害者敘事
- 革命敘事
- 科技進步敘事
- 個人成長敘事

這些敘事  
不一定有神，  
也不一定神聖，  
但它們仍然在做與神話相似的事：

- 提供結構
- 提供因果
- 提供角色位置
- 提供價值判斷
- 提供方向感

所以真正活下來的，  
不只是故事本身，  
而是：

```text
人類必須以敘事理解現實的需求
```

這就是為什麼  
神話是文學的起點，  
而敘事則是文學永遠不會消失的骨架。

## 十一、今天最重要的哲學性理解

如果把今天內容再往深一點講，  
其實它最後會導向一個很大的問題：

```text
人是否能直接面對現實？
```

原稿最後一句說：

```text
人類無法直接面對現實，
只能透過故事理解現實
```

這句話很有力量，  
但若要更準確，  
可以稍微修正成：

```text
人類不是完全不能接觸現實，
而是幾乎不可能不透過某種敘事框架來理解現實。
```

也就是說，  
問題不在於“故事是否扭曲真相”，  
而更在於：

```text
沒有敘事，
我們根本很難把真相組織成可理解的世界。
```

這個觀念非常重要。  
因為它會一路通往後面的：

- 史詩
- 悲劇
- 小說
- 現代主義
- 後現代文學

甚至也會連到哲學、歷史、政治與身份問題。  
因為每一種文學形式，  
某種程度上都在回答：

```text
人要用什麼故事，
才能活得下去？
```

## 十三、為後面課程留下一個問題

在今天結束以前，  
先留下這個問題：

```text
如果人類總要透過敘事理解世界，
那麼改變一個人的故事，
是否也就等於改變他所活的世界？
```

這個問題現在不用急著回答。  
但它會陪你走完整條文學課。  
因為從神話、史詩、悲劇到小說，  
文學一直在做同一件事情：

```text
不是複製世界，
而是替世界提供可被理解的形式。
```
