# Day 001｜文學是什麼？
## What is Literature?

今天我們要解決一個最核心、  
但也最容易被誤解的問題：

```text
文學到底是什麼？
```

很多人第一直覺會說：

- 小說
- 詩
- 戲劇
- 名著

但這些其實都只是：

```text
形式
```

而不是文學的本質。  

真正的問題不是：

```text
文學長得像什麼？
```

而是：

```text
什麼使一段文字成為文學？
```

這是一切文學課的起點。  
因為如果連「文學是什麼」都沒有想清楚，  
後面讀任何作品，  
都很容易只停留在劇情、角色和名句，  
而無法真正進入文學本身。

---

## 一、文學 vs 非文學：差別不只在內容，而在語言如何運作

先看兩段文字：

### A（新聞／資訊）

```text
1900年倫敦人口約650萬。
```

### B（文學）

```text
1900年的倫敦像一頭沉睡的巨獸，
呼吸著煤煙與霧氣。
```

這兩段都在講：

```text
倫敦
```

但它們做的事情完全不同。  

前者主要在：

- 傳遞資訊
- 提供事實
- 讓你知道某件事

後者則主要在：

- 創造感受
- 召喚想像
- 建立氣氛
- 讓你經驗某種世界

所以文學與非文學的差別，  
往往不在於**內容是不是虛構**，  
而在於：

```text
語言是否被用來生成經驗
```

可以先記住一句最簡單的話：

```text
非文學讓你知道，
文學讓你感覺。
```

但這句話還不夠完整。  
更精確地說，文學不是只讓你“感覺”，  
而是讓你：

```text
以另一種方式看見世界
```

---

## 二、文學不是「虛構 = 不真實」，而是另一種真

文學最容易被誤解的一點是：

```text
文學是虛構的，
所以是不真實的。
```

這個理解太淺。  

文學的「真」通常不是：

- 事實上的真
- 新聞式的真
- 歷史檔案式的真

而更接近：

- 情感上的真
- 人性上的真
- 經驗上的真
- 心理上的真

例如：

```text
《哈姆雷特》（Hamlet）裡的幽靈
未必是事實性的真，
但猶豫、復仇、內在分裂、存在焦慮，
都是真實的人類經驗。
```

又例如：

```text
《傲慢與偏見》（Pride and Prejudice）
不是歷史紀錄，
但階級偏見、愛情誤解、自我判斷失誤，
都是真實存在的人性結構。
```

所以文學並不是  
用虛構來逃離真實，  
而是用虛構來逼近某種  
更難直接說出的真實。  

換句話說：

```text
文學未必告訴你事情是否真的發生過，
但它會告訴你：
人為什麼會這樣感受、這樣選擇、這樣受傷。
```

這也是為什麼  
虛構作品有時反而比新聞更接近人生。

---

## 三、文學性（Literariness）是什麼？

如果再往前走一步，  
文學理論會提出一個很重要的概念：

```text
文學性（Literariness）
```

它的意思不是：

```text
一本書是不是有名
```

也不是：

```text
是不是經典
```

而是：

```text
一段文字之所以顯得像文學，
到底依靠了哪些特徵？
```

20 世紀的俄國形式主義（Russian Formalism）  
特別重視這個問題。  
像什克洛夫斯基（Viktor Shklovsky）  
就提出過一個很重要的觀點：

```text
文學的重要功能之一，
是把熟悉的東西變得陌生。
```

這叫做：

```text
陌生化（Defamiliarization / Ostranenie）
```

意思是：  
日常生活中的東西，  
我們太熟了，熟到看不見。  
而文學會用特殊方式  
讓它重新變得可見。  

例如日常語言只是說：

```text
天亮了。
```

但文學可能會寫成：

```text
夜色像一層用舊的布，
被清晨慢慢扯開。
```

這時候，  
你不是只知道“天亮了”，  
而是重新看見了“天亮”這件事。  

所以文學的一個核心功能，  
不是提供新資訊，  
而是：

```text
更新感知
```

---

## 四、文學的第一個特徵：語言被設計過

文學最明顯的一個特徵就是：

```text
語言不是透明工具，
而是被有意識設計過的材料
```

普通語言可能只是：

```text
他走進房間。
```

文學語言則可能是：

```text
他像一個遲到的人，
慢慢走進那間不屬於他的房間。
```

差別在哪裡？

- 節奏不同
- 形象不同
- 句子有情緒重量
- 字詞的選擇被安排過
- 語言本身開始產生意義

這裡很重要的一點是：  
在非文學裡，語言常常只是載體；  
但在文學裡，語言本身就是內容的一部分。  

也就是說：

```text
文學不只是「說了什麼」，
也包括「怎麼說」。
```

有時候甚至可以說，  
文學真正的意義  
就藏在那個“怎麼說”裡。

---

## 五、文學的第二個特徵：時間不是鐘錶時間，而是心理時間

現實時間通常是線性的。  
一分鐘就是一分鐘，  
十年就是十年。  

但文學不是這樣。  
文學中的時間可以被：

- 拉長
- 壓縮
- 倒敘
- 插敘
- 重複
- 停頓
- 交錯

也就是說，  
文學不必服從鐘錶時間，  
它更常服從的是：

```text
心理時間
```

例如：

- 一秒鐘的猶豫，可以寫好幾頁
- 十年的婚姻，可以一句帶過
- 回憶可以突然切進現在
- 一個場景可以因情緒而變慢
- 一次創傷可以讓時間永遠停在某處

所以文學中的時間，  
其實更接近人怎麼活、怎麼記得、怎麼感受時間。  

這一點非常重要。  
因為文學不是在複製現實時間，  
而是在呈現：

```text
人如何經驗時間
```

---

## 六、文學的第三個特徵：世界不是隨機事件，而是意義結構

現實生活裡，  
很多事情只是碰巧發生。  
它們沒有安排，  
沒有象徵，  
沒有明確結構。  

但文學通常不會滿足於：

```text
事情只是發生了
```

它會追問：

- 為什麼是這個人物？
- 為什麼是這個時間？
- 為什麼偏偏在這個場景？
- 這件事和人物命運有什麼關係？
- 它是否具有象徵意味？

所以文學中的世界，  
通常不是純粹的事件堆積，  
而是一個：

```text
有意義的結構
```

這也就是為什麼  
文學作品裡的一扇門、一場雨、一封信、一句沉默，  
常常都不只是它自己。  
它們還可能意味著：

- 轉折
- 預兆
- 壓抑
- 渴望
- 失去
- 命運的臨界點

所以文學閱讀  
和一般資訊閱讀最大的不同之一在於：  

你不只是問：

```text
發生了什麼？
```

而是問：

```text
這件事為什麼以這種方式發生？
```

---

## 七、文學真正做的事：把人生變得可理解

現實裡，  
事情常常只是混亂地發生。  
人來不及理解，  
也沒有完整結構。  

但文學會做一件非常重要的事：

```text
把人生變得可理解
```

注意，這不是說文學把人生變簡單。  
恰恰相反，  
文學往往讓人生顯得更複雜。  
但它會把這種複雜  
呈現成某種可以被凝視、被思考、被感受的形式。  

例如文學會讓你開始理解：

- 人為什麼會重複犯同樣的錯
- 為什麼愛常常和誤解綁在一起
- 為什麼人明知有代價還是會選擇
- 為什麼家庭、記憶、階級、欲望會塑造一個人
- 為什麼有些痛苦很難直接說出口

所以我們讀文學時，  
常常會有一種感覺：

```text
好像突然理解了自己
好像突然理解了別人
好像突然看見人生原來有這樣的結構
```

這就是文學的力量。  
它不是把人生變成結論，  
而是把人生變成：

```text
可以被理解的經驗形式
```

---

## 八、文學不是答案，而是問題的深化

很多人讀書時，  
會不自覺地想問：

```text
所以這本書到底在告訴我們什麼？
```

這種問法不能說完全錯，  
但對文學來說常常太急了。  

因為文學最重要的力量，  
往往不是給答案，  
而是讓問題變得更深、更準確。  

例如文學不會直接告訴你：

```text
愛情是什麼
```

但它會讓你問：

- 為什麼人會愛錯人？
- 為什麼明知會痛還是選擇愛？
- 為什麼理解一個人那麼難？
- 為什麼親密常常伴隨誤解？

它也不會直接告訴你：

```text
人生的意義是什麼
```

但它會讓你開始意識到：

- 人為什麼害怕虛無？
- 為什麼人需要被看見？
- 為什麼自由常常伴隨焦慮？
- 為什麼選擇總有代價？

所以文學真正厲害的地方是：

```text
它讓問題更清晰，
但讓答案保持開放。
```

這一點和哲學其實非常接近。  
只是哲學多半直接處理概念，  
文學則把問題活成：

- 人物
- 命運
- 場景
- 對話
- 內心獨白
- 敘事結構

---

## 九、文學不是固定本質，而是一種閱讀方式

還有一點要補充得更準確。  
很多理論家都提醒過：  
文學未必有一個永遠不變、人人同意的本質。  

因為同一段文字，  
在不同時代、不同文化、不同讀法裡，  
可能會被當成：

- 歷史材料
- 宗教文本
- 政治修辭
- 哲學寫作
- 文學作品

也就是說，  
「什麼是文學」  
不只取決於文本本身，  
也取決於：

```text
我們如何閱讀它
```

這很重要。  
因為它提醒我們：  
文學不是一個死的分類，  
而是一種活的文化實踐。  

所以更成熟的說法也許是：

```text
文學不是某種固定物，
而是一種使語言、經驗與世界變得可感、可思、可反覆解釋的閱讀領域。
```

---

## 十、如果用一句話定義文學

如果今天一定要先給一個  
暫時性的簡化定義，  
那可以這樣說：

```text
文學是透過被設計過的語言，
把人類經驗組織成可感、可思、可反覆解釋的形式。
```

這個定義比「文學是小說、詩和戲劇」更準，  
也比「文學就是虛構作品」更深。  
因為它把幾個重點都放進去了：

- 語言
- 經驗
- 結構
- 感受
- 思考
- 解釋的開放性

## 十一、今日核心總結（Day 001）

今天請記住這幾句話：

```text
第一，
文學不是資訊，
而是經驗。
```

```text
第二，
文學不主要追求事實上的真，
而更追求人性、情感與經驗的真。
```

```text
第三，
文學的本質之一，
是把熟悉的世界變得陌生，
讓我們重新看見它。
```

```text
第四，
文學不是替人生提供標準答案，
而是把人生變得更可思考。
```

```text
第五，
文學不只是「寫了什麼」，
也是「怎麼寫」與「怎麼讀」。
```

## 十二、為整門課先留下一個問題

在開始讀作品以前，  
今天最後先留下這個問題：

```text
如果一段文字讓你第一次真正看見
自己原本早已習慣的一件事，
那麼它是否就已經開始成為文學？
```

這個問題，  
你現在不一定要回答。  
但它會陪你走完整門課。  
因為從史詩、悲劇、小說到現代主義，  
幾乎所有重要文學作品，  
都在做這件事：

```text
讓我們重新看見
原本以為自己早就知道的人生
```
