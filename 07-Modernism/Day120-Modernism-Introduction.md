# Day 120｜現代主義導論
## Modernism Introduction

從今天開始，  
進入新的文學階段：

```text
Modernism
（現代主義）
```

在前面的俄羅斯文學中，  
已經看到的是：

- 靈魂
- 信仰
- 道德
- 痛苦
- 自由

但到了二十世紀，  
這一切開始發生  
明顯變化。  

---

## 一、一個新的世界開始了：現代主義不是突然出現，而是對巨變的反應

十九世紀結束之後，  
人類經歷了  
劇烈震動：

- 工業化加速
- 城市急速擴張
- 戰爭規模前所未有
- 傳統信仰崩解

這句話非常重要。  

因為：

```text
現代主義不是一種單純風格，
而是一種反應。
```

它回應的是：

- 世界不再穩定
- 意義不再明確
- 人不再完整

也就是說，  
現代主義不是  
文學家忽然想玩形式。  
而是他們開始覺得：

```text
舊的寫法，已經不足以表達新的世界。
```

---

## 二、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
現代主義的核心，
不是描述世界，
而是描述「世界如何被經驗」。
```

這是一個  
非常巨大的轉變。  

但若更嚴格，  
最好補上：

```text
現代主義並不是完全放棄現實，
而是懷疑傳統寫實主義那種穩定、透明、線性的再現方式，
是否還足以表達現代人對世界的主觀感知、時間斷裂、
心理流動與存在不確定性。
```

這句話非常重要。  

也就是說，  
現代主義真正改變的  
不只是題材。  
而是：

- “現實”該怎麼被寫
- “人”該怎麼被理解
- “故事”還能不能保持傳統秩序

從這裡開始，  
小說不再只是  
把外部世界  
清楚排列給讀者看。  
它更想呈現的是：

```text
人如何在內部經歷這個世界。
```

---

## 三、從「世界」到「感知」：現代主義如何改寫小說的重心

在傳統小說中，  
通常會預設：

- 世界是穩定的
- 人物在其中行動
- 故事有清晰結構

但現代主義  
改變了這一點。  

它讓人感受到：

- 世界變得模糊
- 感知變得主觀
- 時間變得不穩定

這句話非常重要。  

更精確地說，  
現代主義的一個  
重大轉向是：

```text
把現實理解為不再只是外部事件的總和，
而是被意識、感知、語言形式與記憶機制
共同組織出來的經驗世界。
```

也就是說，  
現代主義關心的  
不只是“發生了什麼”。  
而是：

- 這件事如何被感受到
- 記憶如何改變它
- 語言如何塑造它
- 一個人的內在如何扭曲或重組它

---

## 四、人不再是中心：主體開始變得破碎、流動而不穩定

在現代主義中：

```text
人不再是清晰完整的主體。
```

而更像是：

- 破碎的意識
- 流動的感覺
- 矛盾的自我

這句話非常重要。  

在傳統文學裡，  
“人物”往往比較：

- 清晰
- 一致
- 可理解

但現代主義中的人  
則常常顯得：

- 矛盾
- 分裂
- 不穩定

也就是說，  
小說不再預設  
人是一個  
穩定透明的“我”。  
而是開始追問：

```text
所謂自我，
是否本來就是斷裂而不完整的？
```

---

## 五、內在世界成為主舞台：思想、記憶與時間如何取代外部事件

現代主義  
不再像傳統小說那樣  
主要強調外部事件。  
它更關注：

- 思想如何流動
- 記憶如何運作
- 時間如何被感知

這句話非常重要。  

因此在現代主義中：

```text
時間不再線性。
```

而是：

- 過去與現在交錯
- 記憶影響現實
- 一瞬間可能展開成巨大的內在經驗

也就是說，  
現代主義常常讓讀者看到：  

```text
真正重要的不一定是事件本身，
而是事件在意識裡留下的波紋。
```

這也就是  
為什麼“記憶”與“時間”  
會成為現代主義  
如此核心的結構。  

---

## 六、語言不再透明：它不只是工具，也會改變現實感

現代主義  
還提出一個  
非常重要的觀點：

```text
語言不再只是工具。
```

這句話非常重要。  

在較傳統的小說觀裡，  
語言常被看作  
透明媒介，  
只是拿來清楚描述  
外部世界。  

但現代主義開始懷疑：  

```text
語言本身也會扭曲、重組、
延遲甚至製造現實感。
```

也就是說，  
人不是先完整看見世界，  
再把它用語言說出來。  
很多時候，  
人對世界的理解  
本來就已經被：

- 語言形式
- 敘事方式
- 聯想結構
- 心理節奏

所影響。  

因此現代主義  
會特別重視：

- 句子的節奏
- 敘事的斷裂
- 聲音與語感
- 語言的實驗性

---

## 七、四位關鍵作家：他們如何共同重塑小說形式

接下來會深入  
四位核心人物：

- 弗朗茨·卡夫卡（Franz Kafka）
- 馬塞爾·普魯斯特（Marcel Proust）
- 詹姆斯·喬伊斯（James Joyce）
- 維吉尼亞·吳爾芙（Virginia Woolf）

他們共同重塑了  
小說的形式。  

這句話非常重要。  

雖然四人的風格  
非常不同，  
但他們共享  
一個核心前提：

```text
現實不是固定的，
而是被意識所經驗、組織與變形的。
```

也因此，  
現代主義  
不是單一路線。  
而是一組  
共同面對現代世界的  
不同寫作方法。  

---

## 八、卡夫卡、普魯斯特、喬伊斯、吳爾芙：四條典型路線

```text
卡夫卡
（Franz Kafka）
```

展示的是：

- 制度變得不可理解
- 個體無法掌控命運
- 現實本身變得荒謬

```text
普魯斯特
（Marcel Proust）
```

關心的是：

- 時間如何在記憶中被重新創造
- 過去如何不斷回到現在

```text
喬伊斯
（James Joyce）
```

則徹底改變了  
小說語言：

- 意識流
- 碎片化敘事
- 語言實驗

```text
吳爾芙
（Virginia Woolf）
```

則特別關注：

- 瞬間感受
- 心理流動
- 日常中的深層意義

這句話非常重要。  

也就是說，  
四位作家  
雖然各自不同，  
卻都在同一個方向上  
推動文學前進：  

```text
讓小說更接近意識實際運作的樣子。
```

---

## 九、現代主義的共同感受：不確定、模糊、開放，甚至焦慮

現代主義  
最基本的感受，  
往往是：

- 不確定
- 模糊
- 開放
- 焦慮

這句話非常重要。  

為什麼會這樣？  
因為現代世界  
本身變了：

- 戰爭改變了人對文明的信心
- 都市改變了人的節奏與感官經驗
- 科技改變了世界的速度
- 心理學改變了人對自我的理解

也就是說，  
現代主義  
不是故意把小說  
寫得難懂。  
它更像是在說：

```text
如果世界本身已經不確定，
文學又怎麼能假裝它仍然穩定透明？
```

## 十、從俄羅斯到現代主義：課程焦點如何發生轉換

如果把整條課程  
放在一起看，  
就會發現一個  
非常清楚的轉換。  

俄羅斯文學關心的是：

```text
靈魂的深度。
```

而現代主義  
更進一步關心的是：

```text
意識的形式。
```

這句話非常重要。  

也就是說，  
現在正從：

```text
「人是誰？」
```

逐漸轉向：

```text
「人如何感知世界？」
```

這正是  
Day 120  
在整個課程中的位置。  

因為到了這裡，  
文學開始不只問：

- 人有沒有靈魂
- 人有沒有道德
- 人有沒有信仰

而是開始問：

- 人的意識如何運作
- 時間如何在內心裡流動
- 記憶如何改造現實
- 語言如何塑造經驗

這就是  
進入現代主義之後  
最關鍵的轉向。

## 十二、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果現代主義真正改變的，
不是單純的寫作技巧，
而是對“現實如何被感知、被記憶、被語言組織”的整體理解，
那麼接下來閱讀現代主義作品時，
是否最重要的能力之一，
就是不只問“故事發生了什麼”，
而要問“這個世界是如何透過角色的意識、時間感、記憶方式與語言形式被經驗出來的”，
以及“當人物變得破碎、時間變得斷裂、語言變得不透明時，
文學是否反而更接近現代人真正活在世界中的方式”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看情節推進
- 不只看角色做了什麼
- 不只看故事有沒有清楚結局

而是進一步去看：

- 角色如何感知世界
- 記憶如何改造現實
- 時間如何在意識中流動
- 語言如何參與意義生成
- 破碎形式是否本身就在表達現代經驗

因為從今天開始，  
文學不再只是  
描寫“世界是什麼樣子”。  

更準確地說：

```text
從今天開始，
正式把“現代主義”理解成文學對現代經驗的一次根本性改寫——
在這裡，
真正重要的已不只是外部現實本身，
而是現實如何被意識感知、被記憶重組、被語言塑形、被時間撕裂；
而現代主義之所以偉大，
正因為它不再滿足於用穩定透明的形式描述一個已經不再穩定透明的世界，
而是勇敢地讓小說本身也變得斷裂、流動、主觀與不確定，
從而更接近現代人實際經驗世界的方式。
```
