# Day 131｜現代感知
## Modern Perception

今天要進入  
現代主義的一個關鍵轉折點：

```text
Modern Perception
（現代感知）
```

核心問題非常直接：

```text
我們所「看到」的世界，是客觀存在，
還是被我們的意識加工過的結果？
```

這個問題  
看起來像哲學，  
其實會直接改變  
對文學與現實的理解。  

---

## 一、你看到的世界，真的是世界嗎？

人很容易直覺地以為：  

- 世界就在那裡
- 我們去看它
- 感知只是把它記錄下來

就像照相機一樣。  

這是很自然的想法。  
也是古典思維中  
非常重要的一個前提。  

但現代主義開始懷疑：  
事情真的這麼簡單嗎？  

這句話非常重要。  

因為一旦仔細觀察，  
就會發現你所“看到”的  
從來不只是物體本身。  
你看到的總同時包含：

- 光線條件
- 記憶痕跡
- 當下情緒
- 注意力方向
- 身體位置
- 語言分類方式

也就是說，  
你不是單純看見世界。  
你更像是在：

```text
生成一個被感知出來的世界。
```

---

## 二、今天最重要的一句話

今天一定要記住  
最重要的一句話：

```text
現代主義發現：
我們從未直接看見世界，
我們只看見「被感知過的世界」。
```

這句話幾乎重寫了  
現代文學的基礎。  

但若更嚴格，  
最好補上：

```text
現代主義並不一定否認外部世界的存在，
而是強調：
世界一旦進入人的經驗，
就必然已經經過注意、情緒、記憶、身體位置與語言形式的中介。
```

這句話非常重要。  

同時也最好再補一句：

```text
這並不意味著一切都只是任意主觀，
而是說現實對人而言總以某種被感知、被組織、被解釋的方式出現；
現代主義關心的，
正是這個「出現過程」本身。
```

也就是說，  
現代主義不是在說  
“世界不存在”。  
而是在說：

```text
我們無法跳出感知本身，
直接抵達一個純粹、未經中介的世界。
```

---

## 三、傳統觀點 vs 現代主義：從透明記錄，到感知建構

在較傳統的觀點裡，  
世界往往被理解為：

- 先存在
- 可被觀察
- 可被中立記錄

感知則像：

```text
客觀記錄現實的裝置。
```

這句話非常重要。  

但現代主義的顛覆在於：

```text
感知不是記錄，
而是建構。
```

也就是說，  
感知不是把外界  
原樣複製進心裡。  
相反地，  
它總在做很多事：

- 選擇哪些東西被看見
- 忽略哪些東西被排除
- 把零散刺激組織成形式
- 賦予當下場景某種情緒與意義

因此“看到”  
從來不是純接收。  
而是一種主動形成。  

這也就是  
現代主義  
對感知問題  
最根本的革命。  

---

## 四、你不是看見世界，而是在條件中生成世界

現代主義很重要的一點是：

```text
你看到的不是物體本身，
而是：
光線 + 記憶 + 情緒 + 注意力
```

這句話非常重要。  

也就是說，  
一張椅子不只是一張椅子。  
它可能同時是：

- 某段童年記憶的觸發點
- 某個等待時刻的象徵
- 某種孤獨感的背景
- 某種疲憊身體的停靠處

同一物體  
進入不同人的感知時，  
會呈現出完全不同的現實質地。  

因此現代主義  
對“看見”的理解  
已經不再是：

```text
世界 → 眼睛 → 影像
```

而更接近：

```text
刺激 → 感知組織 → 經驗世界
```

也就是說，  
你不是消極看見。  
你是在條件中  
不斷生成世界版本。

## 五、同一個世界，不同的人：為什麼兩個人會看到不同的“現實”

兩個人看同一場景，  
可能會看到  
完全不同的東西。  

因為：

- 經驗不同
- 情緒不同
- 注意力不同
- 期待不同

這句話非常重要。  

例如同一場雨：

- 一個人看到麻煩
- 一個人看到浪漫
- 一個人看到童年
- 一個人只看到交通延誤

外部場景  
看似沒有變。  
但“世界”其實已經變了。  

這並不表示  
大家任意亂想。  
更準確地說：

```text
現實對每個人而言，
都總是以某種被感知、被組織、被解釋的版本出現。
```

而現代主義真正關心的，  
正是這些不同版本  
如何形成。

## 六、伍爾芙、普魯斯特、喬伊斯、卡夫卡：四種現代感知的文學形式

如果把幾位核心作家  
放在一起看，  
現代感知的輪廓  
會非常清楚。  

### 伍爾芙  
```text
Virginia Woolf
```

在她的作品中，  
一片光影、  
一個聲音、  
一個轉瞬即逝的場景，  
都可能引發  
整段意識流動。  

也就是說：

```text
現實不是穩定背景，
而是在瞬間中生成。
```

### 普魯斯特  
```text
Marcel Proust
```

指出：

```text
感知永遠與記憶混合。
```

你看到的現在，  
其實早已包含過去。  

### 喬伊斯  
```text
James Joyce
```

展示：

```text
感知不是單一畫面，
而是語言、記憶、文化引用與身體反應交織成的信息場。
```

### 卡夫卡  
```text
Franz Kafka
```

則呈現：

```text
世界看似正常，
但感知無法確定其意義。
```

這句話非常重要。  

也就是說，  
這些作家雖然方向不同，  
卻都共同完成一件事：

```text
讓感知本身成為文學的主要現實。
```

## 七、感知不是被動，而是主動：注意力決定世界的樣子

現代主義的一個核心發現是：

```text
感知不是接收，
而是選擇與組織。
```

這句話非常重要。  

你注意什麼，  
會決定你看到什麼；  
你忽略什麼，  
也會決定你看不到什麼。  

所以可以說：

```text
注意力 = 現實的篩選器
```

也就是說，  
世界不是完整地  
自動灌進意識。  
而是被注意力  
切分、強調、排序。  

例如在一個房間裡，  
有人先注意光線，  
有人先注意人的表情，  
有人先注意自己的不安。  

因此感知  
從來不是全景鏡頭。  
它是一種  
有方向、有偏好、  
有遺漏的主動工作。  

---

## 八、情緒如何改變世界：現實沒變，但世界變了

現代主義  
還非常敏感地看到：  

```text
情緒會改變感知。
```

這句話非常重要。  

同一個場景裡：

- 快樂時，世界明亮
- 悲傷時，世界灰暗
- 焦慮時，一切都像威脅
- 平靜時，同樣場景又顯得柔和

也就是說，  
外在現實  
可能沒有改變。  
但你所經驗到的世界  
已經改變。  

因此現代主義  
對現實的理解  
不再停留在  
“外面有什麼”。  
它更關心：

```text
外面的東西，
如何在某種情緒狀態下被世界化。
```

這也是為什麼  
現代文學中的場景  
常常不是單純背景。  
它們其實是  
人物感知結構的外化。

## 九、感知的三層結構：感官輸入、心理加工、語言解釋

如果把現代主義的感知  
稍微整理一下，  
可以把它拆成三層：

### 1. 感官輸入  
光、聲音、觸覺、氣味等刺激進來。  

### 2. 心理加工  
記憶、情緒、期待、身體狀態  
開始對刺激進行組織。  

### 3. 語言解釋  
我們用詞語、概念、敘述  
把這個世界說出來。  

這句話非常重要。  

也就是說，  
我們從來不只是  
“有感覺”。  
我們還會：

- 把感覺轉成經驗
- 把經驗轉成可理解形式
- 把可理解形式轉成語言世界

因此當你說  
“我看到一棵樹”時，  
其實已經不是  
純粹感官事件。  
它同時包含：

- 感知選擇
- 心理加工
- 語言命名

所以人  
從來沒有直接接觸  
赤裸的現實。  
人接觸的總是：

```text
被過濾過的現實版本。
```

## 十、從世界本身，到世界如何被經驗：現代主義的哲學轉向

這裡其實連接到  
一個非常大的哲學轉向。  

傳統問題更像是：

```text
世界是什麼？
```

但現代問題  
慢慢變成：

```text
世界如何被經驗？
```

這句話非常重要。  

這也是為什麼  
現代主義會和：

- 現象學（phenomenology）
- 存在主義
- 心理學

發生深刻連接。  

因為這些思想  
都在關心同一件事：  

```text
現實不是單純在那裡等人複製，
而是總在經驗中出現。
```

也就是說，  
現代主義讓“現實”  
變成一種：

```text
主體依賴的結構。
```

但這不是隨便主觀化一切。  
更準確地說，  
它是在研究：

```text
現實如何對主體顯現。
```

這就是現代感知  
真正的哲學深度。

## 十一、沒有唯一世界，只有不同經驗版本的世界

當感知被這樣理解後，  
就會導向一個  
非常劇烈的結論：

```text
沒有唯一世界，
只有不同經驗版本的世界。
```

這句話需要小心理解。  
它不是說  
外部世界根本不存在。  
而是說：

```text
對人而言，
世界永遠是以某種被感知、被組織、被解釋的形式出現。
```

這句話非常重要。  

因此“世界”  
不再只是物理總和。  
它同時也是：

- 被注意到的世界
- 被情緒染色的世界
- 被記憶滲透的世界
- 被語言命名的世界

也就是說，  
現代主義  
讓世界從“單一客體”  
變成：

```text
多種經驗版本的現實。
```

這也就是  
為什麼現代文學  
如此重視觀點、  
內心、  
感官、  
片段與主體性。

Day 131

## 十四、為接下來的閱讀留下最關鍵的一個問題

今天最後先留下這個問題：

```text
如果現代主義真正發現的，
不是“人對世界有主觀看法”這麼簡單，
而是“世界一旦被人經驗，
就必然已經經過注意、情緒、記憶、身體位置與語言形式的中介，
因此我們從來只活在被感知、被組織、被解釋的現實之中”，
那麼接下來閱讀現代文學時，
是否最重要的能力之一，
就是不只問“作品描寫了什麼世界”，
而要問“這個世界是透過什麼樣的感知結構被生成出來的”，
以及“如果我們永遠無法跳出感知本身直接抵達一個純粹世界，
那麼所謂現實，
是否本來就是一種被主體經驗出來的形狀”？
```

這個問題現在不用急著回答。  
但它會直接改變  
接下來的閱讀方式：

- 不只看場景描寫了什麼
- 不只看人物看見了什麼
- 不只看外部現實是否真實

而是進一步去看：

- 注意力如何篩選世界
- 情緒如何改變場景色調
- 記憶如何滲入感知
- 語言如何把感知轉成現實
- 文學如何把“世界的出現方式”寫成主要內容

因為從今天開始，  
文學不再只是  
描寫“世界如何存在”。  

更準確地說：

```text
從今天開始，
正式把“現代感知”理解成現代文學對現實本身的一次根本改寫——
在這裡，
真正重要的已不再只是外部世界客觀上是什麼樣子，
而是那個世界如何透過注意、情緒、記憶、身體位置與語言形式
被經驗為某種特定的現實版本；
而現代主義之所以如此關鍵，
正因為它比許多傳統寫法更清楚地承認：
人從來不是站在世界外面觀看世界，
而總是身在感知之中，
並在感知之中與世界一起生成。
```
